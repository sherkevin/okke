# Token-Local Resonance Anchoring: Structure-Preserving Decode-Time Intervention for MLLM Grounding

## Abstract
Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. Existing inference-time mitigation strategies—such as contrastive decoding, attention control, and layer-contrast methods—are highly competitive sequence-level heuristics. However, they leave open the question of whether injecting explicit *token-local visual evidence* directly into decode-time logits can improve physical object grounding without collapsing linguistic structure or incurring catastrophic latency. We present **Token-Local Resonance Anchoring (TLRA)**, a decode-time intervention protocol that reweights candidate tokens using localized spatial support extracted from contextualized visual states.

TLRA operates on three structural commitments. First, it extracts token-specific spatial support via an **Adaptive Top-$k$** mechanism over a strictly bounded candidate window ($M$). Second, to avoid the prohibitive latency of autoregressive part-of-speech tagging, we introduce **Vocabulary-Anchored Semantic Masking (VASM)**—an offline, pre-computed WordNet static mask coupled with dynamic $O(1)$ Byte-Pair Encoding (BPE) continuation inheritance, mathematically bypassing functional syntax and abstract reasoning tokens. Third, acknowledging that native visual and lexical states lack perfect alignment, we structurally separate our method into `TLRA_zero` (a pure zero-shot viability probe) and `TLRA_calib` (a Test-Time Adaptation module using a neutrally-trained projection). To ensure rigorous scientific claims, we enforce a strict, hypothesis-driven evaluation protocol: auditing Average Generation Length (AGL) to reject truncation artifacts, defining a strict $|\Delta| < 1.5\%$ flat-line negative control on dense text OCR (DocVQA), replacing numerically unstable logit-range penalties with robust dispersion scaling, and testing spatial routing against a calibrated MeanPool baseline on strictly out-of-calibrator-distribution (unseen) object categories to prevent data-leakage artifacts.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable reasoning capabilities, yet their autoregressive decoding remains vulnerable to hallucination. The model frequently generates text that is fluent and plausible but weakly supported by the actual visual input. This failure is acutely problematic in the grounding of physical entities—such as small-object recognition, spatial positioning, and counting—where the necessary visual evidence is highly localized in the patch space.

Current inference-time mitigation strategies represent major advances in the field and serve as our primary competitive baselines. Contrastive decoding methods like VCD contrast different input noise conditions to reshape distributions; attention-based control methods like OPERA penalize abnormal generation patterns; and layer-contrast methods like DoLa exploit differences between shallow and deep representations. These are highly effective *sequence-level or distribution-level language heuristics*. However, they leave a specific, complementary proposition untested: 

**Can we inject token-local spatial visual evidence into decode-time logits to outperform sequence-level heuristics on physical grounding, without destroying language structure or incurring catastrophic latency?**

Executing this token-local intervention is mechanically non-trivial due to three explicit bottlenecks:
1. **The Autoregressive Latency Paradox:** Running NLP pipelines (e.g., POS taggers) to isolate physical entity tokens at decode-time is fundamentally flawed. Taggers require complete sentences to resolve ambiguity, and executing them dynamically for $M$ candidate tokens per step incurs catastrophic latency.
2. **Embedding Asymmetry:** Visual patch states and output logits do not natively align. Bridging this gap purely zero-shot is often noisy, necessitating a structural split between pure zero-shot probes and externally calibrated adaptation modules.
3. **Candidate Bound & Memory Bandwidth (The TPOT Tax):** Post-hoc intervention can only reweight tokens the base model keeps alive in its top-$M$ predictions. Dynamically retrieving spatial support for $M$ candidates across $N_v$ visual patches introduces a severe memory-bandwidth (Time-Per-Output-Token, TPOT) tax.

To address these challenges, we introduce **Token-Local Resonance Anchoring (TLRA)**. TLRA utilizes an **Adaptive Top-$k$** mechanism to dynamically route localized patch support to candidate tokens within a bounded logit window. To solve the latency paradox, we introduce **Vocabulary-Anchored Semantic Masking (VASM)**—an offline, pre-computed static dictionary mapping strict WordNet hypernym paths, combined with an $O(1)$ tokenizer-agnostic BPE inheritance state machine. VASM mathematically zeroes out intervention on syntax, verbs, and abstract concepts, protecting the model's structural integrity.

To ensure methodological rigor, we explicitly divide our approach: `TLRA_zero` acts as a pure zero-shot viability probe to compare against training-free baselines, while `TLRA_calib` (utilizing a neutrally trained feature translator) operates as a Test-Time Adaptation (TTA) module. We commit to a strict, hypothesis-driven evaluation protocol built on three evidence chains: validating hallucination reduction via an Average Generation Length (AGL) audit; confirming structure preservation by using dense text OCR (`DocVQA`) purely as a flat-line negative control ($|\Delta| < 1.5\%$); and proving the efficacy of *local* evidence by demanding our spatial routing outperform a calibrated global pool (`MeanPool`) strictly on verified *out-of-calibrator-distribution* object categories.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits produced under different visual conditions. Attention-based methods such as `OPERA` detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and involve different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large spatial token fields, and in video settings they further inherit spatio-temporal structure. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region of the image or a short temporal window. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser internal alternatives. That claim is experimentally verifiable through internal ablations such as `TLRA_zero` vs. `TLRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded object words or OCR-bearing entities. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this through a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Fairness Boundary: `TLRA_zero` vs. `TLRA_calib`

Because visual patch states and output logits often lack native alignment, we separate TLRA into two tracks to prevent "calibrator blackbox" ambiguity:

1. **`TLRA_zero` (Zero-Shot Probe)**: The base model's native language head is applied directly to contextualized visual states. This track serves primarily as a parameter-isolated viability probe and is the only variant directly comparable to pure zero-shot heuristics (DoLa, VCD).
2. **`TLRA_calib` (Lightweight TTA / PEFT)**: The operational variant utilizing a 2-layer MLP projection, $\Phi_{calib}$, to bridge embedding asymmetry. Crucially, to ensure a fair spatial parity test, $\Phi_{calib}$ is trained strictly via standard next-token prediction (cross-entropy) on a subset of 50k simple conceptual captions *without* the TLRA decode-time logit intervention active. It acts solely as a neutral feature translator. Its primary ablation baseline is `TLRA_MeanPool`—a variant using the exact same $\Phi_{calib}$ weights but globally pooled visual features.

### 3.2 Bounded Candidate Filtering and Adaptive Top-$k$ Resonance

At decoding step $t$, TLRA restricts its intervention to a bounded candidate set: $\mathcal{C}_t = \text{Top-}M(L_{orig})$, where $M \ll |V|$ is a strictly locked candidate window (e.g., $M=50$). Computing spatial support across $M$ tokens and $N_v$ patches (e.g., $N_v=576$) introduces a theoretical memory bandwidth tax, forcing $M$ to remain small to preserve real-time viability.

For each candidate token $c \in \mathcal{C}_t$, we compute the cosine similarity alignment score with each spatial patch $v_j$:
$$
\text{score}(v_j, c) = \frac{\phi_v(v_j) \cdot \phi_t(c)}{\|\phi_v(v_j)\| \|\phi_t(c)\|}
$$
TLRA then computes local support via an Adaptive Top-$k$ rule, determined by a locked spatial ratio $\rho$:
$$
k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)
$$
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{\text{score}(v_j, c)}{\tau_{sim}}
$$

### 3.3 Structure Preservation via Offline VASM and BPE Inheritance

Naive token-level intervention suppresses critical syntax. To avoid the severe latency of decode-time NLP tagging, TLRA introduces **Vocabulary-Anchored Semantic Masking (VASM)**. 

**1. Offline Static Dictionary:** Prior to inference, the entire vocabulary is cross-referenced against WordNet. Only tokens whose primary synsets trace an unbroken hypernym path to `physical_entity.n.01` or `object.n.01` are assigned a mask value $\gamma = 1$. Function words, verbs, and abstract syntax are permanently assigned $\gamma = 0$. 

**2. Dynamic Tokenizer-Agnostic BPE Inheritance:** Modern tokenizers heavily fragment words. To handle subwords at decode step $t$ for candidate $c_t$, VASM applies an $O(1)$ inheritance rule based on continuation markers:
$$
\gamma(c_t) = \begin{cases} 
\gamma(c_{t-1}^{\text{root}}), & \text{if } c_t \text{ is a continuation subword} \\
\text{Lookup}(c_t \text{ in WordNet}), & \text{otherwise}
\end{cases}
$$
*Concrete Example:* Consider Llama-3 tokenizing "refrigerator" as `_re` (root), `friger` (continuation), and `ator` (continuation). At decode-time, VASM looks up `_re`. If `_re` successfully maps to a physical entity ($\gamma=1$), the subsequent subwords `friger` and `ator` inherit $\gamma=1$ in $O(1)$ time without further dictionary lookups. If the root token is too fragmented to match a WordNet entry, VASM safely defaults to $\gamma=0$, intentionally missing an intervention rather than corrupting syntax.

### 3.4 Stabilized Logit Intervention

TLRA converts raw local support into a relative evidence score over the candidate set:
$$
\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)
$$
To apply the penalty, prior formulations relying on the absolute logit range ($\max - \min$) proved numerically unstable. Instead, TLRA dynamically scales the penalty using the standard deviation of the candidate logits, $\sigma_L = \text{Std}(L_{orig}(\mathcal{C}_t))$. The final logit is explicitly gated by the VASM mask $\gamma(c)$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \sigma_L \cdot (1 - \hat S(c))
$$
If a token is functional syntax ($\gamma(c)=0$), the penalty zeroes out entirely, yielding $L_{final}(c) = L_{orig}(c)$, mathematically protecting linguistic structure.

## 4. Evaluation Protocol and Experimental Design

To ensure methodological transparency, our evaluation is structured as a rigorous protocol mapping explicit hypotheses to specific evidence chains. All hyperparameters are strictly locked uniformly based on a single validation set.

### 4.1 Evidence Chain A: Hallucination Reduction and Truncation Audit
**Hypothesis:** Token-local evidence reduces physical hallucination more effectively than sequence-level heuristics, without artificially inflating precision through premature truncation.
- **Protocol (Table 1 Planned):** Evaluated on POPE and CHAIR. 
  - *Zero-Shot Group:* Base, VCD, OPERA, DoLa, and `TLRA_zero`.
  - *TTA Group:* `TLRA_MeanPool`, `TLRA_AdaptiveTopK`.
- **The AGL Audit:** We introduce a mandatory column for **Average Generation Length (AGL)** measured in words. If any TLRA variant reduces AGL by >10% compared to the Base model, the hallucination reduction claim must be downgraded as a "silence engine" truncation artifact.
- **VASM Trigger Rate:** We log the percentage of generated tokens where $\gamma = 1$ to empirically prove the mask intervenes during physical grounding but stays largely dormant elsewhere.

### 4.2 Evidence Chain B: Structure Preservation and Negative Controls
**Hypothesis:** VASM mathematically protects abstract reasoning, and its deliberate blindness to dense text prevents unintended corruption of non-spatial tasks.
- **Protocol (Table 2 Planned):** Evaluated on MMBench V1.1, MME, MMMU (Hard), and explicitly **DocVQA**. We ablate Base, `TLRA_no_VASM`, and `TLRA_full`.
- **Validation Criteria:** 
  1. `TLRA_no_VASM` must show a catastrophic drop on MMMU, proving VASM is computationally necessary.
  2. **Strict Negative Control Margin:** `DocVQA` must remain mathematically flat, defined strictly as $|\Delta \text{Accuracy}| < 1.5\%$ between Base and `TLRA_full`. If performance drops beyond this margin, VASM is erroneously penalizing OCR tokens. Any claim of "improving OCR" is explicitly prohibited.

### 4.3 Evidence Chain C: Local Evidence Value (The Parity Test)
**Hypothesis:** Spatial locality directly drives grounding performance, independent of global calibration priors.
- **Protocol (Table 3 Planned):** Evaluated on FREAK. The table is explicitly divided into **"Seen Categories"** and **"Unseen Categories"**. To prevent data leakage, "Unseen" is defined rigorously: targets must be mathematically disjoint from the 50k MSCOCO captions, verified via exact WordNet synset exclusion (intersection scripts provided in Appendix). We benchmark `TLRA_MeanPool` against `TLRA_AdaptiveTopK` using identical $\Phi_{calib}$ weights.
- **Validation Criteria (Contingency Plan):** `AdaptiveTopK` must statistically outperform `MeanPool` specifically on the *Unseen* split. If it only wins on the "Seen" split, the spatial routing hypothesis is invalidated as a data-leakage artifact, and the paper's primary claim will gracefully devolve to establishing `MeanPool + VASM` as a fast, structure-preserving TTA.

### 4.4 Systemic Latency Pareto, Failure Modes, and Ablations
To quantify the severe memory-bandwidth (TPOT) tax and mechanical limitations of TLRA, we standardize the following measurements:
- **Systemic Cost Pareto (Figure 2 Planned):** A scatter plot mapping Time-Per-Output-Token (TPOT in ms on a single NVIDIA A100 80GB GPU, $N_v = 576$) against POPE F1, plotting Base, heuristics, and TLRA variants.
- **Ablation on Candidate Window $M$:** A planned curve tracking TPOT latency (expected to scale linearly) vs. POPE F1 (expected to plateau) for $M \in \{10, 50, 100\}$.
- **Target Token Rank Histogram & Failure Case Analysis (Figure 3 Planned):** 
  - *Out-of-Candidate Audit:* A histogram showing the base model's raw logit rank for the correct visual entity at the exact step it hallucinates, justifying the $M=50$ boundary.
  - *Visual Failure/Success Cases:* (1) A success case where VASM saves the model by zeroing out the penalty on a highly probable abstract verb with spurious visual similarity; (2) A failure case demonstrating the VASM polysemy risk (e.g., penalizing the word "mouse" as an animal when the image contains a computer mouse).

## 5. Discussion and Limitations

The Token-Local Resonance Anchoring (TLRA) protocol offers a highly structured, falsifiable approach to decode-time intervention in MLLMs. By architecting Vocabulary-Anchored Semantic Masking (VASM) as an offline static dictionary tracking `physical_entity` hypernym paths with $O(1)$ BPE inheritance, we resolve the autoregressive latency paradox that plagues NLP-dependent controls. Furthermore, by rigorously separating our evaluations into a zero-shot probe and a neutrally-trained TTA module, and committing to a calibrated MeanPool parity test on strictly *unseen* object categories, we isolate the true value of spatial routing from mere parameter calibration.

The integrity of this framework is governed by explicit engineering limitations and empirical realities:
1. **Calibrator Category Leakage and Contingency:** Because $\Phi_{calib}$ leverages 50k captions, TLRA relies heavily on the strictly partitioned Seen vs. Unseen parity test. If local routing fails to generalize to mathematically disjoint objects outside the calibrator's distribution, the core claim must structurally devolve from "superior spatial routing" to "structure-preserving test-time adaptation."
2. **Lexical Rigidity, Polysemy, and BPE Fragility:** VASM prioritizes latency over perfect contextual disambiguation. Relying strictly on primary WordNet synsets means polysemous words (e.g., "mouse" or "bat") risk misclassification. Furthermore, the $O(1)$ inheritance rule is fragile to the base tokenization: if the *root* subword of a physical entity is too fragmented to match a WordNet entry, the entire word safely defaults to $\gamma=0$, missing the intervention entirely. We estimate a 3-5% structural error rate due to these factors.
3. **The Severe TPOT Tax:** Dynamically computing visual-text alignment for $M$ candidates across $N_v$ patches (e.g., $N_v=576$) at *every single generation step* imposes a massive memory-bandwidth bottleneck. As demonstrated in our planned $M$-ablation, scaling $M$ linearly degrades TPOT, making large-window intervention practically unusable for high-throughput real-time applications compared to lightweight sequence heuristics like OPERA.
4. **Out-of-Candidate Bounds:** Decode-time reweighting mathematically cannot recover an entity token if the base model has already discarded it from the initial Top-$M$ candidate pool.
5. **Modality Extensions (Spatial vs. Temporal):** The core mechanism is optimized strictly for static spatial ($H \times W$) reasoning. Extending token-local visual evidence to spatio-temporal dynamics (e.g., video) lacks native token alignment and is relegated strictly to an exploratory secondary pilot (detailed in Appendix), explicitly excluded from the primary spatial grounding claims.

Ultimately, this protocol establishes a transparent blueprint: proving that token-local visual evidence can reduce hallucination without destroying language structure, provided it is governed by rigorous bounds, strict negative controls, and an honest accounting of its computational cost.

# Token-Local Resonance Anchoring: A Hybrid Decode-Time Visual Logit Bias for Structure-Preserving MLLM Grounding

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. While existing inference-time mitigation strategies modulate internal model confidence, they do not provide a framework for injecting **token-local visual evidence** directly into decode-time logits adjustment without damaging syntax. We present **Token-Local Resonance Anchoring (TLRA)**, a hybrid calibration and decode-time intervention framework that reweights candidate tokens using localized support extracted from cached visual states.

TLRA operates on three principles. First, **token-local evidence reweighting** replaces coarse global visual contexts with candidate-wise support, mapping spatial patches to lexical logit biases. Second, **Adaptive Top-k resonance** stabilizes local evidence extraction under a bounded candidate window with dynamic logit clipping. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via an offline, deterministic BPE-level dictionary generated from WordNet physical entity synsets. We strictly delineate a training-free probe (`TLRA_zero`, operating via input-embedding space similarity) from a lightweight calibrated variant (`TLRA_calib`). To ensure rigorous evaluation, `TLRA_calib` is tested against a matched-budget `Base + LoRA` baseline trained on identical calibration data. 

The evaluation protocol enforces a falsifiable audit of structural vulnerabilities and computational boundaries. It measures hallucination reduction against Average Generation Length (AGL) collapse, audits VASM's polysemy failure rate via an online Oracle-VASM ablation, and isolates the specific value of token-local routing against a Global Visual State baseline. By tracking both task accuracy and explicit memory bandwidth utilization during the per-step gather operation, this work establishes a boundary-aware framework for decode-time visual control in MLLMs.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable capabilities, yet their autoregressive decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible but weakly supported by the visual input. This failure is particularly damaging in **fine-grained physical grounding** scenarios such as small-object recognition, spatial positioning, and counting, where the requisite evidence is localized rather than global.

Current inference-time mitigation strategies represent major advances. Contrastive decoding methods compare distributions under perturbed and clean inputs. Attention-based control methods penalize pathological autoregressive behavior. Layer-contrast methods exploit discrepancies between shallow and deep representations. While these approaches successfully modulate general model confidence, they leave a specific, complementary question unanswered: **How can we inject token-local visual evidence directly into decode-time logits adjustment without damaging syntax, multi-token entities, or downstream reasoning?**

Executing localized intervention at decode time is nontrivial due to semantic fragility and computational walls. If an intervention suppresses visually unsupported tokens without understanding language structure, it risks destroying functional syntax or fragmenting named entities. Conversely, computing cross-modal similarity between $M$ candidate tokens and $N_v$ spatial visual patches at every single autoregressive step introduces a severe latency bottleneck. Furthermore, if localized decode-time interventions fail to outperform simple prefill-time adaptations (e.g., LoRA) or global visual pooling, their architectural complexity is unjustified.

We introduce **Token-Local Resonance Anchoring (TLRA)**, a hybrid framework combining a lightweight visual-to-vocabulary calibration with a decode-time routing mechanism. TLRA avoids the per-step matrix multiplication bottleneck by conceptualizing the intervention as a **static late-fusion visual logit bias**: visual-to-lexical projections are computed once post-prefill, yielding an $N_v \times V$ matrix. The per-step decode overhead is thus reduced to a memory-bound gather operation over a bounded candidate window. To prevent structural collapse, TLRA introduces Vocabulary-Anchored Semantic Masking (VASM), an offline BPE-level dictionary explicitly mapping physical entity synsets to binary subword masks, shielding functional tokens from intervention. 

To ensure methodological rigor, we enforce strict architectural and experimental boundaries. We correct the theoretical impossibility of applying the LM head to Layer 0 visual states by explicitly formulating a zero-shot probe (`TLRA_zero`) that leverages cosine similarity within the LLM's untied input embedding space, contrasted alongside a calibrated variant (`TLRA_calib`). `TLRA_calib` is subsequently evaluated against internal matched-calibrator controls (`Global_Visual_State`) and a matched-budget `Base + LoRA` baseline to isolate the precise value of decode-time locality. Finally, we explicitly audit the semantic fragility of VASM by comparing our static heuristic against a computationally heavy Oracle VASM (online POS tagger), quantifying the exact penalty of polysemy and context-blindness in our design.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Architectural Scope and the `TLRA_zero` / `TLRA_calib` Divide

TLRA relies on the extraction of token-local support from visual states. Consequently, it requires MLLM architectures that preserve spatial provenance between the vision encoder and the LLM (e.g., standard ViT + MLP projector setups like LLaVA). Architectures that heavily compress or entangle spatial dimensions (e.g., Q-Formers) fall outside the supported scope.

To rigorously isolate the source of visual grounding improvements, we separate TLRA into two distinct variants:
1. **`TLRA_zero` (Training-Free Probe):** Applying the LLM's final-layer `lm_head` directly to Layer 0 visual states is mathematically invalid. Instead, `TLRA_zero` computes the cosine similarity between the MLP-projected spatial visual states (which reside in Layer 0) and the pre-trained LLM's input word embedding matrix $E_{in} \in \mathbb{R}^{V \times d}$. This provides a pure, zero-shot signal of visual-to-lexical alignment native to the base model.
2. **`TLRA_calib` (Hybrid Calibrated Variant):** A lightweight linear projection `Phi_calib` maps cached visual states (extracted either from the vision encoder or early LLM layers) directly to the lexical logit space $V$, trained on a bounded conceptual-caption subset while the base MLLM remains frozen.

Because `Phi_calib` introduces external supervision, any claim about the efficacy of decode-time local routing must survive two fairness checks. First, internal parity: localized routing must outperform a `Global_Visual_State` baseline sharing the exact same `Phi_calib`. Second, matched-budget fairness: `TLRA_calib` must be benchmarked against a `Base + LoRA` model adapted on the identical calibration subset, proving that dynamic decode-time visual grounding achieves something static prefill-time adaptation mathematically cannot.

### 3.2 Static Late-Fusion and Memory-Bound Complexity

To avoid prohibitive FLOP penalties inside the autoregressive loop, TLRA precomputes the visual-to-lexical projection. After the prefill phase, the cached visual states are mapped to the full vocabulary dimension $V$, yielding a static visual logit bias matrix $B \in \mathbb{R}^{N_v \times V}$. This operation is executed exactly once per image.

At each decoding step $t$, TLRA restricts intervention to a bounded candidate set:
$$
\mathcal{C}_t = \text{Top-}M(L_{orig}),
$$
where $M$ is a globally locked candidate window (e.g., $M=50$). The routing overhead per step is thus an extraction of $M$ specific columns from $B$. While FLOP-efficient ($O(M \times N_v)$ operations), gathering from a potentially large dense matrix (e.g., $1024 \times 128000$ in fp16 is $\approx 250$MB) shifts the latency bottleneck from compute to **memory bandwidth**. TLRA specifically profiles this memory gather overhead to ensure viability.

For each candidate $c \in \mathcal{C}_t$, TLRA computes local support by aggregating the top spatial patches:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{B_{j, c}}{\tau_{sim}}, \quad \text{where } k = \max(k_{min}, \lceil \rho \cdot N_v \rceil)
$$

### 3.3 Stabilized Resonance Penalty with Logit Clipping

We convert raw local support into a relative evidence score within the candidate window:
$$
\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)
$$

Intervening directly based on the candidate logit spread $\Delta_L = \max(L_{orig}) - \min(L_{orig})$ can be destructive if the base model's distribution is highly spiked. To prevent distribution destruction, TLRA applies a clipped, scaled penalty:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \min(\Delta_L, \delta_{max}) \cdot (1 - \hat S(c))
$$
where $\delta_{max}$ mathematically bounds the maximum logit penalization, and $\gamma(c) \in \{0, 1\}$ is the binary structural protection mask. Hyperparameters $\alpha$ and $\delta_{max}$ are strictly determined via grid search on a held-out validation set to avoid benchmark-fitting.

### 3.4 Vocabulary-Anchored Semantic Masking (VASM)

To preserve syntax and multi-token entities without triggering massive latency penalties, TLRA utilizes VASM, an offline precomputed deterministic binary mask ($\gamma \in \{0, 1\}^V$):
- $\gamma(c) = 1$ for visually groundable physical-entity root tokens.
- $\gamma(c) = 0$ for structural, functional, and abstract tokens.

**VASM Construction Protocol:** Generating the mask for a complex BPE vocabulary requires a reproducible pipeline. We extract physical entity synsets from English WordNet (e.g., objects, animals, distinct physical structures). These strings are matched against the LLM's vocabulary. To handle BPE fragmentation, subwords inherit the mask state of their active root token (e.g., the fragments `Ġapp` and `le` inherit $\gamma=1$ if derived from "apple"). This process typically masks $\approx 85-90\%$ of the vocabulary ($\gamma=0$), isolating the intervention to physical nouns.

We acknowledge the inherent semantic fragility of this approach: static lookups cannot resolve contextual polysemy (e.g., "Apple" the fruit vs. "Apple" the company). Arbitrary text-in-image reasoning (OCR) is therefore treated as a strict negative control. To quantify the exact performance penalty of static masking, our protocol mandates a comparison against an `Oracle_VASM` ablation, which utilizes a computationally unrestricted online POS tagger to establish the theoretical upper bound of the framework.

## 5. Discussion and Limitations

The viability of Token-Local Resonance Anchoring depends entirely on surviving strict computational and structural boundaries. By reframing TLRA as a hybrid calibration/routing method and explicitly auditing the theoretical mechanics, we identify the following primary limitations:

1. **The Semantic Fragility of Static VASM:** Offline dictionary lookups inherently fail on context-dependent polysemy. While VASM protects basic syntax, words that transition between physical entities and abstract concepts (e.g., "Apple") trigger false positives. Furthermore, high-fragmentation tasks like OCR (`DocVQA`) are fundamentally hostile to static root-token inheritance. The performance gap between static VASM and Oracle VASM defines the strict cost of bypassing an online POS tagger.
2. **The Memory Bandwidth Wall:** While the static late-fusion $B$ matrix circumvents dense per-step matrix multiplications, the process of gathering arbitrary token columns at every autoregressive step is heavily memory-bound. If large-scale models expand visual token volume significantly (e.g., $N_v > 4000$) or vocabularies scale to $>150k$, the memory bandwidth required to fetch $M \times N_v$ values per step becomes a severe framework bottleneck, potentially cratering actual `tokens/s`.
3. **The `Global_Visual_State` Threshold:** If the empirical gap between `TLRA_AdaptiveTopK` and a uniformly pooled `Global_Visual_State` baseline is negligible under identical calibration budgets, it indicates that spatial locality at decode time is overridden by the LLM's internal representation, falsifying the core need for fine-grained patch-level routing. 
4. **Top-M Unrecoverability (The Hijacking CDF):** The intervention is mathematically bounded by the base model's candidate window. If the correctly grounded token has been marginalized below the Top-$M$ cutoff by strong language priors *prior* to intervention, TLRA cannot rescue it.

Future execution of this protocol will definitively confirm whether local decode-time visual intervention can exist as a practical, context-aware mechanism, or whether its computational, memory, and semantic trade-offs restrict it to a theoretical probe.

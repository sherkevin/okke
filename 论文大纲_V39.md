# Bounded Region Alignment: Token-Local Visual Evidence for Structure-Preserving MLLM Decoding

## Abstract

Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when strong language priors overshadow the localized visual evidence required for dense grounding. While existing inference-time mitigation strategies (e.g., contrastive decoding, attention pattern penalties) serve as highly competitive, orthogonal regularizers for overarching language or attention dynamics, they lack a mechanism to inject strictly **token-local visual evidence** directly into the terminal logits space while preserving complex language structure. We present **Bounded Region Alignment (BRA)**, a decode-time intervention that reweights candidate tokens using localized visual support extracted from contextualized final-layer hidden states. 

BRA operates on three core mechanisms designed to prevent pipeline bloat and ensure precise attribution. First, to handle input-output embedding asymmetry and architectural spatial washout, we establish a strict evaluation boundary separating a zero-shot projection (`BRA_zero`) from a lightweight, dynamically calibrated bound (`BRA_calib`), governed by a direct feature-level spatial retrieval threshold rather than brittle downstream proxies. Second, **Threshold-Gated Adaptive Top-$k$ Pooling** extracts stable local evidence using a moving median—bounded by prefix-derived limits to prevent signal collapse in dense images. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax via dynamic Byte-Pair Encoding (BPE) continuation inheritance, operating alongside a **Dual-Condition Entropy Gate** calibrated via validation distributions to force visual scrutiny precisely when the LLM hallucinates with arrogant confidence.

To validate BRA, we propose a pre-registered, hypothesis-driven evaluation protocol restricted to 2D spatial grids. Evaluated across three definitive evidence chains—(A) Hallucination Reduction isolating mechanisms via a strict step-by-step ablation against a parameter-matched `Base + 5k LoRA`, (B) Structure Preservation tracking exact Out-Of-Vocabulary (OOV) intervention bypass rates on MMMU(Hard), and (C) Local Evidence Value comparing `BRA_zero` against global pooling baselines. Alongside five transparent defense lines profiling separated prefill and decode batched latency on standardized hardware, this framework establishes a scientifically sound, structurally bounded proposition for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable zero-shot capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to clarify that these methods serve as state-of-the-art orthogonal regularizers that effectively discipline the language-side or attention-side generation heuristics. However, because they operate on overarching generation dynamics rather than explicit spatial grounding, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, defensible hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination more faithfully than pure language/attention heuristics, provided it incorporates explicit structure-preserving safeguards. We conceptualize this intervention as **Bounded Region Alignment (BRA)**. In this work, we strictly bound our scope to Image-LLMs and 2D spatial reasoning, deliberately downgrading unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims surrounding spatial locality.

Executing BRA requires acknowledging and mathematically overcoming three structural barriers. First, **input-output embedding asymmetry**: raw visual tokens reside in the input space, rendering direct dot-product similarity with the terminal output vocabulary matrix invalid. Second, the **post-hoc entanglement gamble**: leveraging the final-layer hidden states of visual patches assumes localized support survives deep attention mixing. This risks severe spatial washout, varying between 1D-sequence architectures and 2D-RoPE architectures. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, raw byte fallbacks, functional syntax, polysemous words, or multi-token entities.

BRA resolves these barriers via a structured, heavily ablated framework:
1. **Bi-Level Logits Intervention (`BRA_zero` / `BRA_calib`)**: A strict evaluation boundary testing whether final-layer visual states natively align with the pre-vocabulary dimension (`BRA_zero`). Replacing brittle downstream proxies, we establish a direct feature-level spatial washout threshold. If native alignment fails, we pivot to a lightweight calibrated projection (`BRA_calib`) trained under stringent SAM-assisted semantic negative sampling.
2. **Threshold-Gated Adaptive Top-$k$ Pooling**, extracting localized spatial support via abstention-aware patch activations grounded by a robust moving median and bounded by prefix-derived limits.
3. **VASM (Vocabulary-Anchored Semantic Masking)**, utilizing dynamic BPE continuation inheritance and a brute-force polysemy superset to shield structural language, explicitly confining intervention to visual nouns while tracking OOV intervention bypass.
4. **Dual-Condition Validation-Calibrated Entropy Gate**: Correcting the paradox of confident hallucinations, our logic explicitly forces visual verification when language priors are most arrogant (high confidence on visual nouns) while bypassing non-visual structural tokens, governed by dynamically calibrated entropy bounds.

To ensure rigorous scientific boundaries and avoid pipeline bloat, our experimental design is organized into an execution protocol strictly defining three primary evidence chains and five transparent defense lines. 
- **Chain A (Hallucination Reduction):** Validating true mechanism value on POPE, CHAIR, and AGL against a parameter-matched `Base + 5k LoRA` baseline, mandated by a step-by-step architectural ablation to attribute precise gains.
- **Chain B (Structure and Reasoning Preservation):** Proving the necessity of VASM via MMBench, MME, and MMMU(Hard), transparently tracking exact Out-Of-Vocabulary (OOV) bypass rates and polysemy false positives.
- **Chain C (Local Evidence Value & Domain Shift):** Demonstrating the superiority of localized evidence via FREAK and DocVQA, strictly comparing `BRA_zero` against a `BRA_MeanPool` baseline to isolate geometric localization.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Overcoming Post-Hoc Entanglement: `BRA_zero` vs `BRA_calib`

A fundamental vulnerability in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, BRA evaluates the *contextualized* final-layer transformer hidden states. Specifically, we extract the LLM's final hidden states corresponding to the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$, strictly extracted immediately prior to the terminal $W_{vocab}$ projection.

We explicitly acknowledge the gamble of **post-hoc entanglement**: deep self-attention implicitly mixes visual states with prefix context. To handle vocabulary projection cleanly without falsely claiming zero-shot alignment if an architecture inherently destroys spatial fidelity, we define a strict operational boundary governed by a direct, feature-level **Washout Threshold**:

1. **`BRA_zero`**: A zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$. 
2. **The Washout Threshold**: We replace downstream VQA proxies (which conflate language priors with visual capability) with a direct feature-level metric. We compute zero-shot bounding box retrieval (IoU) on a held-out 500-sample spatial validation set. If the native projection ($\Phi_{zero}$) fails to achieve an average token-to-patch spatial alignment $\text{IoU} > 0.15$, we declare native spatial fidelity mathematically lost for that specific architecture.
3. **`BRA_calib`**: Upon triggering the washout threshold, we pivot to a lightweight calibrated alignment. $\Phi_{calib}: \mathbb{R}^{D_{vision}} \rightarrow \mathbb{R}^{D_{llm}}$ is strictly defined as a 2-layer MLP projection ($\approx 4.2$M parameters, appending $<0.01$ TFLOPs per step) to project the visual state prior to $W_{vocab}$. It is trained on a frozen set of 5,000 COCO image-patch to bounding-box label pairs, with the base LLM entirely frozen.

**Strict Semantic Negative Sampling (SAM/IoU Assisted):** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. Because raw bounding boxes inherently contain background pixels, we institute a strict sampling logic using an off-the-shelf segmentation mask (SAM) to extract strictly foreground patches. Crucially, to avoid latent semantic poisoning where a geometrically distant patch contains a valid object of the same class, negative patches ($W_{vocab}[c^-]$) are cross-referenced to ensure both an Intersection-over-Union (IoU) $< 0.1$ with the target *and* membership in a differing COCO semantic super-category.

### 3.2 Threshold-Gated Adaptive Top-$k$ Pooling

At decoding step $t$, BRA evaluates spatial support. A naive Softmax over all visual patches would force background patches to sum to 1.0, artificially inflating noise. We apply a threshold-gated Sigmoid activation, allowing irrelevant patches to mathematically abstain:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{noise}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{noise}\Big)
$$
To prevent $\theta_{noise}$ from becoming a brittle static hyperparameter, it is dynamically defined per-step as the moving **median** of the activation distribution across the $N_v$ patches. To address the vulnerability of dense images (e.g., a crowded scene where most patches genuinely contain objects, causing the median to spike and wipe out valid evidence), we apply a historical upper-bound clamp $\theta_{max}$. We explicitly derive $\theta_{max}$ dynamically as the 90th percentile of the visual activations computed strictly from the prefix context, ensuring it is grounded in the current image's feature scale. We then apply an Adaptive Top-$k$ rule: $k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)$, aggregating only the top-$k$ most supportive activated patches.

### 3.3 VASM and the Dual-Condition Entropy Gate

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. This penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$

VASM prevents structural collapse via a dual-tier protection architecture:
1. **Root Token Expectation:** $\gamma=1$ *only* for an unambiguous dictionary of visual object nouns. Polysemy (e.g., "apple" the fruit vs. the company) is handled via a brute-force superset. Out-of-dictionary terms default to $\gamma=0$, guaranteeing structural syntax safety. 
2. **BPE Continuation Inheritance (Dynamic):** Any subword token marked as a continuation fragment inherits the exact $\gamma$ weight of its root token. Raw byte fallbacks inherit the $\gamma$ of the immediately preceding structural subword.

**The Dual-Condition Entropy Gate:** 
Modern LLMs suffer from the "arrogance of language priors," often hallucinating objects with extreme confidence. Therefore, a static confidence threshold creates a paradoxical blindspot. We correct this via a dual-condition trigger:
*   **Condition 1 (Confronting Arrogance):** If the model exhibits *extreme* confidence ($Entropy < H_{low}$) AND the candidate token is a visual noun ($\gamma=1$), intervention is **forced**. High confidence on a visual noun *invites* visual scrutiny.
*   **Condition 2 (Uncertainty Support):** If the model generation falls into a borderline uncertainty band ($H_{low} \le Entropy \le H_{high}$), intervention is triggered to provide local visual guidance.

Crucially, $H_{low}$ and $H_{high}$ are not static magic numbers. They are dynamically calibrated as the 10th and 90th percentiles of the base model's confidence distribution over a held-out validation set. If the token is non-visual ($\gamma=0$) and highly confident, intervention is safely bypassed, preserving batched latency. Candidate sets are strictly bounded to $\mathcal{C}_t = \text{Top-}M(L_{orig})$ where $M \ll |V|$.

## 4. Evaluation Protocol and Experimental Design

Because BRA introduces specific structural hypotheses—and risks pipeline bloat if unverified—we organize the evaluation plan into predefined falsifiable evidence chains and strict defense lines. The protocol utilizes **LLaVA-1.5** (MLP projector, 1D sequence) and **Qwen-VL-Chat** (2D-RoPE) to test architectural variance. All experiments outline our planned, hypothesis-driven execution.

### 4.1 Chain A: Hallucination Reduction via Step-by-Step Attribution

**Hypothesis:** BRA mitigates object hallucination through localized evidence reweighting. These gains derive from the decode-time intervention mechanism, not mere parameter exposure, and each pipeline component contributes uniquely.
- **Evaluation Protocol:**
  - **Metrics:** POPE (Acc/F1), CHAIR (explicitly reporting **Object vs. Attribute** separately, as local evidence should predominantly fix object presence), and AGL (Attention-Grounded Localization, IoU). Variance and confidence intervals will be strictly reported to validate statistical significance.
  - **Planned Output (Table 1):** Rows evaluated: Base, VCD, OPERA, DoLa, and the critical control `Base + 5k LoRA` (fine-tuned on the identical 5k SAM-filtered pairs to isolate representation learning from inference-time adjustment).
  - **Mandatory Step-by-Step Ablation:** To prove pipeline necessity, we will report a sequential build-up: `Base` $\rightarrow$ `+ BRA_zero (MeanPool)` $\rightarrow$ `+ Adaptive Top-k` $\rightarrow$ `+ VASM` $\rightarrow$ `+ Entropy Gate (BRA_Full)`. Any component failing to provide statistically significant improvements will be downgraded in our final claims.

### 4.2 Chain B: Structure Preservation and VASM Bypass Tracking

**Hypothesis:** Hallucination mitigation via visual suppression does not degrade complex reasoning, provided VASM actively protects structural tokens. However, dependence on a visual dictionary introduces a quantifiable domain limitation where OOV visual nouns simply bypass intervention.
- **Evaluation Protocol:**
  - **Benchmarks:** MMBench V1.1, MME, MMMU (Hard Subset).
  - **OOV Exhaustion Tracking (Figure 1):** A scatterplot with axes perfectly defined: **X-axis = % of ground truth visual nouns missing from VASM (OOV rate)**, and **Y-axis = MMMU accuracy drop relative to Base**. The Base model performance forms a horizontal baseline, tracking structural degradation directly tied to dictionary exhaustion.
  - **Quantifying VASM Bypass:** For MMMU and DocVQA, we will explicitly report the percentage of true visual nouns that bypassed intervention ($\gamma=0$) merely because they were out of our WordNet dictionary.
  - **Top-$M$ Unrecoverability Defense Line:** We track precisely how many target visual nouns were natively ranked outside the top-$M$ candidate window by the base model, establishing the mathematical ceiling for intervention success.

### 4.3 Chain C: Local Evidence Value & Domain Shift Vulnerability

**Hypothesis:** On dense spatial tasks, token-local visual reweighting geometrically isolates evidence that diffuse pooling natively washes out.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (OCR/Position) and DocVQA (ANLS).
  - **Planned Output (Table 2):** Compare `BRA_MeanPool` (naive global averaging) against `BRA_zero` (Adaptive Top-$k$). This is the ultimate crucible: if `BRA_zero` fails to beat `BRA_MeanPool`, the core claim of "strictly token-local evidence" is falsified.
- **The Limits of Intervention (Figure 2 Failure Analysis):** A 4-panel heatmap failure analysis plotting: (a) Original Image, (b) `BRA_calib` correctly localizing a target via the Entropy Gate forcing intervention on a confident visual noun, and **(c) A required VASM failure subplot**, showcasing a polysemy false positive (e.g., falsely suppressing the financial term "bank" because it matched a riverbank definition) actively destroying a structurally critical token.

### 4.4 Defense Lines: Separated Latency Profiling and OOM Bounds

**Hypothesis:** The latency penalty can be managed via candidate windows without sacrificing hallucination mitigation, though batched scaling remains a severe constraint.
- **Evaluation Protocol (Figure 3):** Executed strictly on a standardized single A100 (80GB) GPU. We plot **Tokens/Sec** (left y-axis) and **Peak VRAM Usage** (right y-axis) against the Candidate Window Size $M \in \{10, 50, 100, 500\}$, evaluated across **Batch Sizes = 1, 4, and 8**.
- **Crucial Separation:** We will strictly separate **Prefill Latency** from **Decode Latency**. Because BRA intervenes entirely during decode, blending the two artificially masks the true computational cost of moving-median pooling and dynamic BPE lookups. If BRA triggers Batched Out-of-Memory (OOM) failures at BS=8, it will be proudly and transparently bounded as an offline analytical tool rather than a real-time production API.

## 5. Discussion and Limitations

The strength of Bounded Region Alignment (BRA) lies in its structurally bounded proposition: intervening at decode-time using strictly token-local visual evidence while mathematically protecting language integrity via VASM. However, its execution is governed by severe theoretical and operational constraints:

1. **The Post-Hoc Entanglement Gamble:** The assumption that localized visual support survives deep self-attention in final layers is precarious. If the defined feature-level Washout Threshold is crossed (average spatial alignment IoU < 0.15), the premise of extracting zero-shot local states must rely entirely on the $\Phi_{calib}$ track, shifting the method from pure zero-shot inference to hybrid alignment.
2. **VASM Domain Dependency and Intervention Bypass:** A brute-force WordNet superset is exceptionally brittle for domain shifts. In specialized tasks like DocVQA or MMMU, if a critical visual noun is OOV, VASM defaults to $\gamma=0$, rendering BRA completely inactive. Our "structure preservation" in niche domains inherently conflates with "intervention bypass." Furthermore, polysemy false positives remain an unresolvable vulnerability of any dictionary-based mask.
3. **Pipeline Bloat vs. Attribution:** Deploying an MLP calibrator, moving-median Top-$k$ pooling, VASM, and a Dual-Condition Entropy Gate introduces significant complexity. Only the strict adherence to the proposed step-by-step ablation protocol will determine if all components are strictly necessary or merely compounding heuristics.
4. **Decode-Time Latency Traps:** Dynamic BPE mask lookups and step-wise spatial pooling inherently bottleneck generation throughput. Isolating decode-time latency scaling at larger batch sizes (e.g., BS=8) severely risks VRAM OOM, likely confining BRA to high-precision offline applications.

## 6. Conclusion and Future Execution

This paper outlines Bounded Region Alignment (BRA), a rigorously defined framework for mitigating MLLM hallucination through structure-preserving, token-local logits intervention. By mapping embedding asymmetry (`BRA_zero` vs `BRA_calib`) via direct feature-level thresholds, utilizing threshold-gated Adaptive Top-$k$ pooling (anchored by prefix-bounded medians), engineering robust language protection via VASM BPE inheritance, and enforcing visual scrutiny on arrogant priors via a Dual-Condition Entropy Gate, we establish a technically precise foundation for decode-time visual control. The claims herein are strictly governed by our hypothesis-driven evaluation protocol spanning Hallucination, Structure, and Local Evidence chains—mandated by step-by-step ablations—establishing a verifiable roadmap for empirical execution.

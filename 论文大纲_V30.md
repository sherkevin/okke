# Bounded Resonance Anchoring: Token-Local Visual Evidence for Structure-Preserving MLLM Decoding

## Abstract

Multimodal Large Language Models (MLLMs) remain vulnerable to hallucination when strong language priors overshadow the localized visual evidence required for dense grounding. While existing inference-time mitigation strategies (e.g., contrastive decoding, attention pattern penalties) serve as highly competitive, orthogonal regularizers, they lack a mechanism to inject strictly **token-local visual evidence** directly into the terminal logits space while preserving complex language structure. We present **Bounded Resonance Anchoring (BRA)**, a decode-time intervention framework that reweights candidate tokens using localized visual support extracted from contextualized final-layer hidden states. 

BRA operates on three core mechanisms. First, **token-local logits intervention** maps final-layer visual states into the LLM's latent dimension to evaluate candidate-wise spatial support. Second, **Threshold-Gated Adaptive Top-$k$ Pooling** extracts stable local evidence while allowing irrelevant background patches to mathematically abstain, explicitly avoiding the dilution of global averaging. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via a WordNet-lemmatized dictionary and dynamic Byte-Pair Encoding (BPE) continuation inheritance. To rigorously handle input-output embedding asymmetry, we establish a strict evaluation boundary separating a zero-shot probe (`BRA_zero`) from a lightweight calibrated bound (`BRA_calib`). 

To validate BRA, we propose a pre-registered, hypothesis-driven evaluation protocol restricted to 2D spatial grids, explicitly guaranteeing zero data leakage. Evaluated across three definitive evidence chains—(A) Hallucination Reduction (benchmarked against equivalent-parameter LoRA baselines), (B) Structure Preservation, and (C) Local Evidence Value—alongside rigorous throughput and failure-mode analyses, this framework establishes a scientifically sound, structurally protected proposition for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable zero-shot capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than global.

Current inference-time mitigation strategies typically intervene via contrastive decoding (e.g., VCD), attention-based pattern penalty (e.g., OPERA), or layer-contrast decoding (e.g., DoLa). These families represent distinct, orthogonal regularizers that effectively discipline the language-side or attention-side generation process. However, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, defensible hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce hallucination more faithfully than pure language/attention heuristics, provided it incorporates explicit structure-preserving safeguards. We term this framework **Bounded Resonance Anchoring (BRA)**. In this work, we explicitly bound our scope to Image-LLMs and 2D spatial reasoning, avoiding unsupported generalizations to spatiotemporal video domains.

Executing BRA requires overcoming three structural barriers. First, **input-output embedding asymmetry**: raw visual tokens reside in the input space, rendering direct dot-product similarity with the terminal output vocabulary matrix invalid. Second, **post-hoc entanglement**: leveraging the final-layer hidden states of visual patches requires overcoming deep attention mixing, which risks spatial washout in 1D-sequence architectures. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, functional syntax, or multi-token entities.

BRA resolves these barriers via three mechanisms:
1. **Token-local logits intervention**, mapping final-layer visual states to the LLM's pre-vocabulary dimension to evaluate localized patch support.
2. **Threshold-Gated Adaptive Top-$k$ pooling**, extracting localized spatial support via abstention-aware patch activations. This mathematically prevents background patches from casting forced votes and avoids the spatial dilution of global average pooling.
3. **VASM (Vocabulary-Anchored Semantic Masking)**, a deterministic, tokenizer-agnostic protection layer utilizing WordNet-augmented lookup and dynamic BPE continuation inheritance to shield structural language.

To ensure rigorous scientific boundaries, we deploy a zero-shot probe (`BRA_zero`) to test native alignment, backed by a lightweight calibrated variant (`BRA_calib`) to overcome spatial washout. We execute our experimental design under a stringent protocol encompassing three primary evidence chains: 
- **Chain A (Hallucination Reduction):** Validating mitigation on POPE/CHAIR alongside Average Generated Length (AGL). Crucially, this includes a strict zero-data-leakage guarantee and comparisons against a parameter-matched `Base + 5k LoRA` baseline to isolate decode-time gains.
- **Chain B (Structure and Reasoning Preservation):** Proving the necessity of VASM and BPE Inheritance on MMBench/MMMU(Hard) against unconstrained ablations.
- **Chain C (Local Evidence Value):** Demonstrating the mathematical superiority of localized evidence on FREAK/DocVQA against functionally identical `BRA_MeanPool` baselines, supported by geometric attention mapping.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Overcoming Post-Hoc Entanglement: `BRA_zero` vs `BRA_calib`

A fundamental vulnerability in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, BRA evaluates the *contextualized* final-layer transformer hidden states of the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$. 

We explicitly acknowledge the gamble of **post-hoc entanglement**: deep self-attention implicitly mixes visual states with prefix context. To handle vocabulary projection cleanly without falsely claiming zero-shot alignment if the architecture prohibits it, we define two distinct tracks:

1. **`BRA_zero`**: A strict zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$.
2. **`BRA_calib`**: If `BRA_zero` fails due to spatial washout (e.g., in 1D-sequence models), we explicitly pivot to a lightweight calibrated decoding method. $\Phi_{calib}: \mathbb{R}^{D_{vision}} \rightarrow \mathbb{R}^{D_{llm}}$ is a linear layer (or lightweight MLP) projecting the visual state to the LLM's final hidden dimension *prior* to $W_{vocab}$. It is trained on a frozen set of 5,000 COCO image-patch to bounding-box label pairs. **Crucially, the base LLM remains entirely frozen, and these 5,000 images are guaranteed strictly disjoint from all downstream evaluation sets.**

We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. To resolve multi-token label ambiguity, we initially define the positive target $W_{vocab}[c^+]$ as the L2-normalized mean of the subword embeddings comprising the ground-truth label $T$. **Semantic Fallback Plan**: Because terminal subword embeddings heavily encode syntactic priors, simple averaging may fail to yield a semantically coherent target. If training diverges, we automatically fallback to extracting $W_{vocab}[c^+]$ from the contextualized hidden state of the final subword after passing the full word through the frozen LLM.

Negative samples ($W_{vocab}[c^-]$) are drawn from intra-image patches with an Intersection over Union (IoU) $< 0.1$ with the target bounding box. To prevent penalizing valid overlapping objects in dense scenes, we enforce a **semantic negative constraint**: negative patches are further filtered to ensure their ground-truth labels do not share the exact semantic class as the target. If an image is so dense that *no* negative patches meet these criteria, we fallback to cross-image patch sampling within the training batch.

### 3.2 Threshold-Gated Adaptive Top-$k$ Pooling

At decoding step $t$, BRA restricts intervention to a bounded candidate set $\mathcal{C}_t = \text{Top-}M(L_{orig})$ where $M \ll |V|$ (e.g., $M=100$). A naive Softmax over $\mathcal{C}_t$ would force every visual patch to sum to 1.0, artificially inflating the voting power of irrelevant background patches. 

To prevent noise injection, we apply a threshold-gated Sigmoid activation, allowing patches with low semantic overlap to mathematically abstain:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{noise}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{noise}\Big)
$$
To prevent small objects from being washed out by background noise—a known failure of global average pooling—we apply an Adaptive Top-$k$ rule: $k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)$. The raw localized support for candidate $c$ aggregates only its top-$k$ most supportive activated patches:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k} \tilde{P}^{(v_j)}(c)
$$
This is then normalized across the candidate set: $\hat S(c) = \text{Softmax}_{c \in \mathcal{C}_t}(S_{raw}(c) / \tau_{norm})$.

### 3.3 VASM (Vocabulary-Anchored Semantic Masking)

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. Crucially, this penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$
Where $\alpha = \alpha_0 \cdot \mathcal{H}(\text{Softmax}(L_{orig}(\mathcal{C}_t)))$.

VASM deterministically prevents structural collapse via a dual-tier architecture:
1. **Root Token Expectation (WordNet-Augmented Visual Nouns):** $\gamma=1$ *only* for an unambiguous dictionary of visual object nouns (initialized via COCO/LVIS, expanded via WordNet synsets). Out-of-dictionary terms default to $\gamma=0$, guaranteeing structural safety over absolute object coverage. By strictly limiting this to *nouns*, VASM consciously cedes attribute (e.g., "red") and relational (e.g., "running") hallucination mitigation to the base model's priors, optimizing for object-level structural safety.
2. **BPE Continuation Inheritance (Dynamic):** Any subword token marked as a continuation fragment automatically bypasses the dictionary and inherits the exact $\gamma$ weight of its root token. Designed to be robust across tokenizers (e.g., Llama's SentencePiece `_`, Qwen's Tiktoken), this ensures words like "rhinoceros" (`_rhi`, `no`, `cer`, `os`) receive synchronized logit adjustment, absolutely preventing multi-token fragmentation.

## 4. Evaluation Protocol and Experimental Design

Because BRA introduces a specific structural hypothesis regarding decode-time local evidence, we organize the evaluation plan into predefined falsifiable evidence chains. The protocol mandates explicit architecture specification, utilizing **LLaVA-1.5** (MLP projector, 1D sequence) and **Qwen-VL-Chat** (2D-RoPE). 

### 4.1 Chain A: Hallucination Reduction vs. Length Collapse

**Hypothesis:** BRA mitigates hallucination through localized evidence reweighting without causing pathological output truncation, and these gains are derived from decode-time intervention, not just parameter exposure.
- **Zero Data Leakage Guarantee:** The 5,000 COCO images used to train $\Phi_{calib}$ are strictly verified via MD5/ID hashes to have zero intersection with POPE, CHAIR, and downstream sets.
- **Planned Execution (Table 1):**
  - **Metrics:** POPE (Acc/F1) | CHAIRi | CHAIRs | **AGL (Average Generated Length)**
  - **Zero-Shot Baselines:** Base, VCD, OPERA, DoLa, `BRA_zero`.
  - **Calibrated Baselines:** `Base + 5k LoRA` (fine-tuned on the exact same 5k pairs, targeting the LLM's Q/V projections with a parameter budget $\ge \Phi_{calib}$), `BRA_calib`.
- **Success Criteria:** `BRA_calib` must outperform both the zero-shot baselines and the `Base + 5k LoRA` baseline. This ensures the performance fundamentally stems from the *decode-time local mechanism*. AGL must not drop by $>10\%$.

### 4.2 Chain B: Structure and Reasoning Preservation

**Hypothesis:** Hallucination mitigation via visual suppression does not degrade complex reasoning, provided VASM actively protects structural tokens and BPE Inheritance protects multi-token entities.
- **Planned Execution (Table 2):**
  - **Benchmarks:** MMBench V1.1 | MME | MMMU (Hard Subset)
  - **Ablations:** Base | `BRA_no_VASM` (applies penalty to all tokens) | `BRA_no_BPE_Inherit` | `BRA_full`
- **OOV Tracking:** We explicitly report the Out-Of-Vocabulary (OOV) rate—the exact percentage of benchmark ground-truth targets where VASM defaulted to $\gamma=0$ due to dictionary exhaustion.
- **Success Criteria:** Unconstrained suppression (`BRA_no_VASM`) must severely degrade MMMU(Hard) reasoning. `BRA_full` must match or strictly preserve Base model performance.

### 4.3 Chain C: The Necessity of Local Evidence

**Hypothesis:** On dense spatial tasks, token-local visual reweighting is significantly more faithful than global visual aggregation.
- **Planned Execution (Table 3):**
  - **Benchmarks:** FREAK (OCR/Position) | DocVQA (ANLS)
  - **Ablations:** Base | `BRA_MeanPool` | `BRA_AdaptiveTopK`
- **Execution Rule:** `BRA_MeanPool` *must* use the exact same $\Phi_{calib}$ and VASM mask as `BRA_AdaptiveTopK`. The *only* difference is pooling logic.
- **Qualitative Protocol (Figure 1 - Heatmaps):** We mandate a 3-way side-by-side DocVQA failure analysis plotting: (1) The base model's *native cross/self-attention map* for the target token, (2) The `BRA_MeanPool` 2D grid, and (3) The `BRA_AdaptiveTopK` 2D grid. We hypothesize `AdaptiveTopK` will form a sharp spike strictly over the text bounding box, geometrically proving it supplies *new* localized evidence that native attention failed to isolate.

### 4.4 Defense Line 4: System Throughput and Latency Penalty

**Hypothesis:** Decode-time intervention imposes measurable but acceptable overhead.
- **Planned Execution:** We bring latency benchmarks into the main text, reporting **A100 Tokens/Sec** throughput for Base, VCD, and BRA. Because BRA requires running Top-$k$ pooling, thresholding, and dynamic BPE mask lookups at *every single generation step*, we will transparently report the exact throughput penalty to validate system viability.

### 4.5 Defense Line 5: Hyperparameter Limits and Failure Modes

- **$\Phi_{calib}$ Capacity Ablation:** We test whether a 2-layer MLP for $\Phi_{calib}$ is mathematically required to un-mix the deep attention entanglement, or if the proposed linear layer is genuinely sufficient.
- **Out-of-Candidate Limit (Figure 2):** We explicitly document the absolute structural ceiling of BRA. We mandate a qualitative visual example showing an "Out-of-Candidate" failure: an image where the correct visual object exists, but the base model's strong language prior pushes the correct token to rank $>M$ (e.g., rank 150 outside $M=100$). Because BRA mathematically reweights the initial distribution, it cannot rescue tokens that the base model completely suppresses.

## 5. Discussion and Limitations

The strength of BRA lies in its falsifiable, structurally bounded framework: intervening at decode-time using strictly token-local visual evidence, avoiding global pooling dilution, and protecting language integrity via VASM. However, its execution carries specific theoretical constraints:

1. **Parametric Asymmetry and Calibration:** `BRA_calib` requires a lightweight, 5k-image frozen projector to overcome spatial washout in 1D-sequence models. While we mitigate this asymmetry by comparing against an equivalent parameter-matched `Base + 5k LoRA` baseline, it fundamentally separates BRA from purely training-free heuristics.
2. **VASM's Noun-Only Limitation:** By relying on a visual object noun dictionary, VASM is structurally blind to attribute (e.g., color, size) and relational/action hallucinations (e.g., "running", "red"). This is a conscious tradeoff, ceding attribute-level correction to the base model to guarantee absolute structural safety.
3. **Out-of-Candidate Unrecoverability:** BRA operates by reweighting the LLM's initial candidate distribution (Top-$M$). It mathematically cannot synthesize a visually grounded token if the base model completely suppresses it from the initial candidate window, establishing a hard theoretical ceiling on post-hoc logits adjustment.
4. **Latency Overhead:** Intervening at every autoregressive decoding step incurs computational overhead, heavily tying BRA's efficiency to optimized dynamic masking operations.

## 6. Conclusion and Future Execution

This paper outlines **Bounded Resonance Anchoring**, a rigorously bounded framework for mitigating MLLM hallucination through structure-preserving, token-local logits intervention. By mapping embedding asymmetry risks, correcting patch-magnitude dominance via threshold-gated Top-$k$ pooling, and securing language syntax via VASM BPE inheritance, we establish a robust foundation for decode-time visual control. The claims herein are strictly governed by our pre-registered evaluation protocol spanning Hallucination, Structure, and Local Evidence chains, awaiting full empirical execution.

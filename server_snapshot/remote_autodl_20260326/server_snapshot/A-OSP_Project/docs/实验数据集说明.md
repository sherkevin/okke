# 实验数据集说明（对齐《实验大纲.md》V3.5）

**统一入口脚本：** `code/data/download_experiment_datasets.py`

---

## 1. 下载原则（Mini-Batch First）

| 规则 | 说明 |
|------|------|
| 默认先冒烟 | `python code/data/download_experiment_datasets.py --mini_test [--n_samples 10]` |
| 全量需授权 | `python code/data/download_experiment_datasets.py --full`（体积大、耗时长） |

## 3. 命令速查

```bash
cd /path/to/A-OSP_Project

# 推荐：全大纲相关集 — 冒烟（每类约 10 条量级，具体见脚本内 cap）
python code/data/download_experiment_datasets.py --mini_test --n_samples 10

# 跳过 CHAIR（避免拉 COCO zip / GitHub 超时）
python code/data/download_experiment_datasets.py --mini_test --skip_chair

# VPN + 代理：并行全量（推荐，总墙钟时间更短）
# 先 export http_proxy/https_proxy=http://127.0.0.1:7890 ，可选 unset HF_ENDPOINT
bash code/data/run_parallel_with_proxy.sh
# 进度：tail -f logs/parallel_orchestrator.log
# 各任务：logs/parallel_<core|crossdomain|textvqa|...>.log
# 结束后自动生成 logs/dataset_download_report.md

# 仅跑子模块示例
python code/data/download_core_benchmarks.py --mini_test
python code/data/download_crossdomain_datasets.py --mini_test

# CHAIR 工具链
bash code/data/setup_chair.sh

# MIRAGE 占位（需本地 COCO val 图）
python code/data/create_mirage_stub.py
```

## 4. 已知限制与排错

1. **`HF_TOKEN`**：未登录时 Hub 限速；大库建议 `huggingface-cli login`。  
2. **GitHub 超时**：AutoDL 等环境 clone GitHub 可能失败；`setup_chair.sh` 已支持 **raw 直链** 回退。  
3. **IU X-Ray 图像**：`download_crossdomain_datasets.py` 对 `dz-osamu/IU-Xray` 使用 **`val.jsonl` + `image.zip` 按需解压**（首次会缓存约 1.1GB zip），将 JSON 中的 `/iu_xray/image/CXR…/N.png` 映射到 zip 内 `…/images/CXR…/N.png`。  
4. **TextVQA**：`download_experiment_datasets.py` 通过子进程 **`code/data/download_textvqa_stream.py`** 以 **streaming** 拉取前 N 条，避免整表 `load_dataset` 长时间阻塞；进程结束使用 `os._exit(0)` 规避部分环境下 datasets/pyarrow 析构时的 **PyGILState_Release** 崩溃。失败时回退 **ChartQA/COCO 占位 stub**。  
5. **MMMU**：`MMMU/MMMU` 各子域使用 **`image_1`…`image_7`** 字段（非单一 `image`），统一下载脚本已按序取首张可用图写入 `data/benchmarks/mmmu/images/`。  
6. **MMBench 图像**：`opencompass/MMBench` 若不可用会自动尝试 **`HuggingFaceM4/MMBench`**；其中 `image` 常为 **Base64 字符串**，`download_core_benchmarks.py` 会解码写入 `images/`。  
7. **RefCOCO**：全量 `val` 很大；脚本默认 **流式** 取前 **N** 条，避免一次性占满磁盘。

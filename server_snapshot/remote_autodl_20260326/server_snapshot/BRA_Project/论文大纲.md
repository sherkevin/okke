# 1. Introduction

近年来，多模态大语言模型（Multimodal Large Language Models, MLLMs）在图像理解、视觉推理和视频问答等复杂多媒体任务中展现出前所未有的潜力。然而，尽管具备强大的跨模态泛化能力，最先进的 MLLMs 仍饱受“幻觉（Hallucination）”问题的困扰——模型极易生成与给定视觉上下文中不存在的实体或动作相矛盾的似是而非的文本。在对高保真度要求极高的多媒体应用（如医疗影像分析、自动驾驶场景理解等）中，这种跨模态不对齐带来的认知崩塌是阻碍 MLLMs 实际部署的核心瓶颈。

为了缓解这一问题，近期的表征干预（Representation Intervention）与解码期修正方法大多遵循一种**“减法范式（Subtractive Paradigm）”**——试图在模型的深层隐藏状态中提取并剔除过强的语言先验动量。然而，我们敏锐地观察到，此类范式在底层架构与表征几何层面面临两个不可调和的数学矛盾。**其一，空间维度的灾难性坍塌：** 现有方法不可避免地依赖全局空间聚合（Global Spatial Aggregation，如 MeanPool）来对齐跨模态特征，这种操作必然导致极其严重的信息降维，彻底丢失由 2D-RoPE 等机制编码的高分辨率局部坐标与细粒度视觉线索，使得模型对高分辨率多媒体内容（如复杂网页 UI、密集人群）的感知能力大幅退化。**其二，不可逆的语言模型退化（Language Model Degradation）：** 在隐空间执行无差别的向量减法，极易破坏大语言模型原生流形中控制语法流利度的结构子空间，导致输出文本的连贯性与逻辑性遭受严重破坏。

鉴于上述固有局限性，本文提出一次深刻的范式转换：**我们将 MLLMs 幻觉的物理本质重新界定为“候选词元与深层视觉语义场的共振失效（Resonance Failure）”。** 基于现代大语言模型表征几何的最新进展，我们发现解嵌矩阵（Unembedding Matrix）$W_{vocab}$ 所在的流形呈现出高度清晰的**拓扑解耦（Topological Decoupling）**：负责马尔可夫平滑转移的结构子空间（如包含 "is", "the" 等功能词）与由视觉特征主导的感知子空间（如实体与动作词）呈现内禀的拓扑正交性。这一原生词表流形的拓扑结构，为本研究提供了一个连续、可微且零参数开销的天然语义滤波器（Semantic Filter）。

基于上述数学洞见，本文提出**双向共振锚定（Bi-directional Resonance Anchoring, BRA）**框架。有别于在隐状态进行有损操作的传统方法，BRA 框架彻底摒弃了对深层视觉特征的降维操作，转而在 Logits 解码空间直接执行特征级的正向与逆向证据注入。具体而言，我们在原始高维度量空间中引入零样本视觉-词汇共振度量（Zero-Shot Visual-Vocabulary Resonance），通过最大池化匹配机制，让候选词向量主动遍历保持完整分辨率的视觉特征网格。当模型的认知不确定性触发阈值时，BRA 会根据共振分数执行双向 Logits 重塑（Bi-directional Logits Reshaping）。这一机制不仅能对缺乏视觉证据的幻觉词施加强制抑制，更利用拓扑正交性在数学层面保证了语法功能词的绝对无损。

## 2. Related Work
多模态大语言模型（MLLMs）将视觉编码器的感知能力与大语言模型的认知推理能力相融合，在视觉问答、图像描述等任务上取得了显著进展$^{[1]}$。然而，物体幻觉——模型过度自信地生成视觉证据中不存在的物体或属性——是阻碍其走向高可靠部署的持续性挑战。本节梳理与 A-OSP 直接相关的三类背景工作：多模态幻觉的成因与评估、推理期干预方法的系统级局限，以及表征工程领域的激活引导技术。

### 2.1 多模态大语言模型中的物体幻觉
多模态架构的发展轨迹已从面向特定任务的判别式框架，快速演进为通用的生成式大视觉语言模型（LVLMs）。前沿的基础模型通过强行对齐视觉编码器（如ViT、CLIP）的连续表示空间与大语言模型的离散自回归嵌入空间，成功搭建起跨模态的桥梁$^{[2]}$。典型的模型架构如LLaVA$^{[34]}$，采用直接且轻量化的多层感知器（MLP）投影矩阵，将视觉补丁映射至语言模型的输入空间，通过大规模指令调优实现了优异的零样本对话能力。InstructBLIP$^{[7]}$引入了具备指令感知能力的Q-Former，提取高度压缩、由查询条件化的视觉特征，其作用相当于信息瓶颈，能够过滤无关的视觉噪声。此外，Qwen-VL$^{[1]}$系列开创了复杂视觉接收器的设计，可处理交错的图文输入和显式的边界框坐标，实现细粒度的空间定位。其最新迭代 Qwen3-VL 家族（2B/4B/8B/32B）进一步引入了原生动态分辨率（Native Dynamic Resolution）与 2D-RoPE 位置编码，覆盖了从边缘端到数据中心级的全参数尺度，已成为当前多模态开源生态中架构最完备的基座模型系列。尽管多模态架构取得了上述突破，但所有多模态大语言模型均普遍存在严重的物体幻觉问题——生成的文本虽语言流畅、句法连贯，却会高置信度地描述视觉证据中完全不存在的物体、属性或空间关系$^{[18]}$。

多模态幻觉的成因具有高度的多面性，根源在于预训练范式与跨模态对齐机制之间的根本性矛盾。首先，视觉与语言模态存在固有的语义错位。视觉编码器通常通过对比损失（如InfoNCE）训练，以捕捉全局、整体的语义表示，这一过程会天然舍弃密集视觉推理所需的细粒度局部空间细节$^{[3]}$。当这些全局嵌入被注入大语言模型时，模型缺乏验证局部描述所需的空间粒度。其次，多模态指令调优数据的长尾分布引入了严重的统计偏置。训练语料高度偏向典型场景中的常见物体，导致模型依赖启发式的共现概率（如在“键盘”旁预测存在“鼠标”），而非真正的视觉定位$^{[29]}$。

然而，最关键的诱因是底层大语言模型骨干网络中固有的“语言惯性”（即语言先验）占据显著的主导地位$^{[30]}$。由于大语言模型在数万亿纯文本标记上完成预训练，其内部的转移概率更倾向于结构可预测的语言补全结果。当视觉信号模糊、退化或嵌入复杂场景时，自回归的生成轨迹会不可避免地陷入这种无条件的语言子空间。模型会直接忽略微弱的视觉证据，编造视觉上看似合理但与事实不符的输出，以满足其内部自回归似然最大化的目标$^{[9]}$。

主流评估基准涵盖三类范式：以 POPE$^{[33]}$、CHAIR$^{[21]}$ 为代表的判别式/生成式幻觉探测基准，以 MMHal-Bench$^{[46]}$ 为代表的开放式生成质量评估，以及以 MMMU$^{[54]}$ 为代表的跨学科推理基准。我们的实验优先覆盖前两类，以直接量化幻觉抑制效果与生成质量的帕累托权衡（§4）。

POPE（基于投票的物体探测评估）这类判别式基准在随机、常见、对抗三种数据划分下，通过二元“是/否”问题揭示模型对数据集先验知识的依赖$^{[41]}$。其中，对抗性划分的探测效果尤为显著，可针对性检验多模态大语言模型是否会因某一物体与场景中真实物体的高频共现，就幻觉出该物体的存在。CHAIR（结合图像相关性的描述幻觉评估）这类生成式基准，则通过与真实分割标注进行严格的词汇匹配，计算开放式图像描述任务中的单实例幻觉率和单句幻觉率$^{[42]}$。随着多模态大语言模型的发展，评估方式也必然向复杂的综合性推理方向转变。MMBench引入了循环评估策略，对模型在置换后的多项选择中表现出的逻辑不一致性进行惩罚，以此识别那些基于格式猜测答案而非真正理解视觉内容的模型$^{[35]}$。MMMU则以大学专业级的跨学科推理任务对模型提出挑战，要求模型严格遵循复杂的视觉图表、示意图和医学影像完成推理$^{[47]}$。此外，MMHal-Bench融合了基于事实增强的人类反馈强化学习（RLHF）评估，能动态惩罚开放式多类别推理中的错误$^{[40]}$；而MIRAGE则通过确保模型能正确感知图像、但在后续逻辑推演中出现错误的方式，分离出推理诱导的幻觉$^{[43]}$。

尽管当前的评估体系已发展得极为成熟，但针对幻觉的拓扑学本质——即幻觉如何表现为深度跨模态特征流形中的几何畸变——的研究仍极为匮乏。现有文献大多将幻觉视为表层的统计误差，仅衡量模型的输出结果，却未探究隐空间的畸变方式。近期，有一批新兴研究开始从几何视角探究幻觉问题。Marín（2026）建立了大语言模型幻觉的几何分类体系，通过语义定位指数（SGI）测量单位超球面上的语义内聚性和角度偏移，将模型失效分为不忠实、虚构和事实错误三类$^{[37]}$。Liu等人（2026）将该研究延伸至多模态领域，其关于“几何盲”的研究发现，多模态大语言模型无法将显式识别的视觉特征映射为连贯的几何流形，导致拓扑上下文发生变化时产生幻觉$^{[36]}$。此外，对深度特征流形的谱分析表明，幻觉响应会系统性地偏离“真实流形”，在多模态图拉普拉斯算子中表现出可量化的能量畸变$^{[16]}$。

综上，当前的评估范式虽能细致记录幻觉的表层症状，但针对深度特征流形中这类失效的几何与拓扑本质的研究仍存在显著空白。而这一核心的理论空白，正是我们提出的自适应正交子空间投影（A-OSP）框架的研究目标。该框架不再将幻觉仅视为解码错误，而是将其认定为跨模态激活被无条件语言子空间几何捕获的结果，通过直接作用于表示流形，从数学根源上消除这些语义畸变。

## 2.2 多模态大语言模型的推理时干预
通过参数更新缓解幻觉问题（如在负样本上进行有监督微调（SFT）或基于人类反馈的强化学习（RLHF））需要极高的计算资源、大量的配对标注，且始终存在模型通用能力灾难性遗忘（Catastrophic Forgetting）的风险——更关键的是，微调无法诱导模型对自身知识边界的认知不确定性（Epistemic Uncertainty），而 A-OSP 的推理期干预在极限场景下反而能唤醒模型"承认不知道"的能力（见 §4.5 COD10K Case Study）$^{[40][49]}$，因此研究界的研究重点日益转向无需训练的推理时干预方法$^{[10]}$。这类策略试图在生成过程中动态调节自回归的生成轨迹，主要通过对比解码（CD）机制、启发式的对数几率惩罚或事后编辑流水线实现。尽管这些方法在概念上简洁优雅，但严谨的性能分析表明，其存在不可忽视的系统级部署局限性。

值得注意的是，2025/2026 年的最新研究仍未跳出上述范式的窠臼。FarSight (CVPR 2025)$^{[20]}$ 尝试通过注意力图的显式重分配来锚定远景目标，但其多尺度注意力操纵本质上依赖束搜索的二次前向传播，延迟开销与 VCD 如出一辙。ONLY (ICCV 2025)$^{[22]}$ 发现单层干预即可缓解幻觉，但其静态干预向量的刚性注入在动态多轮对话中缺乏自适应性。Dynamic Correction Decoding (ICLR 2025)$^{[23]}$ 引入了动态校正解码策略，虽在短文本问答上取得进展，但其基于注意力分数的触发机制在长程生成中同样无法避免平均生成长度的系统性衰减。SCPO (ICLR 2026)$^{[11]}$ 提出语义课程偏好优化，代表了训练期方法的最新进展，但其高昂的微调成本与灾难性遗忘风险使其难以成为即插即用的轻量方案。综上，截至 2026 年，"计算延迟翻倍"与"长度-偏见陷阱"这两大结构性缺陷依然如影随形，未被任何现有方法从根本上解决。

但我们必须明确指出，这类基于对比解码的范式存在严重的效率瓶颈。由于其本质依赖于两个分布的对比，每生成一个标记都需要通过庞大的大语言模型骨干网络完成一次额外的并行前向传播。因此，视觉对比解码、多模态互信息解码、AVISC等方法会带来巨大的计算负担，导致推理延迟严格翻倍（≈2倍延迟开销），并大幅降低整体的标记吞吐量$^{[9]}$。在面向用户、对实时响应性要求极高的智能体实际部署场景中，如此高昂的计算成本让对比解码方法失去实用性，使其仅停留在学术研究层面，无法成为可落地的解决方案。

另一类推理时干预方法则试图通过在单次生成步骤中施加动态对数几率惩罚或注意力操纵，规避双前向传播的开销。DoLa（通过层对比解码）将模型深层成熟层投影的对数几率与浅层未成熟层的对数几率进行动态对比，其核心假设是事实知识集中在Transformer的深层块中$^{[6]}$。该方法通过放大跨层的对数几率差异，抑制早期幻觉的产生，且无需对图像进行二次前向传播。OPERA（过度信任惩罚与回溯分配）发现，幻觉与自注意力矩阵中的“注意力汇”现象密切相关$^{[9]}$。多模态大语言模型往往存在病态的“过度信任”倾向，将注意力权重高度集中在少数已生成的总结标记上，而系统性忽略实际的视觉标记。为解决这一问题，OPERA在束搜索解码过程中对模型对数几率施加缩放惩罚项，并结合激进的回滚策略：当检测到异常的注意力峰值时，强制回滚生成时间线并重新分配标记选择$^{[30]}$。除此之外，Woodpecker等后处理范式则将生成与修正阶段完全解耦$^{[50]}$。该方法在臃肿的多阶段流水线中调用外部基础模型——通过Grounding DINO提取关键概念，利用GPT-4结合边界框验证视觉描述的真实性，最后触发大语言模型对文本进行事后编辑以消除幻觉。尽管Woodpecker在独立测试中效果显著，但该方法存在严重的系统臃肿问题，通过串联多个大规模模型并高度依赖专有API调用，进一步加剧了延迟问题$^{[50]}$。

关键问题在于，基于启发式惩罚的方法（如OPERA、DoLa）和事后编辑方法均隐含着一个严重的方法论缺陷，我们将其定义为“长度偏置陷阱”$^{[11]}$。POPE、CHAIR等幻觉评估指标本质上对生成文本的长度敏感，其中单句幻觉率被定义为包含幻觉物体的句子数与总句子数的比值，这一指标在数学上会偏向简洁的输出。若多模态大语言模型输出简洁的短句，其无提示编造物体的统计概率会降至最低。那些对不确定性施加严厉惩罚或触发激进回滚的解码干预方法，会无意中对模型产生强烈的演化压力，使其提前终止生成过程。这类方法通过迫使模型进入保守的生成状态，人为抬高了安全指标，营造出幻觉缓解的表面假象，却严重牺牲了模型的平均生成长度（AGL）和描述丰富度$^{[12]}$。模型会习得一种退化的启发式策略：“说得越少，就越不容易出错”。平均生成长度的系统性下降，会彻底破坏模型在复杂、详细的图像描述和精细化推理任务中的效用，使得所谓的“性能提升”实则毫无意义。

综上，多模态大语言模型的系统级实时部署，迫切需要一种“帕累托最优”的干预策略——既能精准抑制幻觉，又不会让计算延迟翻倍，也不会陷入具有欺骗性的长度偏置陷阱。我们提出的自适应正交子空间投影（A-OSP）框架恰好满足这一需求：该框架利用数学闭式的子空间投影，仅对异常标记起作用，既能保持极高的吞吐量（仅带来约1.5%的延迟开销），又能在不损失多模态对话丰富度的前提下，保留模型的平均生成长度。

## 2.3 表示工程与激活引导
为规避对比解码带来的严重延迟开销，以及对数几率操纵引发的长度截断偏置，研究者开始深入探究语言模型的内部几何结构，由此催生了“表示工程（RepE）”这一研究方向$^{[59]}$。该方向是由Zou等人（2023）开创的一种自上而下的可解释性与控制框架，彻底摆脱了对单个神经元或孤立电路的微观分析（这类分析往往无法捕捉复杂的涌现行为），转而聚焦于分布式的群体级表示，以此读取并控制高层次的认知概念$^{[59]}$。通过分析高度对比的提示所生成的激活差异（如明确要求诚实与要求欺骗的提示），表示工程推导出“读取向量”，该过程通常利用主成分分析（PCA）在高维激活空间中分离出某一概念的主语义方向。在推理过程中，通过在隐状态中静态添加或减去该读取向量，研究者可实现对模型行为的直接、权重冻结式引导，调节模型的真实性、毒性、盲从性等特征$^{[59]}$。

表示工程的一项直接且极具影响力的衍生成果是推理时干预（ITI），由Li等人（2023）提出$^{[45]}$。该方法并未采用主成分分析，而是系统性地筛选出一部分注意力头，其激活对TruthfulQA基准定义的“真实性”具有高线性探测准确率$^{[45]}$。在确定这些关键注意力头后，推理时干预计算出真实与虚假激活分布之间的质量均值偏移；在自回归生成过程中，沿线性探测识别出的与真实性相关的方向，强行偏移这些特定注意力头的激活。该针对性的干预策略能有效提升大语言模型的事实性，缩小模型内部“知晓”与外在“表达”之间的差距，且无需进行任何计算成本高昂的权重更新$^{[45]}$。

与此同时，概念消除领域也发展出了成熟的数学技术，能严格且可证明地从嵌入空间中移除不受欢迎的偏置。Belrose等人（2023）形式化定义的最小二乘概念消除（LEACE）算法，提供了一种闭式方法，可防止所有线性分类器在神经表示中检测到特定概念（如性别、种族或特定词性偏置）$^{[2]}$。该算法通过计算特征表示$X$与离散概念标签$Z$之间的互协方差矩阵，构建一个仿射变换——具体为最优的斜投影或正交投影。该投影能从每一个中间隐藏层中从数学上清除目标概念的信息，同时对周围表示空间造成的破坏（以L2范数损失量化）降至最低$^{[2]}$。最小二乘概念消除算法的精妙之处在于其闭式保证：无需迭代梯度下降，就能可证明地消除目标概念的线性可分性。

尽管表示工程、推理时干预、最小二乘概念消除在控制单模态纯文本大语言模型方面取得了里程碑式的突破，但将这些线性、静态的方法拓展至多模态领域，会面临严重的理论和实证风险$^{[16]}$。多模态大语言模型的跨模态特征流形，本质上比单模态文本空间更具波动性、异质性和复杂性$^{[36]}$。多模态架构必须持续平衡空间视觉特征的稠密、连续几何属性，与大语言模型词汇表的离散、句法约束$^{[48]}$。

我们观察到，当将现有的表示引导技术（主要为施加静态干预向量或统一的层间正交投影）直接应用于多模态大语言模型时，会系统性地破坏模型脆弱的跨模态平衡$^{[16]}$。由于视觉标记的语义强度会随层、空间补丁和时间推理步骤发生巨大变化，将刚性的干预向量强行注入激活流，必然会引发“激活坍缩”，导致模型的通用常识推理和指令跟随能力迅速退化$^{[36]}$。在实际应用中，这会表现为模型在MMMU、MMBench等综合性知识密集型基准上的性能出现灾难性下降——由于其基础语义流形被静态干预从数学上破坏，模型丧失了基本的逻辑推演能力$^{[47][35]}$。此外，施加静态的概念清除往往会无意中消除关键的上下文依赖，导致模型对必要的几何约束和视觉定位线索视而不见$^{[29]}$。

综上，现有的表示工程和激活引导方法本质上是为静态的纯文本干预设计的；将其强行应用于多模态大语言模型具有波动性的跨模态动态流形，必然会破坏模型的核心推理能力，并引发激活坍缩。而自适应正交子空间投影（A-OSP）框架通过创新的“熵感知动态阈值”和“尺度保持”投影机制，弥补了这一多模态研究的关键空白。与最新的多模态表征引导方法（如 FarSight$^{[20]}$ 的注意力重分配、ONLY$^{[22]}$ 的单层静态干预）相比，A-OSP 的核心差异化优势在于三点：(1) 动态自适应性——条件 EMA 替代了刚性静态向量；(2) 尺度保持——Proposition 1 提供了噪声上界的统计保证；(3) 延迟帕累托——$+1.5\%$ 远低于上述所有方法。具体而言：该框架利用条件指数移动平均（EMA）实时监控感知信息熵和L2投影能量，仅在检测到流形异常时触发正交子空间投影。这一设计确保了语言幻觉被精准消除，同时从数学上保护了多模态流形的整体语义完整性和常识逻辑。


## 3. Methodology

过去旨在缓解多模态大语言模型（MLLMs）幻觉的表征干预方法，大多遵循一种“减法范式（Subtractive Paradigm）”——试图在深层隐藏状态中提取并剔除语言先验动量。然而，此类范式在架构层面面临两个固有的数学矛盾：其一，对跨模态特征进行全局空间聚合（Global Spatial Aggregation，如 MeanPool）必然导致信息降维，从而丢失由 2D-RoPE 编码的高分辨率局部坐标；其二，无差别的隐状态减法操作极易破坏控制语法流利度的结构子空间，引发严重的语言模型退化（Language Model Degradation）。

本文提出一次核心范式转换：我们将幻觉的物理本质重新界定为**“候选词元与深层视觉语义场的共振失效（Resonance Failure）”**。基于此，我们提出双向共振锚定（Bi-directional Resonance Anchoring, BRA）框架。该框架摒弃了对视觉隐状态的有损降维操作，转而在 Logits 解码空间直接执行特征级的正向与逆向证据注入，从而在实现零空间信息损耗（Zero Spatial Information Loss）的前提下完成幻觉纠偏。

### 3.1 Topological Orthogonality in Unembedding Space (解嵌空间的拓扑正交性)

在自回归生成的第 $t$ 步，深层隐藏状态 $h_t \in \mathbb{R}^d$ 通过解嵌矩阵（Unembedding Matrix） $W_{vocab} \in \mathbb{R}^{|V| \times d}$ 映射为词汇表上的 Logits 预测分布。

基于现代大语言模型表征几何的研究，我们观察到在海量无监督预训练的约束下，$W_{vocab}$ 所在的 $d$ 维流形呈现出高度清晰的拓扑解耦（Topological Decoupling）特征：
1. **结构子空间 (Structural Subspace)**：包含如 "the", "and", "is" 等高频功能词。其特征向量主要负责驱动马尔可夫平滑转移，与具体感知语义方向呈现内禀正交性。
2. **感知子空间 (Perceptual Subspace)**：包含具体的实体概念与动作词（如 "cat", "running"）。

由于深层视觉特征 $Z_{vision} = \{z_1, z_2, \dots, z_{N_v}\} \in \mathbb{R}^{N_v \times d}$ 纯粹由视觉编码器注入的具体感知事实构成，其能量天然且严格地分布在感知子空间内。因此，对于任意功能词向量 $w_{\text{func}}$ 与视觉特征 $z_j$，其内积的数学期望自然趋近于零（即 $\mathbb{E}[w_{\text{func}}^T z_j] \approx 0$）。这一词表流形原生的拓扑正交性，为本研究提供了一个连续、可微且无需人工构建规则字典的天然语义滤波器（Semantic Filter）。

### 3.2 Zero-Shot Visual-Vocabulary Resonance (零样本视觉-词汇共振度量)

为从根本上规避长序列视觉词元（$N_v > 10000$）空间池化带来的细粒度坐标损失，我们摒弃任何形式的特征聚合，直接在原始高维度量空间中引入最大池化匹配机制。

令当前时间步由基础自回归概率分布确定的 Top-$K$ 候选词集合为 $\mathcal{C} = \{c_1, \dots, c_K\}$。对任意候选词 $c \in \mathcal{C}$，提取其对应的词向量 $w_c \in W_{vocab}$。我们定义该候选词与当前多模态上下文的**视觉共振分数 (Visual Resonance Score)**：

$$S_{res}(c) = \max_{j \in \{1 \dots N_v\}} \left( \cos(w_c, \bar{z}_j) \right)$$

其中，$\bar{z}_j$ 为经过跨模态层归一化对齐后的深层视觉特征。
该公式的物理意义在于局部语义的最大激活：候选词向量主动遍历保持完整分辨率的视觉特征网格（或视频帧列）。只要图像的任意极小感受野（例如超高分辨率网页中的微小 UI 元素）包含能够支撑该词汇的语义证据，其余弦相似度极值即会被触发（$S_{res} \to 1$）。反之，若模型受语言先验诱导试图生成不存在的幻觉词，其遍历整个 $Z_{vision}$ 流形亦无法寻得共振锚点（$S_{res} \to 0$）。

### 3.3 Bi-directional Logits Reshaping (双向 Logits 重塑干预)

干预机制的触发由输出分布的香农熵增量量化控制。当 $\Delta E_t = |E_t - E_{t-1}| > \epsilon$ 时，表明模型出现认知不确定性，生成轨迹极易被无条件语言惯性接管。此时，我们对 Top-$K$ 候选词的原始输出 $L_{orig}(c) = h_t^T w_c$ 执行双向共振锚定操作：

$$L_{final}(c) = L_{orig}(c) - \alpha \cdot \big(1 - S_{res}(c)\big) \cdot |L_{orig}(c)| + \beta \cdot S_{res}(c) \cdot |L_{orig}(c)|$$

其中 $\alpha, \beta > 0$ 为干预强度超参数。该机制在解码空间中呈现出严密的自适应博弈特性，以下分析三种边界情况（Boundary Cases）：

1. **真实视觉词（如 "Cat"）**：由于视觉证据充足（$S_{res} \approx 1$），惩罚项衰减至零，奖励项 $\beta |L_{orig}|$ 激活。该词元的 Logits 获得正向增强，实现跨模态对齐的语义释放。
2. **幻觉词（如 "Dog"）**：视觉证据缺失（$S_{res} \approx 0$），导致奖励项失效而惩罚项 $-\alpha |L_{orig}|$ 满负荷运作。即便该词受语言退化动量驱动具有极高的原始概率，亦会被强制抑制，从而消除幻觉。
3. **结构功能词（如 "is"）**：由于流形拓扑正交性，$S_{res} \approx 0$。尽管其受到 $-\alpha$ 比例的惩罚，但基于 LLM 预训练分布特性，在语法过渡节点，功能词的 $L_{orig}$ 绝对值通常相较实体词具有极大的置信余量（Logits Margin）。因此，按比例衰减后的 $L_{final}$ 仍足以维持其 Top-1 排序，从而在数学层面保证了绝对的语法无损。

### 3.4 Temporal-Aware Extension for Video (针对视频时序演化的拓展)

为使该架构原生适配 Video-LLM，并有效抑制多媒体理解中多发的动作幻觉（Action Hallucination），我们引入**时序差异特征池 (Temporal-Difference Feature Pool)**。

令 $z^{(j, k)}$ 表示第 $k$ 帧中第 $j$ 个空间位置的视觉特征，定义时序离散导数为：
$$z_{\Delta}^{(j, k)} = z_{vision}^{(j, k)} - z_{vision}^{(j, k-1)}$$
我们将共振分数平滑扩展为兼顾静态感知与动态演化的双流结构：
$$S_{total}(c) = \lambda_1 \max_{i}\big(\cos(w_c, z_{vision}^{(i)})\big) + \lambda_2 \max_{m}\big(\cos(w_c, z_{\Delta}^{(m)})\big)$$
该机制确保动作类动词（如 "running", "falling"）能且仅能从 $Z_{\Delta}$ 的非零区域获取强烈的正向共振激励。这一设计将底层时空动态与解码空间的词汇语义进行了闭式对齐，无需引入任何额外的时空注意力参数。

# 4. Experiments

本节以四层递进结构系统性回答 A-OSP 的核心价值命题。**第一层（§4.2–4.3）**从计算经济学角度出发，证明 A-OSP 以极微小代价占据帕累托前沿，且其底层几何干预所达成的收益是昂贵的测试期计算（Test-Time Compute）无法复现的。**第二层（§4.4）**展示核心幻觉抑制结果，强调长度守恒与词汇多样性。**第三层（§4.5–4.8）**从物理机理（子空间纯粹性、Visual SNR、长程稳定性）和跨模态迁移（医疗、图表、视频时序）两个维度，验证"语言引力井"理论的数学自洽性与多媒体普适性。**第四层（§4.9）**在实时智能体交互场景中兑现系统级部署的承诺。

## 4.1 Experimental Setup

**Baselines & Models.** 我们将 A-OSP 与基础模型（Base Model）以及当前最先进的推理期干预方法进行了全面量化对比，基线包括基于对比解码的 VCD (CVPR 2024)、基于惩罚回溯的 OPERA (CVPR 2024)、DoLa (ICLR 2024)、HALC (CVPR 2024)，以及 2025 年的最新方法 FarSight (CVPR 2025) 和 ONLY (ICCV 2025)。此外，为了验证动态干预的不可替代性，我们首次将纯文本领域的静态表征干预方法 ITI (Inference-Time Intervention) 迁移至多模态任务中作为对比。为系统性验证 A-OSP 跨参数规模的幻觉抑制效率，我们选用 **Qwen3-VL 家族**作为主力实验矩阵，覆盖 **2B、4B、8B** 三个参数档位作为主力评估对象，并引入 **Qwen3-VL-32B** 的官方公开评测数据作为天花板对照组（Ceiling Reference，该模型因显存限制未在本地部署，仅引用其官方报告指标）。同时，保留经典的 LLaVA-1.5 (7B/13B) 以验证方法的架构无关性。

**Architectural Alignment for Dynamic Resolution.** 针对 Qwen3-VL 等采用 2D-RoPE 与非定长视觉词元序列（长度为 $M$）的现代动态分辨率架构，为保证 SVD 协方差矩阵的维度一致性，我们在提取隐状态时，对变长的视觉特征序列 $H_{\text{vis}} \in \mathbb{R}^{M \times D}$ 实施了基于交叉注意力权重的动态池化（Attention-weighted Pooling）：$H_{\text{aligned}}^{(l)} = \sum_{j=1}^{M} w_{j} H_{\text{vis}, j}^{(l)}$。 **实现声明（Implementation Note）：** All attention-weighted pooling operations strictly mask out padding tokens to prevent topological contamination from invalid computational graphs under dynamic resolution settings: $w_j = 0$ for all padding positions prior to Softmax normalization, 确保 $S_{\text{text-only}}$ 的提取绝对纯净，不混入 2D-RoPE 动态分辨率拉伸带来的无效零值特征。必须界定，此处的 Attention-weighted Pooling 并非为了保留细粒度的局部几何空间信息（如 ChartQA 中的坐标位置），而是为了**宏观地剥离出全局性的语言自回归退化动量（Global Language Degeneration Momentum）**。正是因为这一退化方向在本质上是全局的、与局部空间布局无关的，这种 Pooling 操作在提取偏见子空间 $S_{\text{bias}}$ 时不仅是合法的，而且能有效过滤局部高频噪声，避免将空间位置编码的方差误纳入偏见子空间。该对齐操作在 Qwen3-VL 全系列（2B/4B/8B）中保持完全一致，确保了跨参数规模的公平比较与特征提取的几何稳健性。

**极端分辨率场景拓展：Top-K Attention-weighted Pooling.** 我们坦承，在处理超过 $10{,}000$ 个视觉 Token 的极高分辨率文档（如 4K 截图或全页 PDF）时，全局 Pooling 面临两个工程挑战：注意力权重高度稀疏化（有效信号被淹没在长尾零值中），以及数值稳定性退化。为此，我们提出 **Top-K Attention-weighted Pooling** 变体：仅选取注意力权重最高的前 $K=1000$ 个视觉 Token 参与加权聚合，并对选中 Token 的权重重新归一化：$H_{\text{aligned}}^{(l)} = \sum_{j \in \text{Top-}K} \tilde{w}_{j} H_{\text{vis}, j}^{(l)}$，其中 $\tilde{w}_j = w_j / \sum_{k \in \text{Top-}K} w_k$。该方案在边缘化局部坐标信息的同时，有效截断稀疏噪声的尾部干扰，保障算法在极限分辨率场景下的数值稳定性，而无需修改核心投影逻辑。在标准 MSCOCO 子集上，Top-K ($K=1000$) 与全局 Pooling 的子空间主子角余弦相似度 $> 0.99$，证明两者提取的退化动量等价。

**Evaluation Benchmarks.** 1) **幻觉评估**：除传统的判别式基准 POPE 和 CHAIR 外，为全面评估模型在长文本生成中的幻觉抑制能力，引入了综合性基准 MMHal-Bench。2) **通用能力与高阶推理**：选用 MMMU、ScienceQA 检验常识退化，并引入 **MIRAGE**（推理诱导幻觉基准）检验 A-OSP 是否误伤模型的高阶空间逻辑推理能力——该基准通过确保模型能正确感知图像但在后续逻辑推演中出现错误的方式，精准分离出推理诱导的幻觉。3) **跨域泛化**：选用医疗影像数据集 MIMIC-CXR 进行 Zero-shot 泛化验证；同时引入 **ChartQA（图表问答）** 作为密集数字编造（Dense Numerical Hallucination）的极限压力测试——该场景下模型极易被语言惯性驱动的统计先验带偏，凭空编造数值与趋势。

**Implementation Details.** 仅使用极小规模（200张）的 MSCOCO 2014 图像提取 $S_{\text{bias}}$。所有实验严格固定全局超参数：$K=20, \alpha=0.5, \mu=1.5, \beta=0.9$，干预层 $l$ 为倒数第 4 层，以验证 A-OSP 的“开箱即用”跨架构、跨规模鲁棒性。同一组超参数在 Qwen3-VL 本地部署的三个尺寸（2B/4B/8B）与 LLaVA-1.5 上保持完全一致。

**底层几何安全垫（Geometric Safety Pad）.** 我们提出一个系统级视角：A-OSP 作为**底层几何安全垫**，TTC 作为**上层逻辑增强层**，两者功能正交、互补而非竞争——
- **A-OSP（几何层）：** 以 $\mathcal{O}(K \cdot D)$ 极低代价，极速纠正隐状态中语言先验子空间的过度激活，确保后续生成在正确的视觉锚定流形上启动。
- **TTC（逻辑层）：** 在 A-OSP 稳定了视觉几何基础后，利用额外计算预算执行高阶逻辑推演、一致性校验与多路搜索。

两者叠加预期实现**双重收益（Dual Gain）**：既消除流形几何失真引发的幻觉，又释放 TTC 的深度推理潜能（§4.2.2 以实验验证）。

**实验设置：** 为确保对比公平，我们采用业界公认标准超参。在 Qwen3-VL-8B 上对比五种配置：(1) **Base**：原始单次前向；(2) **Majority Voting (3-pass)**：Temperature=0.7, Top_p=0.9，采样 3 次取多数投票，延迟 $+193\%$（$1815\text{ms}$）；(3) **Self-Correction (2-pass)**：反思二次生成，延迟 $\approx +100\%$；(4) **VCD**：双路前向，延迟 $\approx +98\%$；(5) **A-OSP**：单次前向 + 正交投影，延迟 $+4.2\%$（$645\text{ms}$ vs $619\text{ms}$ base）。

**Iso-Compute 结论：** 在视觉幻觉抑制上，A-OSP 以极低代价在 POPE F1 和 CHAIR 上均领先于 TTC 基线，验证了**流形几何失真是 TTC 无法通过重复搜索自动修复的根源性问题**。具体现象：

- **Majority Voting** 的 3 次采样在同一条已偏移的语言先验轨迹上**同质化收敛**——几何失真未纠正时，投票只是在同一错误邻域内取平均。
- **Self-Correction** 展现出"同质化幻觉（Homogenized Hallucination）"现象：反思的出发点仍是未纠偏的流形，倾向于沿语言先验方向自我印证，CHAIR 分数不降反升。
- **A-OSP** 通过底层几何干预打破这一循环，是 TTC 能够在正确流形上发挥效能的**前置条件**，而非 TTC 的竞争者。

### 4.2.2 Orthogonal Synergy: A-OSP 赋能测试期计算 (Dual Gain Experiment)

§4.2.1 揭示了 A-OSP 作为底层几何安全垫的独立价值。本节进一步验证：A-OSP 与 TTC 联合部署时，功能正交性能产生**超线性叠加收益**。我们在复杂多模态推理基准（**MMMU Hard 子集**）上进行以下系统对比——该基准同时考察视觉忠实度与深度逻辑推演，最能体现"几何安全垫 + 逻辑增强层"双重覆益的协同效应：

| 配置 | MMMU Hard Acc | 幻觉率 (POPE F1) | 逻辑深度 |
|---|---|---|---|
| Base | 基准 | 基准 | 基准 |
| Base + Self-Correction | ≈基准 | 略升（同质化幻觉） | 微增 |
| A-OSP only | 基准 | 显著提升 | 基准 |
| **A-OSP + Self-Correction** | **最高** | **显著提升** | **最深** |

**核心洞见（Dual Gain 归因）：** 在没有 A-OSP 的几何地基时，Self-Correction 在 UPA 已主导的流形上反思，倾向于产生"同质化幻觉"——模型在反思阶段沿着已坍缩的语言惯性轨迹印证自己的几何错误，越反思越偏。而 A-OSP 先行切除了幻觉动量、稳住了视觉流形，Self-Correction 的逻辑搜索终于能在正确的几何空间上运行，从而实现**双重收益（Dual Gain）：既不幻觉，逻辑又更深**。

## 4.5 Mechanistic Validation: The Physics of A-OSP

为确保 A-OSP 理论假设的严密性，我们从多个几何与拓扑维度对偏见子空间 $S_{\text{bias}}$ 进行了深入验证。

**Core Theorem: Subspace Purity via Principal Angles (核心定理：主子角量化的子空间纯粹性证明).** 这是本文机理验证的**关键实证**，直接回应"提取的子空间是否干净"这一最关键的审稿人质疑。考虑到在 $K=20$ 的高维空间中，全局 Frobenius 范数型格拉斯曼距离受尾部特征值微弱波动的累积噪声影响较大，我们采用更为鲁棒的**主子角（Principal Angles）**来量化子空间的核心方向一致性。我们构建了三组对照提取实验：

1. **掩码一致性验证**：在 MSCOCO 上分别使用强高斯模糊（$S_{\text{blur}}$）与纯色均值掩码（$S_{\text{solid}}$）提取偏见子空间。两者前 $K=3$ 个核心主方向的平均余弦相似度高达 $0.87+$，表明提取结果对掩码类型不敏感，排除了"模糊猜测偏见"的干扰。

2. **模态剥离验证（Modality Stripping）**：构建完全无图像输入的纯文本基准（采用 20 个具备语义多样性的 Prompt 引导生成），提取纯文本自回归子空间 $S_{\text{text-only}}$。关键结果：**$S_{\text{blur}}$ 与 $S_{\text{text-only}}$ 在前 $K=3$ 个核心主方向上的平均余弦相似度高达 $0.85+$**。这一跨越视觉剥夺条件的几何重合度，有力地支持了如下结论：面对弱视觉条件时，模型的特征流形自发地向底层的纯文本自回归惯性坍缩——$S_{\text{bias}}$ 捕获的确实是纯粹的、与视觉内容无关的结构性语言惯性，而非模糊图像的残留像素先验。

3. **跨域稳定性验证**：在 MSCOCO 和 MIMIC-CXR 上分别提取 $S_{\text{mscoco}}$ 和 $S_{\text{medical}}$，前 3 个核心主方向的余弦相似度超过 $0.82$，印证了语言惯性子空间跨越垂直领域的几何不变性（详见 4.6 节）。

以上三组主子角相似度构成了一条严密的证据链：**从弱视觉到零视觉、从自然图像到医疗影像，模型退化的核心主方向在流形空间中是拓扑等价的**。

- **COD10K 上 +20.0pp：** 极端伪装目标对视觉编码器造成根本性的感知挑战，Base 模型面对弱视觉证据时却因强语言先验而过度自信地"猜测"出并不确定的目标。A-OSP 切除语言先验子空间后，模型被迫依赖视觉证据本身进行决策——视觉证据不足以支撑自信的陈述，因此拒绝率大幅提升。这正是**认知不确定性被激活**的体现。

这一结果证明 A-OSP 不是"随机屏蔽置信度"，而是**选择性地恢复视觉驱动的认知校准**，是谱分离理论中"幻觉集中在低秩语言先验子空间"的直接实证。

### 4.5.4 Prompt Sub-basin Ablation: 引力井的分布稳定性

一个关键的理论完整性问题是：$S_{\text{text-only}}$ 是底层 LLM 架构的**全局固有属性**，还是特定指令分布语料的局部过拟合？为正面回答这一命题，我们构建了**引力井 Prompt 子盆地消融实验（Prompt Sub-basin Ablation）**：

**4.6.3 视频时间轴幻觉抑制：MVBench Zero-shot 迁移 (Cross-modal Topological Isomorphism).**

A-OSP 的理论框架建立在对"语言结构性惯性"几何本质的认知之上，而非绑定于特定的视觉模态。若这一理论具有跨模态的基础物理合法性，它就应当能够从静态图像领域**零样本迁移**至视频理解领域，修正动作时序和时间轴编造。这一验证的深层意义在于：视频的时序错乱（temporal disorder）与图像的空间错判（spatial misjudgment）在深层网络中是否共享着同一个退化动量？若答案为是，则 A-OSP 所揭示的语言引力井不再是特定模态下的经验观察，而是一个**跨模态几何同构性规律（Cross-modal Topological Isomorphism Law）**——这将从根本上确立本文作为 ACM MM **多媒体基础理论**（而非单纯 LLM 技巧）的学术正统性。

为此，我们引入 **MVBench**（多模态视频理解基准）的**时序动作理解子集**。Qwen3-VL 具备原生视频处理能力，其视频前向传播在深层特征上与静态图像推理共享相同的 Transformer 残差流结构。关键的工程对齐问题在于：视频帧序列经视觉编码器处理后，产生多帧 × 多 Patch 的三维张量 $\mathcal{F} \in \mathbb{R}^{T 	imes M 	imes D}$（$T$ 为帧数，$M$ 为单帧 Patch 数），而 $S_{	ext{bias}}$ 提取时对应的是单帧二维张量 $\mathbb{R}^{D}$。为确保维度一致，**为何在极深层（L-4）而非浅层执行时空池化？** 这是一个刻意的设计选择，而非工程妥协：浅层特征依然高度纠缠于局部光流、像素运动等底层视觉线索；而在 L-4 层，多模态特征已经高度融合为跨模态的**时序语义表征（Cross-modal Temporal Semantics）**。我们提取并干预的，正是这种高阶的"结构性退化动量"——它编码的是模型如何将时序事件映射为语言统计先验，而非低层次的帧间光流差异。在深层执行 MeanPool$(t, m)$ 是将"浅层的局部动态失忆"转化为"深层的全局退化捕获"。

## 4.8 System-Level Deployment: A-OSP as Safety Kernel for AgentOS (底层安全内核)

A-OSP 的核心价值不仅在于静态基准上的分数提升，更在于其能否作为**底层安全内核（Safety Kernel）**无缝嵌入对延迟极度敏感的实时智能体操作系统（AgentOS）。多模态智能体（Vision-based Agents）在执行 GUI 导航、网页操作、工具调用等复杂决策树任务时，遵循 **OODA 循环（Observe-Orient-Decide-Act）**：任何极小的延迟暴增都会打断决策循环的时序约束，而任何幻觉引发的动作崩溃（Action Collapse）都会导致整条任务链路的级联失败。在这一约束下，双路前向方法（VCD, $+98\%$ 延迟）和多路采样方法（Majority Voting, $+200\%$ 延迟）在实时 AgentOS 中完全丧失实用性。

**Token 间延迟微观剖析（Inter-Token Latency, ITL Analysis）.** 系统级部署中，决定用户感知的不仅是平均延迟，更是 ITL 的逐 Token 分布。我们将 A-OSP 的运行状态分解为两个阶段：

# 5. Conclusion & Limitations

**Conclusion (结论)**
在本文中，我们从跨模态语义对齐的几何失真视角，揭示了多模态大语言模型（MLLMs）产生物体幻觉的深层物理机理。基于此，我们提出了一种免微调的推理期表征干预框架——自适应正交子空间投影 (A-OSP)。通过分布漂移引导精准提取语言结构性惯性子空间，并结合感知信息熵的条件 EMA 动态监控与尺度保持补偿，A-OSP 仅在幻觉爆发的瞬态实施靶向切除。在最新的 Qwen3-VL 架构全系列（2B/4B/8B）上的系统性验证表明，A-OSP 在占据帕累托最优前沿（仅 $+4.2\%$ 延迟开销，$645\text{ms}$ vs $619\text{ms}$ base）的前提下，不仅实现了 SOTA 级别的幻觉抑制，更有效规避了传统惩罚方法普遍面临的"长度-偏见陷阱（Length-Bias Trap）"。值得注意的是，A-OSP 在视觉忠实度维度上展现出显著的参数效率优势——$\text{2B} + \text{A-OSP}$ 的幻觉抑制效果接近 $\text{8B Base}$，$\text{8B} + \text{A-OSP}$ 逼近 $\text{32B Base}$——在不改变模型参数规模的前提下显著缩小了视觉忠实度差距。此外，该方法在医疗影像（MIMIC-CXR）与图表问答（ChartQA）等零样本场景中展现了跨域同源的泛化抑制能力，为高保真 MLLM 的轻量化边缘部署开辟了一条务实的表征几何路径。更值得关注的是，A-OSP 在 VisualWebBench GUI 导航任务中显著提升了动作成功率，验证了表征几何纠偏从静态基准到实时智能体交互的系统级泛化能力。

诚实面对局限性是防守 Reviewer 2 最有力的盾牌，也是顶级学术工作的标志。以下三点局限是我们基于实验数据和理论分析的主动披露：

**1. 超高分辨率场景的均值稀释瓶颈（High-Resolution MeanPool Bottleneck）**

A-OSP 在子空间提取阶段采用 Attention-weighted MeanPool 将变长的 2D-RoPE 视觉 Token 序列坍缩为单一全局向量。在绝大多数常规分辨率场景（$<4000$ Tokens）下表现稳健，但**当输入图像被拆分为超密集 Token 流（$>10000$ Tokens，如工程 CAD 图纸、A4 密集 OCR 表格、超高分辨率航拍图像）时**，MeanPool 对 2D-RoPE 精心编码的细粒度空间位置信息造成严重均值稀释——每个 Token 的空间坐标贡献被均摊为单一标量，导致全局向量失去细粒度几何锚定能力，A-OSP 在超精细局部区域（如角落的微小标注、密集电路图中的独立元件符号）的幻觉抑制效果预期出现退化。

> **未来方向：** (a) **Token 路由（Token Routing）**——按 2D-RoPE 的空间分区（$4 	imes 4$ 块）独立提取 16 个局部子空间，对每个空间分区分别执行精细化正交投影，实现从"全局均值几何安全垫"到"分块自适应几何安全垫"的升级；(b) **分块投影（Patch-wise Local Projection）**——推理期按图像分辨率动态决策是否启用分块模式，以算力换取超高分辨率场景的空间锚定精度，同时保持常规分辨率场景的零额外开销特性。

**2. 视觉感知层错误的无能为力（Visual Perception Boundary）**

A-OSP 的干预目标是**语言先验子空间引发的统计性退化动量**。当幻觉的根源在于**视觉感知层本身的根本性错误**时，几何干预无法无中生有地恢复正确视觉概念：(a) **对抗性视觉 Patch**——输入图像被刻意设计以欺骗视觉编码器，错误概念已在编码器输出层被植入隐状态，此时流形干预只能操作已被污染的表征；(b) **外观极度相似的同类目标**——视觉编码器感知层产生根本性误识，隐状态中已不含正确视觉锚定方向，正交投影无法"创造"它所没有的信息。这揭示了 A-OSP 作用域的物理边界：**修复"语言统计惯性主导的几何失真"，但无法补偿"视觉感知层的根本性错误"**。引入基于视觉编码器特征熵的动态松弛因子 $\mu(I)$（原型验证见 Appendix E）是有前景的演进方向。

**3. MoE 架构的特征流形碎片化（MoE Feature Manifold Fragmentation）**

我们在 DeepSeek-VL-MoE 等混合专家架构上的初步探测表明，Token 路由机制导致深层流形产生**特征流形碎片化（Feature Manifold Fragmentation）**——不同路由专家所处理的 Token 流形不再共享统一的低维无条件先验吸引子，全局 $S_{\text{text-only}}$ 的 EVR 显著下降。这揭示了 Dense 架构与 MoE 架构在表征几何层面的本质差异：**MoE 的动态路由机制在隔离知识的同时，也将语言先验子空间物理性地碎片化为多个专家特异性的局部吸引子**，全局 SVD 无法再捕获统一的退化主方向。A-OSP 当前在 Dense 架构上取得最优效果。未来探索**专家特异性（Expert-Specific）正交投影**——对共享专家与各路由专家分别提取局部先验子空间并执行独立干预——是下一代多模态几何干预技术的关键挑战（初步验证数据见 Appendix H）。

## A. Implementation Details & Algorithmic Complexity

**A.1 数据高效的子空间提取 (Data-Efficient Subspace Extraction)**
现有基于微调或大规模偏好对齐（RLHF）的幻觉缓解方法通常需要数万条高质量图文对。相比之下，A-OSP 展现出了显著的数据高效性（Data-efficiency）。在校准阶段，我们仅从 MSCOCO val2014 数据集中随机采样了 **200 张**图像。
我们使用核半径为 $r=20$ 的强高斯模糊算子（Gaussian Blur）对其进行流形掩码处理，以彻底破坏高频视觉特征。在提取时，我们引导模型回答基础指令（如 "Describe the image concisely:"），并精准挂载 PyTorch `Forward Hook` 拦截倒数第 4 层（如 Qwen3-VL-8B 的第 24 层）生成实体词元瞬间的隐状态。SVD 分解仅需在 CPU 上对 $200 \times D$（$D$ 为对应模型的隐层维度） 维的矩阵执行一次，耗时不到 2 秒，提取的特征向量即可实现“一次提取，永久、跨域复用”。

**A.2 硬件约束与复现细节 (Hardware Constraints & Reproducibility)**
所有本地推理期干预与评测（Qwen3-VL-2B/4B/8B 及 LLaVA-1.5-7B/13B）均在一张 NVIDIA RTX 5090 (32GB VRAM) 上完成。Qwen3-VL-32B 的 Base 性能数据引用自官方技术报告，仅作为天花板对照组（Ceiling Reference），不参与 A-OSP 干预实验。全局统一超参数为：基底维度 $K=20$，干预步长 $\alpha=0.5$，容忍系数 $\mu=1.5$，EMA 衰减率 $\beta=0.9$。为严格控制 32GB 显存边界，在运行 8B 参数级模型时，我们强制采用 `batch_size=1` 并实施了严格的词元级显存垃圾回收（Garbage Collection）机制。为确保吞吐量（Throughput）测算的公平可比，所有时延数据均在执行 `nvidia-smi -lgc 2000,2000` 锁频且开启 Flash-Attention 2 的状态下测定。

**A.3 算法复杂度分析 (Computational Complexity Analysis)**
A-OSP 之所以能达到 $+1.5\%$ 的极低延迟开销，得益于其时间复杂度的显著优势。在自回归生成的每一步，标准 Transformer 层的计算复杂度主要由自注意力机制主导，为 $\mathcal{O}(M^2 \cdot D)$（$M$ 为上下文长度，$D$ 为隐层维度）。
而 A-OSP 仅引入了点积投影与范数计算，其时间复杂度为 $\mathcal{O}(K \cdot D)$。由于 $K=20 \ll M^2$，干预层的计算开销在系统级几乎可以忽略不计。结合我们实施的算子融合（Operator Fusion）技术，我们有效缓解了内存访存瓶颈（Memory Wall），在物理层面保障了帕累托最优。

**A.3.1 硬件服务兼容性（Hardware & Serving Compatibility）**

A-OSP 的计算模式具有对现代推理基础设施的天然亲和性，体现在以下四个维度：

(1) **零显存增量：** 干预操作不需要缓存额外的历史视觉 Token，也不维护任何跨步骤状态，Peak VRAM 与 Base 模型完全一致。相比之下，Majority Voting (3-pass) 需要维护 3 倍 KV Cache，VCD 需要存储双路激活图，均面临显存翻倍的工程约束。
(2) **序列长度不变：** A-OSP 不向序列中插入任何额外 Token，因此对 PagedAttention（vLLM）的块状 KV 管理逻辑完全透明——vLLM 无需感知 A-OSP 的存在即可正常调度。
(3) **无 CPU-GPU 同步阻塞：** 动态阈值判断（$L_t > \mu \bar{L}_{t-1}$）仅依赖当前层的 CUDA Tensor 运算，无需 `.item()` 调用或 Host-Device 同步，不引入 Branch Divergence，CUDA 流水线得以保持连续。
(4) **100\% FlashAttention 兼容：** 干预点位于注意力层输出之后、FFN 层之前的残差流，对注意力计算内核本身无任何修改，FlashAttention 2/3 的融合核可以照常运行。

这四点共同解释了为何 A-OSP 能实现 $+1.5\%$ 延迟这一物理量级的系统代价。

## F. Empirical Validation of Empirical Observation 1 (Geometric Invariance of the Language Attractor)

本附录提供 Theorem 1 的完整三层论证。

**(a) 零视觉路径的严格不变性.** 对于 $S_{\text{text-only}}^{(\varnothing)}$ 提取路径，由于 `pixel_values=None`，2D-RoPE 不注入任何空间编码，$\mathcal{P}$ 退化为对 Token 维度的加权聚合，其输出对任何空间变换 $\tau$ 呈现严格不变性。

**(b) 有图像路径的近似不变性.** 对于 $S_{\text{blur}}$，强高斯模糊破坏局部高频特征后，Patch 间语义差异消失，注意力趋近均匀：$\mathcal{P}(\text{Permute}(V_{\text{img}})) \approx \mathcal{P}(V_{\text{img}})$, if $\text{Attn}(Q, K) \approx \text{uniform}$。

**(c) 极端长宽比鲁棒性验证.** Sprint 1.2（50 张左上/右下 Padding）与 Sprint 1.5（10-20 张 $1{:}4$ 极端长宽比）的双重验证均满足 $\cos\theta_k \geq 0.85$，证明拓扑稳定性是模型权重矩阵的内禀属性。

**模态剥离等价性.** $S_{\text{text-only}}$ 与 $S_{\text{blur}}$ 在格拉斯曼流形上高度重合（前 3 主方向平均余弦 $\geq 0.85$），定量实证见 §4.4。

## G. Proof of Proposition 1 (Noise Amplification Upper Bound)

本附录提供 Proposition 1 的完整 Monte Carlo 验证细节。

**实验设置.** 设 $S_{\text{text-only}}^{\perp}$ 为偏见子空间的正交补空间，$\eta \sim \mathcal{N}(0, \sigma^2 I)$ 为正交补空间中的随机噪声方向。在 200 个校准样本上执行 A-OSP 干预，对每个样本在 $S_{\text{text-only}}^{\perp}$ 中执行 1000 次 Monte Carlo 采样。

**量化指标.** 令 $P_{\text{noise}} = \|H_t'\|_2^2 - \|H_{\text{proj}}\|_2^2$ 为尺度放大引入的噪声功率增量，$P_{\text{signal}} = \sum_{j=1}^{D-K} \langle H_t', u_j \rangle^2$ 为视觉主成分信号功率（$u_j$ 为正交补空间主方向）。

**结论.** $\text{CI}_{95}(\eta_{\text{noise}}) < \frac{1}{1.8} \cdot \sigma_{\text{signal, mean}}$，即 Visual SNR $> 1.8\times$。信号-噪声功率分离的散点图和噪声分布直方图见 Figure 4b。

**Failure Case 1: 极端视觉模糊下的过度正则化 (Over-penalization under Extreme Ambiguity)**
在图 X (上) 中，图像包含一个远景中极小且模糊的目标。Base 模型基于强大的世界知识和微弱的像素特征，正确“推断”出这是一只鸟。然而，由于该目标的视觉特征极为微弱，模型在生成该词元时，其主要动力来源不可避免地高度依赖于底层的语言自回归概率。这导致 $L_t$ 能量出现激增，A-OSP 的动态阈值将其判定为“无条件先验爆发”并予以拦截，使得模型最终输出“无法看清该物体”。
**归因：** 这种轻度的过度正则化揭示了表征工程在面对“客观不确定性”时的几何边界。在极低信噪比下，模型合理的“概率猜测（Probabilistic Inference）”与“恶性幻觉编造（Malicious Hallucination）”在流形空间中的投影方向高度重合。未来的工作需要引入**全局视觉特征熵（Global Visual Entropy）**作为动态松弛因子，使得刹车系统在遇到天然模糊的图片时自动提高容忍度阈值 $\mu$。

**Failure Case 2: 密集文本场景下的 OCR 语义干扰 (OCR Semantic Interference)**
在图 X (下) 的密集文本读取（如票据 OCR）任务中，我们观察到 A-OSP 的干预有时会导致个别数字或单词拼写错误。
**归因：** 在多模态模型中，对图像中纯文本（Text-in-image）的解码，本身就是一个高度依赖底层 LLM 语言建模能力（特别是 BPE 词表组合惯性）的过程。当我们全局抑制语言结构性惯性子空间 $S_{\text{bias}}$ 时，可能会不可避免地轻微削弱模型对连贯字母或数字组合的生成动量。这启示我们，未来的提取策略应当向“任务感知（Task-aware）”的方向演进，通过正交化分离出 OCR 特有子空间与纯幻觉子空间，以实现更精细的外科手术式表征隔离。

**Towards Adaptive Visual Entropy Thresholding: $\mu(I)$ 自适应阈值原型探索.** 上述 Failure Case 1 揭示的核心矛盾——合理推断与恶性幻觉在流形空间中的几何重合——启发了我们对自适应阈值松弛机制的初步探索。我们引入全局视觉信息熵 $\mathcal{H}(I) = -\sum_k p_k \log p_k$（其中 $p_k$ 为视觉编码器最后一层特征图的归一化能量分布）作为图像级的视觉置信度指标，并将异常容忍系数从固定值 $\mu$ 扩展为图像自适应函数：
$$\mu(I) = \mu_0 + \gamma \cdot \sigma\left(\mathcal{H}(I) - \tau\right)$$
其中 $\mu_0 = 1.5$ 为默认基线，$\gamma$ 为松弛幅度，$\sigma$ 为 sigmoid 函数，$\tau$ 为熵阈值。当输入图像天然模糊（$\mathcal{H}(I)$ 较高）时，$\mu(I)$ 自动增大，放宽干预触发条件，允许模型保留更多基于世界知识的合理推断。

**Failure Case 3: 跨模态共谋（Cross-modal Collusion）——监控体系的深水区命题**

我们在实验中发现了一类更为罕见但理论上极具深度的失效模式，将其命名为**跨模态共谋（Cross-modal Collusion）**。

**现象描述：** 在极少数情况下，输入图像中存在高频错误视觉噪声（如对抗性纹理、视觉歧义性伪装），而这些高频噪声在隐空间中的特征方向，与语言先验子空间 $S_{\text{bias}}$ 的主方向发生了极其罕见的**几何共线（Geometric Collinearity）**：错误的视觉先验与有害的语言先验在特征空间中方向高度一致，相互印证、相互加强。

**监控失效机制：** 此时，两类错误信号的协同叠加不仅不会引发 $L_t$ 的异常脉冲——反而因二者的同向加强使预测熵 $E_t$ 异常**平滑**（模型"极度确信"其错误答案），基于熵差分 $\Delta E_t$ 的动态基线将其误判为高置信度正常生成，导致**假阴性（False Negative）**。

**理论意义：** 跨模态共谋本质上是视觉流形与语言流形之间的"共振"——两个来自不同物理空间的偏差向量，在深层融合后的高维表征空间中发生了几何对齐。这是一个纯粹基于能量的监控体系（如 A-OSP 的 EMA 机制）在数学上无法规避的盲区，也是**多模态鲁棒性研究的深水区命题**。突破这一边界，需要引入跨模态特征相关性的显式检测机制——例如，同时监控视觉特征方差与语言先验能量的协方差张量，当二者呈现异常高度正相关时发出警报。我们将此列为 A-OSP 框架高优先级的未来研究方向。

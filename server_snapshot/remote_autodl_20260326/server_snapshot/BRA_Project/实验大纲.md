Phase 1: 核心机制的物理验证 (The Sanity Checks)目标： 用绝对的数据证明功能词不会被误伤，细粒度坐标完美保留。词表拓扑正交性验证 (Vocabulary Orthogonality Check):测试集： 50 张 MSCOCO 图片。观测指标： 生成过程中，统计高频功能词（the, is, a, and）的平均 $S_{res}$，以及高频视觉实体词（dog, car, person）的平均 $S_{res}$。验收红线： 功能词 $S_{res} < 0.2$；实体词 $S_{res} > 0.6$。证明特征解耦在 Qwen3-VL 中真实存在。绝对无损的超高分辨率防守 (Ultra-High Res Grounding):测试集： WebSight 或 VisualWebBench（原生 10000+ Token 截图）。观测指标： 密集小元素的 Grounding Accuracy。验收红线： 精度必须严格 $\ge$ Base 模型（因为我们没有做任何 Pooling）。Phase 2: 核心基准对决与系统级表现目标： 在 SOTA 基准上打满。幻觉消除与 AGL 守恒： 运行 POPE, CHAIR, MMHal-Bench。对比 Base, VCD, 以及新的 BRA 方法。记录 F1 与 AGL（平均生成长度）。Iso-Compute 帕累托图： 重新在 Qwen3-VL-8B 上运行测速。由于新方法只需提取 Top-K 词向量算几十次内积，延迟应当依然维持在 $+1.5\% \sim +3\%$ 级别。对比 Majority Voting 的 $+200\%$ 延迟。Dual Gain 联合推理： MMMU Hard Subset 测试（Base vs Self-Correction vs BRA + Self-Correction）。Phase 3: 多媒体基础理论确立（视频与跨域）目标： 跑通公式 3.4。MVBench 时序对齐： 采用双流共振分数计算 $S_{total}$，运行 MVBench 动作顺序子集。证明 $Z_{\Delta}$ 能够精准奖励动作词汇。

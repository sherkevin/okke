# Bounded Region Alignment: Token-Local Visual Evidence for Structure-Preserving MLLM Decoding

## Abstract

Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when strong language priors overshadow the localized visual evidence required for dense grounding. While existing inference-time mitigation strategies (e.g., contrastive decoding, attention pattern penalties) serve as highly competitive, orthogonal regularizers for overarching language or attention dynamics, they lack a mechanism to inject strictly **token-local visual evidence** directly into the terminal logits space while preserving complex language structure. We present **Bounded Region Alignment (BRA)**, a decode-time intervention that reweights candidate tokens using localized visual support extracted from contextualized final-layer hidden states. 

BRA operates on three core mechanisms. First, to handle input-output embedding asymmetry and architectural spatial washout, we establish a strict evaluation boundary separating a zero-shot projection (`BRA_zero`) from a lightweight, dynamically calibrated bound (`BRA_calib`) optimized via Intersection-over-Union (IoU) penalized contrastive learning. Second, **Threshold-Gated Adaptive Top-$k$ Pooling** extracts stable local evidence using a moving median to mathematically prevent background patches from casting forced votes. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via a WordNet-lemmatized dictionary and dynamic Byte-Pair Encoding (BPE) continuation inheritance. Furthermore, to prevent catastrophic computational overheads, we introduce dynamically calibrated entropy-gated triggers and strict candidate-window bounds.

To validate BRA, we propose a pre-registered, hypothesis-driven evaluation protocol restricted to 2D spatial grids. Evaluated across three definitive evidence chains—(A) Hallucination Reduction via POPE/CHAIR and Attention-Grounded Localization (AGL), (B) Structure Preservation tracking Out-Of-Vocabulary (OOV) limits on MMMU(Hard), and (C) Local Evidence Value comparing `BRA_zero` against global pooling baselines—alongside five transparent defense lines profiling batched latency and peak VRAM, this framework establishes a scientifically sound proposition for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable zero-shot capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to clarify that these methods serve as state-of-the-art orthogonal regularizers that effectively discipline the language-side or attention-side generation heuristics. However, because they operate on overarching generation dynamics rather than explicit spatial grounding, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, defensible hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination more faithfully than pure language/attention heuristics, provided it incorporates explicit structure-preserving safeguards. We conceptualize this intervention as **Bounded Region Alignment (BRA)**. In this work, we strictly bound our scope to Image-LLMs and 2D spatial reasoning, deliberately downgrading unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims surrounding spatial locality.

Executing BRA requires acknowledging and mathematically overcoming three structural barriers. First, **input-output embedding asymmetry**: raw visual tokens reside in the input space, rendering direct dot-product similarity with the terminal output vocabulary matrix invalid. Second, the **post-hoc entanglement gamble**: leveraging the final-layer hidden states of visual patches assumes localized support survives deep attention mixing. This risks severe spatial washout, particularly varying between 1D-sequence architectures (e.g., LLaVA) and 2D-RoPE architectures (e.g., Qwen-VL). Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, raw byte fallbacks, functional syntax, or multi-token entities.

BRA resolves these barriers via a structured framework optimized against batched latency traps:
1. **Bi-Level Logits Intervention (`BRA_zero` / `BRA_calib`)**: A strict boundary evaluating whether final-layer visual states natively align with the pre-vocabulary dimension (`BRA_zero`), supported by a fallback calibrated projection (`BRA_calib`) trained under stringent SAM-assisted IoU-penalized constraints to prevent contrastive poisoning from background pixels.
2. **Threshold-Gated Adaptive Top-$k$ pooling**, extracting localized spatial support via abstention-aware patch activations grounded by a robust moving median.
3. **VASM (Vocabulary-Anchored Semantic Masking)**, utilizing dynamic BPE continuation inheritance and a brute-force polysemy superset to shield structural language, explicitly confining intervention to visual nouns.

To ensure rigorous scientific boundaries, our experimental design is organized into an execution protocol strictly defining three primary evidence chains and five transparent defense lines. 

- **Chain A (Hallucination Reduction):** Validating true mechanism value on POPE, CHAIR, and AGL (Attention-Grounded Localization) against a parameter-matched `Base + 5k LoRA` baseline to isolate decode-time benefits from pure data exposure.
- **Chain B (Structure and Reasoning Preservation):** Proving the necessity of VASM via MMBench, MME, and MMMU(Hard), transparently tracking exact Out-Of-Vocabulary (OOV) rates and the absolute Top-$M$ unrecoverability ceiling.
- **Chain C (Local Evidence Value):** Demonstrating the superiority of localized evidence via FREAK and DocVQA, strictly comparing `BRA_zero` against a `BRA_MeanPool` baseline to isolate the value of geometric isolation.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Overcoming Post-Hoc Entanglement: `BRA_zero` vs `BRA_calib`

A fundamental vulnerability in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, BRA evaluates the *contextualized* final-layer transformer hidden states of the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$. 

We explicitly acknowledge the gamble of **post-hoc entanglement**: deep self-attention implicitly mixes visual states with prefix context. To handle vocabulary projection cleanly without falsely claiming zero-shot alignment if an architecture inherently destroys spatial fidelity, we define a strict operational boundary:

1. **`BRA_zero`**: A strict zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$.
2. **`BRA_calib`**: If `BRA_zero` fails due to spatial washout, we pivot to a lightweight calibrated alignment. $\Phi_{calib}: \mathbb{R}^{D_{vision}} \rightarrow \mathbb{R}^{D_{llm}}$ projects the visual state to the LLM's final hidden dimension prior to $W_{vocab}$. It is trained on a frozen set of 5,000 COCO image-patch to bounding-box label pairs, with the base LLM entirely frozen.

**Strict Semantic Negative Sampling (SAM/IoU Assisted):** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. Because raw bounding boxes inherently contain background pixels, we institute a strict sampling logic: we apply an off-the-shelf segmentation mask (SAM) to extract strictly foreground patches. Negative patches ($W_{vocab}[c^-]$) are cross-referenced to ensure an Intersection-over-Union (IoU) $< 0.1$ with the target, preventing contrastive poisoning from shared background textures and establishing a pure gradient signal for spatial localization.

### 3.2 Threshold-Gated Adaptive Top-$k$ Pooling

At decoding step $t$, BRA evaluates spatial support. A naive Softmax over all visual patches would force background patches to sum to 1.0, artificially inflating noise. We apply a threshold-gated Sigmoid activation, allowing irrelevant patches to mathematically abstain:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{noise}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{noise}\Big)
$$
To prevent $\theta_{noise}$ from becoming a brittle static hyperparameter, it is dynamically defined per-step as the moving **median** of the activation distribution across the $N_v$ patches. We select the median because it is statistically robust against dense vs. sparse image contexts. We then apply an Adaptive Top-$k$ rule: $k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)$, aggregating only the top-$k$ most supportive activated patches.

### 3.3 VASM (Vocabulary-Anchored Semantic Masking) and Entropy-Gated Triggers

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. This penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$

VASM prevents structural collapse via a dual-tier protection architecture:
1. **Root Token Expectation (WordNet-Augmented Visual Nouns):** $\gamma=1$ *only* for an unambiguous dictionary of visual object nouns. Polysemy is handled via a brute-force superset—if any synset is a visual noun, it triggers intervention. Out-of-dictionary terms default to $\gamma=0$, guaranteeing structural syntax safety. 
2. **BPE Continuation Inheritance (Dynamic):** Any subword token marked as a continuation fragment automatically inherits the exact $\gamma$ weight of its root token. If the tokenizer outputs raw byte fallbacks (e.g., `<0xE2>`), the sequence inherits the $\gamma$ of the immediately preceding structural subword.

**Evading the Batched Latency Trap (Defense Line):** Computing similarities across $V$ tokens and $N_v$ patches inherently risks a batched Out-of-Memory (OOM) failure. We implement two bounds:
1. **Candidate Restriction:** Intervention is strictly bounded to $\mathcal{C}_t = \text{Top-}M(L_{orig})$ where $M \ll |V|$ (e.g., $M=100$).
2. **Dynamically Calibrated Entropy Gate:** We bypass BRA if the base model's Top-1 Confidence exceeds an empirically calibrated safe zone, restricting intervention only to statistically uncalibrated decoding steps.

## 4. Evaluation Protocol and Experimental Design

Because BRA introduces specific structural hypotheses regarding decode-time local evidence, spatial washout, and batched latency, we organize the evaluation plan into predefined falsifiable evidence chains and strict defense lines. The protocol utilizes **LLaVA-1.5** (MLP projector, 1D sequence) and **Qwen-VL-Chat** (2D-RoPE) to test architectural variance. All experiments outline our planned, hypothesis-driven execution.

### 4.1 Chain A: Hallucination Reduction vs. Confident Priors

**Hypothesis:** BRA mitigates object hallucination through localized evidence reweighting. These gains derive from the decode-time intervention mechanism, not mere parameter exposure.
- **Evaluation Protocol:**
  - **Metrics:** POPE (Acc/F1), CHAIR (Object vs Attribute), and **AGL (Attention-Grounded Localization)** to track geometric precision alongside text generation.
  - **Zero-Shot Baselines:** Base, VCD, OPERA, DoLa, `BRA_zero`.
  - **Calibrated Baselines:** `Base + 5k LoRA` (fine-tuned on the identical 5k SAM-filtered pairs, run across 3 random seeds; LoRA rank $r$ matched exactly to the ~16M parameter footprint of $\Phi_{calib}$), and `BRA_calib`.
- **Claim Retreat Contingency:** If `BRA_zero` fails on LLaVA-1.5 but succeeds on Qwen-VL, we will isolate 1D spatial washout as the cause. We will transparently report the "Entropy Gate Override %"—the rate at which highly confident hallucinations (arrogance of language priors) bypass the mitigation mechanism.

### 4.2 Chain B: Structure and Reasoning Preservation

**Hypothesis:** Hallucination mitigation via visual suppression does not degrade complex reasoning, provided VASM actively protects structural tokens and BPE Inheritance avoids subword collisions.
- **Evaluation Protocol:**
  - **Benchmarks:** MMBench V1.1, MME, MMMU (Hard Subset).
  - **Ablations:** Base, `BRA_no_VASM` (naive global intervention), `BRA_no_BPE_Inherit`, `BRA_full`.
  - **OOV Exhaustion Tracking:** We mandate a scatterplot mapping MMMU Performance (y-axis) against OOV Rate (x-axis) across sub-disciplines. This visualizes exactly how VASM dictionary exhaustion correlates with reasoning drop-offs.
  - **Top-$M$ Unrecoverability Defense Line:** We track precisely how many target visual nouns were natively ranked outside the top-$M$ candidate window by the base model, establishing the mathematical ceiling for intervention success.

### 4.3 Chain C: The Necessity of Local Evidence

**Hypothesis:** On dense spatial tasks, token-local visual reweighting geometrically isolates evidence that diffuse pooling natively washes out.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (OCR/Position) and DocVQA (ANLS).
  - **Ablations:** Base, `BRA_MeanPool` (naive global averaging), and `BRA_zero` / `BRA_calib` (Adaptive Top-$k$).
- **Qualitative Protocol (4-Way Visualization):** A side-by-side failure analysis plotting: (A) Base Native Attention, (B) `BRA_MeanPool` 2D grid, (C) `BRA_zero`/`calib` Adaptive Top-$k$ grid, and (D) Entropy Gate Activation Over Time.
- **Entanglement Washout Case:** We must document a failure case where `BRA_zero` highlights the wrong area (e.g., attention sink corner bias) but `BRA_calib` corrects the localization, empirically validating the Bi-level design.

### 4.4 Defense Lines: Latency Profiling and OOM Bounds

**Hypothesis:** The latency penalty can be effectively managed via dynamic entropy-gating and bounded candidate windows without sacrificing hallucination mitigation, though batched scaling remains a severe empirical constraint.
- **Evaluation Protocol:** We plot **Tokens/Sec** and **Peak VRAM Usage** against the Candidate Window Size ($M$), evaluated strictly across **Batch Sizes = 1, 4, and 8**. If BRA triggers Batched Out-of-Memory (OOM) failures beyond BS=1, it will be transparently bounded as a high-precision offline analytical tool rather than a real-time production API.

## 5. Discussion and Limitations

The strength of Bounded Region Alignment (BRA) lies in its structurally bounded proposition: intervening at decode-time using strictly token-local visual evidence while mathematically protecting language integrity via VASM. However, its execution is governed by severe theoretical and operational constraints (our Five Defense Lines):

1. **The Post-Hoc Entanglement Gamble:** The assumption that localized visual support survives deep self-attention in final layers is precarious. If `BRA_zero` fails across architectures, the premise of extracting zero-shot local states must rely entirely on the calibrated $\Phi_{calib}$ track.
2. **The "Arrogance of Language Priors" (Entropy Gate Blindspot):** Relying on model confidence to trigger intervention assumes the LLM is unconfident when hallucinating. Highly confident hallucinations driven by biased language priors will completely bypass the entropy gate.
3. **VASM’s Polysemy & Dictionary Exhaustion:** Operating as a brute-force superset for polysemy during partial generation context is an inherent limitation. Furthermore, if a valid visual noun is out-of-vocabulary (OOV) or bizarrely tokenized, it receives $\gamma=0$, rendering BRA completely inactive for that entity.
4. **Batched Latency and Memory Bandwidth Trap:** Despite entropy-gating and bounding the candidate window to $M$, dynamic BPE mask lookups and moving-median spatial pooling inherently impact VRAM and throughput, severely risking OOM at larger batch sizes.
5. **Out-of-Candidate Unrecoverability:** BRA mathematically reweights the LLM's initial candidate distribution. It cannot synthesize a visually grounded token if the base model completely suppresses it from the initial Top-$M$ candidate window.

## 6. Conclusion and Future Execution

This paper outlines Bounded Region Alignment (BRA), a rigorously defined framework for mitigating MLLM hallucination through structure-preserving, token-local logits intervention. By mapping embedding asymmetry (`BRA_zero` vs `BRA_calib`), utilizing threshold-gated Adaptive Top-$k$ pooling (anchored by robust moving medians), engineering robust language protection via VASM BPE inheritance, and tracking computational bottlenecks via dynamic entropy calibration, we establish a technically precise foundation for decode-time visual control. The claims herein are strictly governed by our hypothesis-driven evaluation protocol spanning Hallucination, Structure, and Local Evidence chains, establishing a verifiable roadmap for empirical execution.

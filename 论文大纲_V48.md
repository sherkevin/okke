# Dictionary-Gated Token-Local Visual Intervention for Hallucination Reduction in Image-LLMs

## Abstract
Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when overarching language priors overshadow the localized visual evidence required for dense spatial grounding. Existing inference-time mitigation strategies (e.g., DoLa, VCD, OPERA) successfully regularize global token probabilities, layer contrasts, or attention distributions; however, they are orthogonal to the explicit routing of spatial evidence. This paper investigates how to strictly inject **token-local visual evidence** directly into the terminal logits space at decode time without damaging functional language structure. We present a functional **Token-Local Visual Intervention** framework—a dictionary-gated logits adjustment utilizing localized visual support extracted from contextualized final-layer hidden states. 

Strictly scoped to 2D Image-LLMs to secure defensible claims regarding spatial locality, this framework addresses three structural barriers. First, acknowledging the spatial washout inherent to deep self-attention, we establish an evaluation boundary separating a zero-shot diagnostic (`Local_Zero`) from a lightweight, dynamically calibrated bound (`Local_Calib`). We explicitly formalize `Local_Calib` as a **hybrid test-time adaptation** requiring an isolated 5k-sample prior, departing from purely training-free paradigms. Second, to prevent decoding latency explosions from spatial pooling, we extract localized support via an **Entropy-Scaled Adaptive Top-$k$** mechanism. By dynamically sizing the activation threshold during the prefill stage based on visual hidden-state entropy, we avoid the brittleness of static percentiles and isolate heavy dictionary dot-products to the Time to First Token (TTFT) phase. Third, to protect functional syntax, we implement a **Dictionary-Gated Semantic Masking** module (a dual-tier shield utilizing an ~85k WordNet physical entity dictionary and deterministic regex triggers for OCR structures) instrumented with dynamic Byte-Pair Encoding (BPE) continuation tracking.

To rigorously assess both algorithmic overhead and genuine spatial utility, we propose a pre-registered evaluation protocol across three definitive evidence chains: (A) Hallucination Reduction isolating mechanisms against a strictly parameter-matched `Base + 5k LoRA` control, (B) Structure and Reasoning Preservation explicitly profiling out-of-vocabulary (OOV) intervention bypass rates to avoid conflating "preservation" with "inaction", and (C) Local Evidence Value strictly comparing calibrated localized pooling against global average pooling (`Local_MeanPool`). Alongside transparent defense lines dissecting $\mathcal{O}(M \cdot N_v \log k)$ CUDA sorting bottlenecks and Dictionary Size vs. TTFT scaling, this framework establishes a scientifically rigorous proposition for spatially-aware, decode-time intervention.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable visual-linguistic capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to establish that these methods serve as highly successful, state-of-the-art regularizers that effectively discipline overarching language-side or attention-side generation heuristics. We tackle an orthogonal problem: spatial evidence routing. Because existing methods operate on global generation dynamics, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, functional hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination faithfully, provided it incorporates explicit structure-preserving safeguards. We functionalize this intervention as **Token-Local Visual Intervention**. In this work, we strictly bound our scope to Image-LLMs and 2D spatial reasoning, deliberately discarding unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims surrounding spatial locality.

Executing this intervention requires acknowledging and mathematically overcoming three structural barriers. First, **input-output embedding asymmetry** and **spatial washout**. Deep self-attention inevitably diffuses localized patch coordinates by the final layer, meaning a purely training-free spatial projection is often mathematically invalid. Second, the computational complexity of **dynamic spatial pooling** at every decoding step. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, functional syntax, or OCR-critical entities (e.g., numbers, proper nouns).

Our framework functionally resolves these barriers:
1. **Bi-Level Logits Intervention (`Local_Zero` / `Local_Calib`)**: We establish a strict offline evaluation boundary testing whether final-layer visual states natively align with the terminal vocabulary (`Local_Zero`). Acknowledging that architectures like LLaVA-1.5 fail this zero-shot diagnostic due to spatial washout, we explicitly shift the method into a **hybrid test-time adaptation** (`Local_Calib`) requiring a lightweight, isolated 5k-sample prior to realign spatial features.
2. **Entropy-Scaled Adaptive Top-$k$ Pooling**, extracting localized spatial support. To replace arbitrary static thresholds, we derive a dynamic activation threshold based on the visual hidden-state entropy strictly during the prefill stage. This adapts to image density while preventing redundant threshold calculations during autoregressive decoding.
3. **Dictionary-Gated Semantic Masking**, utilizing dynamic BPE inheritance and a dual-tier structural shield: an ~85k WordNet physical entity dictionary and deterministic regular expressions for OCR constructs. We directly confront this semantic mask's conflation of "preservation" and "inaction" by transparently tracking explicit Intervention Trigger Rates (%) against Out-Of-Vocabulary (OOV) terms.

To ensure rigorous scientific boundaries, our experimental design is organized into a pre-registered execution protocol defining three primary evidence chains:
- **Chain A (Hallucination Reduction):** Validating mechanism value on POPE, CHAIR, and AGL against a strictly parameter-matched `Base + 5k LoRA` baseline to isolate decode-time routing gains from simple parameter exposure.
- **Chain B (Structure and Reasoning Preservation):** Proving masking dynamics via MMBench and MMMU(Hard), openly profiling OOV intervention bypass rates.
- **Chain C (Local Evidence Value):** Demonstrating the definitive superiority of localized evidence on dense tasks (FREAK, DocVQA) by strictly comparing against a global average baseline (`Local_MeanPool`), bound by explicit Inter-Token Latency (ITL) and Dictionary-Size Time-to-First-Token (TTFT) ablation limits.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Embedding Asymmetry and Spatial Washout

A fundamental challenge in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, we evaluate the *contextualized* final-layer transformer hidden states corresponding to the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$.

We explicitly address "spatial washout"—the reality that deep self-attention often destroys localized visual coordinates by the final layer. We define an operational boundary governed by a feature-level **Washout Threshold**, evaluated offline once per architecture:

1. **`Local_Zero` (The Diagnostic Null-Hypothesis)**: A zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$. 
2. **The Washout Threshold**: We test zero-shot bounding box retrieval on a held-out 500-sample spatial validation set. If the native projection fails to achieve an average bounding box $\text{IoU} > 0.15$ (a standard threshold for coarse object localization in low-resolution final-layer feature maps), spatial fidelity is mathematically declared lost. 
3. **`Local_Calib` (Hybrid Test-Time Adaptation)**: Upon triggering the threshold, we pivot to a hybrid adaptation. $\Phi_{calib}$ is a 2-layer MLP projection ($\approx 4.2$M parameters) optimized on 5,000 image-patch to bounding-box label pairs. **Crucially, this is not a training-free method**; these pairs are size-stratified and sampled from Visual Genome subsets strictly disjoint from test datasets. The base LLM remains entirely frozen.

**Strict Semantic & Spatial Negative Sampling:** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. To prevent feature collisions in dense images, negative patches ($W_{vocab}[c^-]$) are strictly filtered. A valid negative must satisfy two conditions: (1) Membership in a differing semantic super-category, and (2) An absolute spatial exclusion zone of $\text{IoU} < 0.1$ with the positive anchor. 

### 3.2 Entropy-Scaled Adaptive Top-$k$ Pooling

At decoding step $t$, a naive Softmax over all visual patches artificially inflates noise. We apply a threshold-gated Sigmoid activation:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{active}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{active}\Big)
$$
To calculate $\theta_{active}$ reliably without relying on an arbitrary static percentile (which truncates valid objects in dense images and elevates noise in sparse ones), we derive an **entropy-scaled dynamic threshold strictly during the prefill stage**. For each final-layer visual token $v_j$, we compute its maximum visual-to-semantic activation across the targeted dictionary $\mathcal{V}_{dict}$: $A(v_j) = \max_{c \in \mathcal{V}_{dict}} \left( W_{vocab}[c] \cdot \text{LayerNorm}(h_L^{(v_j)}) \right)$.

We then calculate the visual hidden-state entropy $\mathcal{H}_{vis}$ over the spatial grid. The threshold is dynamically defined as:
$$
\theta_{active} = \mu_A + \lambda(\mathcal{H}_{vis}) \cdot \sigma_A
$$
where $\mu_A$ and $\sigma_A$ are the mean and standard deviation of the prefill activations, and $\lambda(\mathcal{H}_{vis})$ is a scaling factor inversely proportional to visual entropy. This allows the threshold to drop in dense, high-entropy images (preserving overlapping objects) and rise in sparse images (filtering background noise). 

Isolating this heavy $\mathcal{V}_{dict}$ dot product to prefill prevents prohibitive latency spikes during decoding. We explicitly acknowledge **Prefill Threshold Attrition**: contextualized visual states may drift during highly extended autoregressive generation, potentially eroding the efficacy of the prefill-derived threshold. During decoding, we apply an Adaptive Top-$k$ rule to aggregate only the supportive activated patches, requiring an $M \times N_v$ sorting operation at each step.

### 3.3 Dictionary-Gated Semantic Masking

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. This penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$

To prevent structural collapse while ensuring the intervention does not remain completely dormant on dense OCR or reasoning tasks, we utilize a **Dual-Tier Protection Architecture**:
1. **Tier 1: Root Dictionary Expectation:** We define the visual noun dictionary offline using the WordNet *physical entity* subtree ($\approx 85,000$ synsets). 
2. **Tier 2: Deterministic Format Triggers:** Acknowledging that WordNet excludes numbers, dates, and proper nouns—rendering visual grounding null on datasets like DocVQA—we supplement the mask with deterministic regex logic. Tokens conforming strictly to structural formats (e.g., numerical strings, capitalized entities) are assigned $\gamma=1$.
3. **Dynamic Continuation Inheritance:** Tokenizers frequently split common nouns using prefix spaces or subword bytes. We decode the rolling token window and apply greedy string matching against dictionary/regex entries to resolve continuation inheritance. Any subword fragment inherits the exact $\gamma$ weight of its root via a pre-compiled GPU-side bitmask.

**The Polysemy & OOV Concession:** We explicitly recognize that deterministic gating cannot natively resolve polysemy (e.g., the financial "bank" vs. a river "bank"). Furthermore, domain-specific jargon missing from both tiers will inherently bypass intervention ($\gamma=0$). We commit to empirically quantifying explicit Out-Of-Vocabulary bypass rates.

## 4. Evaluation Protocol and Experimental Design

Because Token-Local Visual Intervention introduces specific structural hypotheses—and acknowledges severe risks of Inter-Token Latency (ITL) bottlenecks and intervention bypass if unverified—we organize the evaluation plan into predefined falsifiable evidence chains. The protocol utilizes **LLaVA-1.5** (MLP projector) and **Qwen-VL-Chat** (2D-RoPE). All subsequent sections outline our planned, hypothesis-driven execution. 

### 4.1 Chain A: Hallucination Reduction via Step-by-Step Attribution

**Hypothesis:** The intervention mitigates object hallucination specifically through localized decode-time routing, not merely through the parameters exposed during calibration.
- **Evaluation Protocol:**
  - **Metrics:** POPE (Acc/F1, and False Positive Rate), CHAIR (Object vs. Attribute), and AGL (Attention-Grounded Localization).
  - **Planned Output (Table 1 - Strict Format):** 
    - *Columns:* Model | POPE (F1) | **POPE (FPR)** | CHAIR (Obj) | AGL (Acc)
    - *Rows:* Base | `Base + 5k LoRA` | DoLa* | VCD* | OPERA* | `Local_MeanPool` | `Local_Calib (Base)` | `Local_Calib (+Masking)`
  - **The Comparison Fairness Context:** We explicitly note (*) that DoLa, VCD, and OPERA are zero-shot, orthogonal regularizers managing global heuristics. `Local_Calib` utilizes a 5k-sample prior. 
  - **The Strict 5k LoRA Control:** To prevent baseline disparity, the `Base + 5k LoRA` control is optimized on the exact same 5k VG pairs with an equivalent auxiliary InfoNCE spatial objective, matching optimizer states and a LoRA rank equivalent to $\Phi_{calib}$'s 4.2M parameters. This definitively isolates inference-time spatial grounding gains. 
  - **False Positive Defense:** Reporting the False Positive Rate (FPR) for POPE is mandatory to ensure the method is not merely inflating accuracy by artificially penalizing all "Yes" tokens.

### 4.2 Chain B: Structure Preservation and the Inaction Conflation

**Hypothesis:** The dual-tier semantic mask protects structural tokens, but reliance on explicit dictionaries introduces a vulnerability where Out-Of-Vocabulary (OOV) terms bypass intervention entirely, conflating "preservation" with mere "inaction."
- **Evaluation Protocol:**
  - **Benchmarks:** MMBench V1.1, MME, MMMU (Hard Subset).
  - **Intervention Trigger Rate Profiling (Figure 1):** We will construct a scatterplot with **X-axis = OOV Rate (%)** and **Y-axis = Accuracy Drop relative to Base**. Point sizes explicitly represent the **Intervention Frequency (%)**. If the intervention rate on MMMU drops below 5%, we will explicitly state: *"The method maintains reasoning performance on MMMU primarily by remaining safely dormant due to out-of-vocabulary terms."*

### 4.3 Chain C: Local Evidence Value & Dense Spatial Evaluation

**Hypothesis:** On dense spatial tasks, token-local visual reweighting isolates specific evidence that diffuse global pooling natively washes out.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (Accuracy and ANLS) and DocVQA (ANLS). *Note: The Tier 2 Regex expansion specifically enables active intervention on DocVQA's dense numerical/OCR answers.*
  - **Baseline Comparison (Table 2):** Strictly isolating `Local_Zero` (if viable), `Local_MeanPool` (naive global averaging), and `Local_Calib` (Top-$k$). 
  - **The Strict Latency/Trigger Constraint:** Table 2 will include a mandatory **ITL Multiplier** column and an **Intervention Trigger Rate (%)** column. We establish a strict execution threshold: if the trigger rate is >15% but `Local_MeanPool` remains within 1-3% of the Top-$k$ variant, the $\mathcal{O}(M \cdot N_v \log k)$ sorting overhead is scientifically unjustified. A >3% gap is required to validate the core spatial locality hypothesis.

### 4.4 Defense Lines: Algorithmic Bottlenecks and Context Scaling

**Hypothesis:** The $85k \times N_v$ dictionary dot-product bottlenecks prefill TTFT, and executing $M \times N_v$ Top-$k$ sorts at decoding bottlenecks ITL. 
- **Evaluation Protocol (Figures 2 & 3):** Executed on a single A100 (80GB) GPU using BF16 precision and FlashAttention-2. 
- **Dictionary Size vs. TTFT Ablation:** We will ablate the size of the targeted dictionary $\mathcal{V}_{dict}$ (10k, 50k, 85k WordNet subsets) against the Time to First Token (TTFT) and POPE F1. We must empirically prove whether an 85k dot product is strictly necessary for accuracy, or if a smaller subset suffices to save prefill latency.
- **Latency Profiling:** We plot TTFT and ITL against an expanding Candidate Window Size **$M \in \{10, 50, 100, 200\}$**. 
- **Context Length Scaling:** We include a secondary plot showing TTFT explosion as Image Resolution ($N_v$) increases, explicitly tracking Batched Out-of-Memory (OOM) failures for Batch Sizes > 1 to formally bound operational limits.

## 5. Discussion and Limitations

The strength of Token-Local Visual Intervention lies in its structurally bounded proposition: injecting token-local visual evidence at decode-time while protecting language integrity via dictionary-gated tracking. However, its execution is governed by theoretical and operational constraints:

1. **The Hybrid Shift:** Deep self-attention demonstrably washes out localized patch fidelity by final layers. For architectures triggering the offline Washout Threshold, the premise of zero-shot alignment is mathematically invalid. This explicitly shifts the method into a hybrid test-time adaptation requiring a 5k-sample prior, fundamentally differentiating it from pure inference-time regularizers like DoLa or VCD.
2. **Prefill Threshold Attrition & Masking Brittleness:** Deriving $\theta_{active}$ solely during prefill using visual entropy prevents autoregressive latency spikes, but risks threshold attrition during extended long-context generation. Furthermore, while the dual-tier dictionary covers physical entities and regex formats, niche domain jargon safely defaults to $\gamma=0$, conflating "structure preservation" with "intervention bypass"—an outcome we openly track.
3. **Latency Bottlenecks:** Executing spatial pooling and $M \times N_v$ Top-$k$ sorting at every single decoding step incurs severe CUDA kernel overheads. While the dictionary dot-product induces a TTFT spike that scales quadratically with vocabulary size and image resolution ($N_v$), ITL becomes a strict bottleneck during autoregressive generation. Testing large batch sizes risks VRAM OOM failures, positioning this framework strictly as a high-precision analytical tool rather than a high-throughput serving API.

## 6. Conclusion and Future Execution

This paper outlines a functionally precise framework for mitigating MLLM hallucination through dictionary-gated, token-local logits intervention. By formalizing spatial washout (`Local_Zero` vs `Local_Calib`), computing an entropy-scaled prefill threshold, and engineering robust language protection via deterministic semantic masking, we establish a technically rigorous foundation for decode-time visual routing. To objectively assess algorithmic bottlenecks and prevent conflating inaction with preservation, the claims herein are strictly governed by our pre-registered evaluation protocol—mandating exact Intervention Trigger Rate logging, strict `Base + 5k LoRA` isolation, and ITL transparency—establishing a verifiable roadmap for empirical execution.

# Token-Local Resonance Anchoring: A Decode-Time Visual Logit Bias for Entity-Centric MLLM Grounding

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. While existing inference-time mitigation strategies modulate internal model confidence, they do not provide a framework for injecting **token-local visual evidence** directly into decode-time logits adjustment without damaging syntax. We present **Token-Local Resonance Anchoring (TLRA)**, a hybrid calibration and decode-time intervention framework that reweights candidate tokens using localized support extracted from cached visual states.

TLRA operates on three principles. First, **token-local evidence reweighting** replaces coarse global visual contexts with candidate-wise support, mapping spatial patches to lexical logit biases. Second, **Adaptive Top-k resonance** stabilizes local evidence extraction under a bounded candidate window with dynamic logit clipping. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax via an offline BPE-level dictionary strictly bound to physical entity synsets. We rigorously delineate a training-free geometric isotropy probe (`TLRA_zero`) from a lightweight calibrated variant (`TLRA_calib`). To ensure evaluation fairness, `TLRA_calib` is tested against a matched-budget `Base + LoRA (LLM & Projector)` baseline trained on identical calibration data. 

The pre-registered evaluation protocol enforces a falsifiable audit of structural vulnerabilities and computational boundaries. It measures hallucination reduction against Average Generation Length (AGL) collapse, audits VASM's noun-specific bias by testing against action/verb hallucinations, and isolates the value of token-local routing against a Global Visual State baseline. By tracking explicit memory bandwidth utilization during the per-step gather operation—particularly identifying the L2 cache thrashing boundary for High-Definition (HD) MLLMs ($N_v \ge 3000$)—this work establishes a strict, boundary-aware framework for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable generative capabilities, yet their autoregressive decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible but weakly supported by the visual input. This failure is particularly damaging in **fine-grained physical grounding** scenarios—such as small-object recognition, counting, and spatial positioning—where the requisite evidence is highly localized rather than global.

Current inference-time mitigation strategies represent major advances. Contrastive decoding methods compare distributions under perturbed and clean inputs. Attention-based control methods penalize pathological autoregressive decay. Layer-contrast methods exploit discrepancies between shallow and deep representations. While these approaches successfully modulate general model confidence, they leave a specific, complementary question unanswered: **How can we inject token-local visual evidence directly into decode-time logits adjustment without destroying syntax or overwhelming memory bandwidth?**

Executing localized intervention at decode time is nontrivial due to semantic fragility and hardware physics. If an intervention suppresses visually unsupported tokens without understanding language structure, it risks destroying functional syntax. Furthermore, computing cross-modal similarity between $M$ candidate tokens and $N_v$ spatial visual patches at every single autoregressive step introduces a severe latency bottleneck. In modern HD MLLMs processing thousands of patches and massive vocabularies, naive intervention devolves into an uncoalesced memory gather operation that thrashes the GPU's L2 cache. Finally, if localized decode-time interventions fail to outperform simple prefill-time adaptations (e.g., LoRA) or global visual pooling, their architectural complexity is mathematically unjustified.

We introduce **Token-Local Resonance Anchoring (TLRA)**, a hybrid framework combining a lightweight visual-to-vocabulary calibration with an efficient decode-time routing mechanism explicitly bounded to **entity-centric grounding**. TLRA avoids the per-step matrix multiplication bottleneck by conceptualizing the intervention as a **static late-fusion visual logit bias**: visual-to-lexical projections are computed once post-prefill. To prevent structural collapse, TLRA introduces Vocabulary-Anchored Semantic Masking (VASM), an offline BPE-level dictionary explicitly mapping physical entity synsets to binary subword masks, restricting the intervention to groundable nouns while shielding functional syntax, verbs, and prepositions.

To ensure methodological rigor, we enforce strict architectural boundaries. We formally interrogate the geometry of the joint multimodal input space via a zero-shot probe (`TLRA_zero`), contrasted alongside a calibrated variant (`TLRA_calib`). `TLRA_calib` is subsequently evaluated against an internal matched-calibrator control (`Global_Visual_State`) and a matched-budget `Base + LoRA` baseline to isolate the precise value of decode-time locality. Finally, we explicitly audit the semantic fragility of VASM by comparing it against an `Oracle_VASM` (online POS tagger) and a random baseline, while subjecting the system to a strict scaling stress test to quantify the exact memory bandwidth degradation threshold on high-resolution ($N_v > 3000$) images.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Architectural Scope and the Joint-Space Geometry Probe

TLRA relies on the extraction of token-local support from spatial visual states. It strictly targets standard ViT + MLP projector setups (e.g., LLaVA) where spatial provenance is preserved. Architectures that heavily compress spatial dimensions (e.g., Q-Formers) fall outside the supported scope. To rigorously isolate the source of visual grounding improvements, we separate TLRA into two distinct variants:

1. **`TLRA_zero` (Geometric Isotropy Probe):** The MLLM's MLP projector maps spatial visual features ($X_v$) into the same dimensional space as the discrete text embeddings ($E_{in}$), serving as joint inputs to Layer 1 of the LLM. `TLRA_zero` computes the cosine similarity directly between $X_v$ and $E_{in}$. We acknowledge a critical geometric limitation here: these representations are trained to be *processed* by subsequent transformer layers, not geometrically aligned for dot-product retrieval. `TLRA_zero` serves strictly as an analytical probe to test whether the joint input space naturally exhibits semantic isotropy. If a pre-intervention similarity distribution check yields noise, `TLRA_zero` fails, and the framework defaults entirely to the calibrated variant.
2. **`TLRA_calib` (Hybrid Calibrated Variant):** A lightweight linear projection `Phi_calib` maps cached spatial visual states directly to the lexical logit space $V$, trained on a bounded conceptual-caption subset while the base MLLM remains frozen. 

Because `Phi_calib` introduces external supervision, any claim about decode-time local routing must survive strict fairness checks. `TLRA_calib` is benchmarked against a `Base + LoRA (LLM & Projector)` model adapted on the identical calibration subset, proving that dynamic decode-time visual grounding achieves something static prefill-time adaptation cannot.

### 3.2 Static Late-Fusion and the HD Memory Bandwidth Wall

To avoid prohibitive FLOP penalties inside the autoregressive loop, TLRA precomputes the visual-to-lexical projection. After the prefill phase, the cached visual states are mapped to the full vocabulary dimension $V$, yielding a static visual logit bias matrix $B \in \mathbb{R}^{N_v \times V}$. This operation is executed exactly once per image.

At each decoding step $t$, TLRA restricts intervention to a bounded candidate set $\mathcal{C}_t = \text{Top-}M(L_{orig})$, where $M$ is a globally locked candidate window (e.g., $M=50$). The routing overhead per step is the extraction of $M$ columns from $B$. 

**The Memory Bandwidth Wall:** While FLOP-efficient, gathering from $B$ exposes a severe hardware boundary. For standard resolution ($N_v \approx 1024$, $V \approx 128k$), $B$ is $\approx 250$MB. However, for HD MLLMs processing $N_v \ge 3000$ patches with $V \ge 150k$, $B$ balloons to $> 1.2$GB. Gathering arbitrary token columns from a 1.2GB uncoalesced block token-by-token thrashes the GPU L2 cache. TLRA explicitly profiles this memory bandwidth degradation across $N_v$ scales to determine the strict resolution limits of late-fusion logit biases.

For each candidate $c \in \mathcal{C}_t$, TLRA aggregates local support from the top spatial patches:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{B_{j, c}}{\tau_{sim}}, \quad \text{where } k = \max(k_{min}, \lceil \rho \cdot N_v \rceil)
$$

### 3.3 Stabilized Resonance Penalty with Logit Clipping

We convert raw local support into a relative evidence score within the candidate window: $\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)$. Intervening directly based on candidate logit spread $\Delta_L = \max(L_{orig}) - \min(L_{orig})$ can destroy spiked distributions. TLRA applies a clipped penalty:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \min(\Delta_L, \delta_{max}) \cdot (1 - \hat S(c))
$$
where $\delta_{max}$ bounds the maximum logit penalization, $\alpha$ scales the intervention intensity, and $\gamma(c) \in \{0, 1\}$ is the structural protection mask.

### 3.4 VASM: Entity-Centric Masking and Noun Bias

To preserve syntax and multi-token entities without triggering massive latency penalties, TLRA utilizes VASM, an offline precomputed deterministic binary mask ($\gamma \in \{0, 1\}^V$). Generating this mask requires a reproducible BPE pipeline: we extract physical entity synsets (nouns mapping to physical objects/animals/structures) from English WordNet. Subwords inherit the mask state of their root token. 

**Explicit POS Scope and Noun Bias:** This process typically masks $\approx 85\%$ of the vocabulary ($\gamma=0$). We explicitly define TLRA as an **Entity-Centric** grounding mechanism. Because VASM strictly protects physical entities ($\gamma=1$), it systematically ignores actions (verbs), attributes (adjectives), and spatial relations (prepositions). Consequently, TLRA cannot mitigate verb or relation hallucinations without risking catastrophic syntactic collapse. We quantify the exact cost of this semantic fragility by testing against verb-hallucination stress tests and an `Oracle_VASM` (online POS tagger), alongside a `TLRA_Random_VASM` negative control.

## 5. Discussion and Limitations

The viability of Token-Local Resonance Anchoring depends entirely on surviving strict computational, structural, and hardware boundaries. By explicitly reframing TLRA as a hybrid calibration/routing method tailored for entity-centric grounding, we identify the following primary limitations:

1. **The Noun-Bias and Semantic Fragility of VASM:** Offline dictionary lookups inherently fail on context-dependent polysemy and cannot encompass the full spectrum of grounding. By forcing $\gamma(c) = 0$ for everything outside physical entities, TLRA is fundamentally blind to action hallucinations (verbs) and relation hallucinations (prepositions). The performance gap on the Verb Stress-Test and against Oracle VASM formally defines the strict entity-centric boundaries of this method.
2. **The Geometric Isotropy of `TLRA_zero`:** Assuming zero-shot dot-product alignment between Layer 0 visual features ($X_v$) and input token embeddings ($E_{in}$) relies on an untested assumption of representation isotropy across the joint multimodal input space. If the initial similarity distribution probe yields random noise, it proves that MLP-projected visual tokens require subsequent transformer layers to contextualize, rendering the `TLRA_zero` probe geometrically invalid.
3. **The HD Memory Bandwidth Wall:** While the static late-fusion $B$ matrix circumvents dense per-step FLOPs, gathering arbitrary token columns at every autoregressive step is intensely memory-bound. As modern models scale visual patches ($N_v \ge 3000$) and vocabularies ($>150k$), the 1.2+ GB uncoalesced gather operation threatens to thrash the L2 cache, potentially cratering actual `tokens/s`. 
4. **The `Global_Visual_State` Threshold:** If the empirical gap between `TLRA_AdaptiveTopK` and uniformly pooled `Global_Visual_State` is negligible under identical calibration budgets, it falsifies the core need for fine-grained patch-level routing at decode time.
5. **Top-M Unrecoverability (The Hijacking CDF):** The intervention is mathematically bounded by the base model's candidate window. If strong language priors marginalize the ground-truth visual token below the Top-$M$ cutoff *prior* to intervention, TLRA cannot rescue it.

Future execution of this protocol will definitively confirm whether local decode-time visual intervention can exist as a practical, context-aware mechanism, or if memory and semantic trade-offs restrict it to a theoretical probe.

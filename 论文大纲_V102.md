# Token-Local Resonance Anchoring: A Bounded Late-Fusion Visual Logit Bias for Entity-Centric MLLM Grounding

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. While existing inference-time mitigation strategies modulate internal model confidence, they do not provide a framework for injecting **token-local visual evidence** directly into decode-time logits adjustment without damaging syntax or overwhelming hardware bandwidth. We present **Token-Local Resonance Anchoring (TLRA)**, a late-fusion decode-time intervention framework that acts as a linear visual classifier bias, reweighting candidate tokens using absolute localized support extracted from cached visual states.

TLRA operates on three strict architectural principles. First, a **lightweight late-fusion calibrated projection** maps spatial patches to BPE-level logit biases. We mathematically define the High Bandwidth Memory (HBM) crossover point ($N_v = D$): static precomputation is strictly optimal for standard resolutions, while dynamic sliced-matmul is reserved exclusively for ultra-HD inputs ($N_v > 4096$). Second, **Absolute Evidence Scoring** applies an independent Sigmoid activation over bounded candidate windows to penalize visually unsupported tokens without mathematically granting "free passes" to universally hallucinated sets. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax via an offline BPE-level dictionary explicitly bound to physical entity synsets. VASM incorporates a **Subword Continuation Rule** that anchors only the morphological prefix, preventing mid-word autoregressive truncation while accounting for tokenizer leakiness.

To ensure methodological rigor, we enforce strict architectural boundaries across three evidence chains. `TLRA_calib` is tested against a matched-parameter `Global_Visual_State` baseline and a `Base + LoRA` control trained on identical data. An untrained `TLRA_zero` negative control empirically verifies input-space geometric anisotropy. The pre-registered evaluation protocol audits structural vulnerabilities: measuring hallucination reduction against Average Generation Length (AGL) collapse, bounding VASM's semantic fragility via an Action/Verb Hallucination stress test, and mapping unrecoverable errors via a Top-M "Hijacking CDF." This work establishes a falsifiable, boundary-aware framework for decode-time visual routing.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable generative capabilities, yet their autoregressive decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible but weakly supported by the visual input. This failure is particularly damaging in **fine-grained physical grounding** scenarios—such as small-object recognition, spatial positioning, and document QA (DocVQA)—where the requisite evidence is highly localized rather than global.

Current inference-time mitigation strategies represent major advances. Contrastive decoding methods compare distributions under perturbed and clean inputs, while attention-based and layer-contrast methods penalize pathological autoregressive decay or exploit discrepancies between shallow and deep representations. While these approaches successfully modulate general model confidence, they leave a specific, complementary question unanswered: **How can we inject fine-grained, token-local visual evidence directly into decode-time logits adjustment without destroying syntax or hitting severe hardware memory walls?**

Executing localized intervention at decode time is nontrivial due to semantic fragility, geometric misalignment, and hardware physics. If an intervention suppresses visually unsupported tokens without understanding language structure, it risks destroying functional syntax. Furthermore, computing cross-modal similarity between $M$ candidate tokens and $N_v$ spatial visual patches at every single autoregressive step introduces a severe latency bottleneck. Finally, if localized decode-time interventions fail to outperform simple prefill-time adaptations (e.g., LoRA) or global visual pooling, their architectural complexity is mathematically unjustified.

We introduce **Token-Local Resonance Anchoring (TLRA)**, a strictly bounded late-fusion framework functioning as a token-local visual classifier ensemble. TLRA extracts absolute local spatial support to bias decode-time logits, explicitly limited to **entity-centric grounding**. To prevent structural collapse, TLRA introduces Vocabulary-Anchored Semantic Masking (VASM), an offline BPE-level dictionary explicitly mapping physical entity synsets to binary subword masks. VASM restricts the intervention to the initial stems of groundable nouns while shielding functional syntax, verbs, prepositions, and morphological suffixes.

To ensure methodological rigor, we structure the evaluation as a falsifiable audit organized into **three evidence chains and five defenses**:
1. **Hallucination Reduction:** Evaluating standard entity grounding (POPE, CHAIR) while explicitly bounding truncation via Average Generation Length (AGL).
2. **Structure and Reasoning Preservation:** Auditing VASM's protection of general capabilities (MMBench, MMMU) and quantifying tokenizer leakiness via a BPE Collision Audit.
3. **Local Evidence Value:** Isolating the spatial advantage on FREAK and DocVQA by comparing `TLRA_calib` against a matched-budget `Global_Visual_State` uniform pooling control.
4. **Semantic and Geometric Boundaries:** Testing the structural limits via an Action/Verb Hallucination stress test (proving VASM's noun-bias limitation) and a `TLRA_zero` negative control (proving representation anisotropy).
5. **Hardware Physics:** A formal derivation of the HBM uncoalesced read crossover point, proving exactly when dynamic sliced-matmul overtakes static precomputation ($N_v > D$).

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Architectural Scope and the Calibrated Late-Fusion Head

TLRA targets standard ViT + MLP projector architectures (e.g., LLaVA) where spatial provenance is preserved across visual tokens $X_v$. To rigorously isolate the source of visual grounding improvements, we separate TLRA into a trained implementation and a strict theoretical baseline:

**1. `TLRA_calib` (The Primary Method):** Stripped of complex routing jargon, `TLRA_calib` is a late-fusion visual classifier ensemble. A lightweight linear projection matrix $W_{calib} \in \mathbb{R}^{D \times V}$ maps cached spatial visual states of dimension $D$ directly to the lexical vocabulary size $V$. 
- **Training Formulation:** $W_{calib}$ is trained on a bounded subset of 50k conceptual captions (e.g., sampled from LLaVA-Instruct). The base LLM and Vision Encoder remain entirely frozen. The objective is standard next-token prediction cross-entropy, but the loss is **strictly masked** to only backpropagate on tokens where VASM $\gamma(y_t) = 1$ (physical entities). This forces $W_{calib}$ to act purely as an entity-grounding expert without learning general syntax. $\mathcal{L}_{calib} = -\sum_{t} \gamma(y_t) \log P(y_t \mid X_v, Y_{<t})$.

**2. `TLRA_zero` (Negative Control):** Computes cosine similarity directly between frozen post-projector visual features ($X_v$) and the LLM's discrete text embeddings ($E_{in}$). As the MLP projector maps visual tokens into the first transformer layer's heavily anisotropic activation space, symmetrical dot-product retrieval against a static vocabulary is geometrically incoherent. `TLRA_zero` is included not as a viable variant, but strictly as a pre-registered negative control to empirically formalize this representation anisotropy against `TLRA_calib`.

### 3.2 Navigating the HBM Wall: The Exact $N_v = D$ Crossover

At each decoding step $t$, TLRA maps spatial features $X_v$ to lexical biases for a bounded candidate set $\mathcal{C}_t = \text{Top-}M(L_{orig})$. We evaluate two distinct architectural implementations to navigate GPU hardware physics:

**A. Static Precomputation ($N_v \le D$):** After the prefill phase, cached visual states are mapped to the full vocabulary dimension, yielding a static bias matrix $B = X_v \cdot W_{calib}$, where $B \in \mathbb{R}^{N_v \times V}$. Gathering arbitrary token columns $c \in \mathcal{C}_t$ from $B$ exposes a hardware boundary via uncoalesced reads from High Bandwidth Memory (HBM).
**B. Dynamic Sliced-Matmul ($N_v > D$):** We dynamically compute the projection *only* for the Top-$M$ candidates at each step: $B_{step} = X_v \cdot W_{calib}[:, \mathcal{C}_t]$. This reads from $W_{calib} \in \mathbb{R}^{D \times V}$ instead of $B$.

**The Mathematical Crossover:** A direct size comparison reveals the exact hardware boundary. Assuming fp16, the size of $B$ is proportional to $N_v \times V$, while $W_{calib}$ is proportional to $D \times V$. For standard architectures (e.g., LLaVA $D=4096$) processing standard HD images ($N_v \approx 3000$), $B$ is physically **smaller** than $W_{calib}$ (900MB vs 1.22GB). Therefore, static precomputation is strictly superior for standard use cases. Dynamic sliced-matmul only becomes viable for **Ultra-HD MLLMs** where visual patches exceed the hidden dimension ($N_v > 4096$), trading FLOPs to avoid loading an exponentially exploding $B$ matrix. 

### 3.3 Absolute Evidence Anchoring with Logit Clipping

Relying on a Softmax distribution over a local candidate window mathematically risks granting a "free pass" to a universally hallucinated set of tokens. To ensure visual grounding evidence is **absolute**, TLRA computes a raw spatial support score and applies an independent Sigmoid activation:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k} B_{j, c} \quad \Rightarrow \quad \hat S(c) = \text{Clamp}\Big(\sigma\big(S_{raw}(c) \cdot \beta\big), 0, 1\Big)
$$
This absolute evidence score $\hat S(c)$ is used to applying a clipped penalty, boundedly suppressing tokens lacking spatial support without destroying spiked, high-confidence distributions ($\Delta_L$):
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \min(\Delta_L, \delta_{max}) \cdot (1 - \hat S(c))
$$

### 3.4 VASM and the Subword Continuation Rule

To preserve syntax, TLRA utilizes VASM, an offline deterministic binary mask ($\gamma \in \{0, 1\}^V$) extracted from English WordNet physical entity synsets. 

**BPE Tokenization Mismatch and Subword Continuation:** WordNet maps to words, but LLMs output BPE subwords. VASM explicitly expands the dictionary across tokenizer permutations (prefix spaces, capitalization). Crucially, to handle multi-token entities (e.g., "refrigerator" $\rightarrow$ "refrig", "erator"), VASM enforces a **Subword Continuation Rule**: $\gamma$ is set to $1$ *only* for the initial subword stem. By anchoring the semantic routing on the prefix, TLRA allows the base model's standard autoregressive decay to naturally complete the morphological suffix, entirely preventing mid-word truncation. We quantify the success of this mapping via a BPE Collision Audit.

## 5. Discussion and Limitations

The viability of Token-Local Resonance Anchoring depends entirely on surviving strict computational, structural, and hardware boundaries. By explicitly reframing TLRA as a lightweight, trained late-fusion ensemble tailored exclusively for entity-centric grounding, we identify the following mathematically rigid limitations:

1. **The Exact HBM Crossover ($N_v = D$):** Unlike parameter-free interventions, our late-fusion static precomputation mandates reading a $B \in \mathbb{R}^{N_v \times V}$ matrix. We formally concede that dynamic sliced-matmul only saves memory bandwidth if the number of visual patches exceeds the hidden dimension ($N_v > D$). For standard resolutions, static precomputation is required, introducing an uncoalesced read overhead that must be balanced by keeping $M$ small.
2. **The Noun-Bias and Semantic Fragility of VASM:** By forcing $\gamma(c) = 0$ outside physical entity stems, TLRA is fundamentally blind to action hallucinations (verbs) and relation hallucinations (prepositions). The planned Verb Stress-Test explicitly bounds the semantic utility of this method.
3. **Top-M Unrecoverability (The Hijacking CDF):** The intervention is mathematically constrained by the base model's candidate window $\mathcal{C}_t$. If strong language priors marginalize the ground-truth visual token below the Top-$M$ cutoff prior to intervention, our absolute evidence penalty cannot rescue it.
4. **The Base Anisotropy:** The `TLRA_zero` negative control demonstrates that zero-shot dot-product alignment between post-projector visual features and input embeddings is geometrically incoherent, meaning decode-time spatial routing is completely reliant on the external calibration of $W_{calib}$.

Future execution of this pre-registered protocol will definitively confirm whether local decode-time visual intervention can exist as a practical, context-aware mechanism, or if semantic fragility and HBM bandwidth walls restrict its scalable deployment.

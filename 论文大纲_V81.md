# Token-Local Resonance Anchoring: Structure-Preserving Decode-Time Intervention for MLLM Grounding

## Abstract
Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. Existing inference-time mitigation strategies—such as contrastive decoding, attention control, and layer-contrast methods—are highly competitive sequence-level heuristics. However, they leave open the question of whether injecting explicit *token-local visual evidence* directly into decode-time logits can improve physical object grounding without collapsing linguistic structure. We present **Token-Local Resonance Anchoring (TLRA)**, a decode-time intervention protocol that reweights candidate tokens using localized spatial support extracted from contextualized visual states.

TLRA operates on three structural commitments. First, it extracts token-specific spatial support via an **Adaptive Top-$k$** mechanism over a strictly bounded candidate window ($M$). Second, to avoid the prohibitive latency of autoregressive part-of-speech tagging, we introduce **Vocabulary-Anchored Semantic Masking (VASM)**—an offline, pre-computed WordNet static mask coupled with dynamic $O(1)$ Byte-Pair Encoding (BPE) continuation inheritance, ensuring functional syntax and abstract reasoning tokens are mathematically bypassed. Third, acknowledging that native visual and lexical states lack perfect alignment, we structurally separate our evaluation: `TLRA_zero` serves as a pure zero-shot viability probe, while our primary operational module, `TLRA_calib` (using a lightweight projection trained on 50k captions), is explicitly classified as a Test-Time Adaptation (TTA) method. To ensure rigorous scientific claims, we enforce a strict evaluation protocol auditing Average Generation Length (AGL) to reject truncation artifacts, utilizing DocVQA as a mathematical negative control, and testing spatial routing against a calibrated MeanPool baseline on strictly unseen object categories.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable reasoning capabilities, yet their autoregressive decoding remains vulnerable to hallucination. The model frequently generates text that is fluent and plausible but weakly supported by the actual visual input. This failure is acutely problematic in the grounding of physical entities—such as small-object recognition, spatial positioning, and counting—where the necessary visual evidence is highly localized in the patch space.

Current inference-time mitigation strategies represent major advances in the field and serve as our primary competitive baselines. Contrastive decoding methods like VCD contrast different input noise conditions to reshape distributions; attention-based control methods like OPERA penalize abnormal generation patterns; and layer-contrast methods like DoLa exploit differences between shallow and deep representations. These are highly effective *sequence-level or distribution-level language heuristics*. However, they leave a specific, complementary proposition untested: 

**Can we inject token-local spatial visual evidence into decode-time logits to outperform sequence-level heuristics on physical grounding, without destroying language structure or incurring catastrophic latency?**

Executing this token-local intervention is mechanically non-trivial due to three explicit bottlenecks:
1. **The Autoregressive Latency Paradox:** Running NLP pipelines (e.g., POS taggers) to isolate physical entity tokens at decode-time is fundamentally flawed. Taggers require complete sentences to resolve ambiguity, and executing them dynamically for $M$ candidate tokens per step incurs catastrophic latency.
2. **Embedding Asymmetry:** Visual patch states and output logits do not natively align. Bridging this gap purely zero-shot is often noisy, meaning practical implementations require external calibrators, shifting the method from pure zero-shot to lightweight adaptation.
3. **Candidate Bound & Memory Bandwidth:** Post-hoc intervention can only reweight tokens the base model keeps alive in its top-$M$ predictions. Dynamically retrieving spatial support for $M$ candidates introduces a severe memory-bandwidth (TPOT) tax.

To address these challenges, we introduce **Token-Local Resonance Anchoring (TLRA)**. TLRA utilizes an **Adaptive Top-$k$** mechanism to dynamically route localized patch support to candidate tokens within a bounded logit window. To solve the latency paradox, we introduce **Vocabulary-Anchored Semantic Masking (VASM)**—an offline, pre-computed static dictionary combined with an $O(1)$ BPE inheritance state machine. VASM mathematically zeroes out intervention on syntax, verbs, and abstract concepts, protecting the model's structural integrity.

To ensure methodological rigor, we explicitly classify our primary operational variant, `TLRA_calib` (calibrated on 50k captions), as a Test-Time Adaptation (TTA) module, separating it from the pure zero-shot probe `TLRA_zero`. Furthermore, we commit to a strict evaluation protocol built on three evidence chains: validating hallucination reduction via an Average Generation Length (AGL) audit to rule out truncation artifacts; confirming structure preservation by using dense text OCR (`DocVQA`) purely as a flat-line negative control; and proving the efficacy of *local* evidence by demanding our spatial routing outperform a calibrated global pool (`MeanPool`) strictly on *unseen* object categories.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits produced under different visual conditions. Attention-based methods such as `OPERA` detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and involve different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large spatial token fields, and in video settings they further inherit spatio-temporal structure. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region of the image or a short temporal window. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser internal alternatives. That claim is experimentally verifiable through internal ablations such as `TLRA_zero` vs. `TLRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded object words or OCR-bearing entities. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this through a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Fairness Boundary: `TLRA_zero` vs. `TLRA_calib`

Because visual patch states and output logits often lack native alignment, we separate TLRA into two tracks to prevent "calibrator blackbox" ambiguity:

1. **`TLRA_zero` (Zero-Shot Probe)**: The base model's native language head is applied directly to contextualized visual states. This track serves primarily as a viability probe and is the only variant directly comparable to pure zero-shot heuristics (DoLa, VCD) regarding parameter isolation.
2. **`TLRA_calib` (Lightweight TTA / PEFT)**: The operational variant utilizing a 2-layer MLP projection, $\Phi_{calib}$, trained on 50k conceptual captions to bridge embedding asymmetry. We explicitly acknowledge that $\Phi_{calib}$ injects external priors. Consequently, `TLRA_calib` is a Test-Time Adaptation module. Its primary ablation baseline is not VCD or DoLa, but `TLRA_MeanPool`—a variant using the exact same $\Phi_{calib}$ weights but globally pooled visual features, ensuring fair comparison.

### 3.2 Bounded Candidate Filtering and Adaptive Top-$k$ Resonance

At decoding step $t$, TLRA restricts its intervention to a bounded candidate set: $\mathcal{C}_t = \text{Top-}M(L_{orig})$, where $M \ll |V|$ is a strictly locked candidate window (e.g., $M=50$). We acknowledge that extracting patch-level support for $M$ candidates per step introduces a memory-bandwidth tax, which we explicitly audit via TPOT.

Let $h_L^{(v_j)}$ denote the contextualized final-layer visual states, with visual token volume $N_v = H \times W$. For each candidate $c \in \mathcal{C}_t$, TLRA computes local support via an Adaptive Top-$k$ rule, determined by a locked spatial ratio $\rho$:
$$
k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)
$$
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{\text{score}(v_j, c)}{\tau_{sim}}
$$

### 3.3 Structure Preservation via Offline VASM and BPE Inheritance

Naive token-level intervention suppresses critical syntax. To avoid the severe latency of decode-time NLP tagging, TLRA introduces **Vocabulary-Anchored Semantic Masking (VASM)**. VASM utilizes an offline, pre-computed static dictionary and dynamic $O(1)$ state-machine updates to guarantee intervention only targets visually groundable entities.

**1. Offline Static Dictionary:** Prior to inference, the entire vocabulary ($|V| \approx 32k - 128k$) is cross-referenced against WordNet. Tokens unambiguously mapped to physical objects/nouns are assigned a mask value $\gamma = 1$. Function words, verbs, and abstract syntax are permanently assigned $\gamma = 0$.

**2. Dynamic BPE Continuation Inheritance:** Modern tokenizers fragment words. To handle subwords at decode step $t$ for candidate $c_t$, VASM applies an $O(1)$ inheritance rule:
$$
\gamma(c_t) = \begin{cases} 
\gamma(c_{t-1}^{\text{root}}), & \text{if } c_t \text{ is a continuation subword (e.g., lacks space prefix or has '##')} \\
\text{Lookup}(c_t \text{ in WordNet}), & \text{otherwise}
\end{cases}
$$
*Limitation Concession:* Hyphenated words or severe BPE fragmentations that fail exact WordNet root matches safely default to $\gamma=0$, intentionally missing an intervention rather than corrupting syntax.

### 3.4 Stabilized Logit Intervention

TLRA converts raw local support into a relative evidence score over the candidate set:
$$
\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)
$$
The final logit is dynamically scaled by the candidate window's logit range ($\Delta_L = \max(L_{orig}) - \min(L_{orig})$), explicitly gated by the VASM mask $\gamma(c)$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \Delta_L \cdot (1 - \hat S(c))
$$
Crucially, if a token is functional syntax ($\gamma(c)=0$), the penalty zeroes out entirely, yielding $L_{final}(c) = L_{orig}(c)$. The base model's native distribution for structure remains mathematically untouched.

### 3.5 Methodological Boundaries and Negative Controls

TLRA's claims are bounded by explicit limits:
- **The OCR Concession:** Because VASM's static dictionary enforces lexical rigidity and ignores abstract text-in-image strings, TLRA is mathematically blind to OCR. Tasks like `DocVQA` serve strictly as flat-line negative controls to prove VASM does not erroneously corrupt text reading.
- **Out-of-Candidate Bound:** Post-hoc reweighting cannot recover a correct entity token if the base model has already ejected it from the initial Top-$M$ set at step $t-1$.

## 4. Evaluation Protocol and Experimental Design

To ensure methodological transparency, our evaluation is structured as a rigorous protocol mapping explicit hypotheses to specific evidence chains. **Crucially, all hyperparameters ($\tau_{sim}, \tau_{evidence}, \alpha, M, \rho$) are locked uniformly across all tasks based on a single validation set.**

### 4.1 Evidence Chain A: Hallucination Reduction and Truncation Audit
**Hypothesis:** Token-local evidence reduces physical hallucination more effectively than sequence-level heuristics, without artificially inflating precision through premature truncation.
- **Protocol (Table 1):** We evaluate on POPE (Accuracy, F1) and CHAIR (CHAIRi, CHAIRs). Baselines include Base, VCD, OPERA, DoLa, `TLRA_MeanPool`, and `TLRA_AdaptiveTopK`.
- **The AGL Audit:** We introduce a mandatory column for **Average Generation Length (AGL)**. 
- **Validation Criteria:** If `TLRA_AdaptiveTopK` reduces AGL by >10% compared to the Base model, the hallucination reduction claim must be heavily downgraded, as it would indicate the method functions as a "silence engine" (truncating outputs) rather than a true grounding engine.

### 4.2 Evidence Chain B: Structure Preservation and Negative Controls
**Hypothesis:** VASM mathematically protects abstract reasoning, and its deliberate blindness to dense text prevents unintended corruption of non-spatial tasks.
- **Protocol (Table 2):** We evaluate on general multimodal benchmarks: MMBench V1.1, MME, MMMU (Hard), and explicitly **DocVQA**. We ablate Base, `TLRA_full` (with offline VASM), and `TLRA_no_VASM`.
- **Validation Criteria:** 
  1. `TLRA_no_VASM` must show a catastrophic drop on reasoning tasks (MMMU), proving VASM is not optional.
  2. `DocVQA` must remain strictly *flat* between Base and `TLRA_full`. If `TLRA_full` drops, VASM is erroneously penalizing OCR tokens. Any claim of "improving OCR" is explicitly prohibited.

### 4.3 Evidence Chain C: Local Evidence Value (Parity Test)
**Hypothesis:** Spatial locality (Adaptive Top-$k$) directly drives grounding performance, independent of global calibration priors or randomized subset regularization.
- **Protocol (Table 3):** Evaluated on FREAK (Spatial Relations, Object Existence, Counting). The dataset is strictly partitioned into **"Seen Categories"** (objects present in $\Phi_{calib}$'s 50k dataset) and **"Unseen Categories"**. We benchmark `TLRA_MeanPool`, `TLRA_RandomK` (ablation), and `TLRA_AdaptiveTopK`, forcing all three to use the *exact same frozen $\Phi_{calib}$ weights*.
- **Validation Criteria / Regression Baseline:** The core scientific claim derives solely from this table. `AdaptiveTopK` must statistically outperform `MeanPool` on the *Unseen* split. If it only wins on the "Seen" split, the spatial routing hypothesis is invalidated as a data-leakage artifact. The paper's contribution would then cleanly devolve to: "`MeanPool + VASM` is a highly effective, structure-preserving TTA."

### 4.4 Systemic Latency Pareto and Failure Mode Audit
To quantify the memory-bandwidth tax of extracting spatial support for $M$ candidates per step:
- **Systemic Cost Pareto (Figure 2):** A scatter plot mapping Time-Per-Output-Token (TPOT in ms on an A100 GPU, X-axis) against POPE F1 (Y-axis). We plot Base, VCD, DoLa, and TLRA at $M \in \{10, 50, 100\}$.
- **Out-of-Candidate Failure Mode (Figure 3):** A line graph tracking the raw logit rank of a correct physical entity token across generation steps. This explicitly documents the physical boundary of the method: showing exactly when a correct token drops below rank $M=50$ at step $t-1$, rendering it mathematically unrecoverable by any post-hoc logits intervention.

## 5. Conclusion & Limitations

The Token-Local Resonance Anchoring (TLRA) protocol offers a highly structured, falsifiable approach to decode-time intervention in MLLMs. By architecting Vocabulary-Anchored Semantic Masking (VASM) as an offline static dictionary with $O(1)$ BPE inheritance, we resolve the autoregressive latency paradox that plagues NLP-dependent decode-time controls. Furthermore, by explicitly defining `TLRA_calib` as a Test-Time Adaptation module and benchmarking it against a calibrated MeanPool baseline on strictly unseen object categories, we isolate the true value of spatial routing from mere parameter calibration.

The integrity of this framework is bounded by several explicit limitations:
1. **Calibrator Category Leakage:** Because $\Phi_{calib}$ leverages 50k captions, TLRA relies on the strictly partitioned Seen vs. Unseen parity test (Table 3) to ensure performance gains stem from spatial routing rather than PEFT memorization.
2. **Out-of-Candidate Bounds:** Decode-time reweighting mathematically cannot recover an entity token if the base model has already discarded it from the initial Top-$M$ candidate pool.
3. **Systemic Latency (TPOT Tax):** Despite VASM bypassing NLP tagger latency, dynamically retrieving spatial support for $M$ candidates per token imposes a severe memory-bandwidth bottleneck, translating to an unavoidable TPOT overhead relative to the base model.
4. **Lexical Rigidity:** The static nature of VASM guarantees speed but introduces lexical rigidity, relying on primary WordNet exact-matches and conservatively defaulting to $\gamma=0$ on extreme BPE fragmentations.
5. **Modality Transferability:** The core mechanism is optimized strictly for spatial ($H \times W$) reasoning. Its extension to spatio-temporal settings (video) lacks native token alignment and remains an exploratory limitation rather than a primary capability. 

Ultimately, this paper serves as a transparent blueprint: establishing that token-local visual evidence can reduce hallucination without destroying language structure, provided it is governed by rigorous bounds, strict negative controls, and mathematically sound syntax protection.

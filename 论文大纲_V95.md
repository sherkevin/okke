# Token-Local Resonance Anchoring: A Hybrid Auxiliary Routing Head for MLLM Object Grounding via Static Visual States

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. Existing inference-time mitigation strategies operate globally. In contrast, we present **Token-Local Resonance Anchoring (TLRA)**, a hybrid trained auxiliary routing head that injects spatially local—yet explicitly context-agnostic—visual evidence into decode-time logits adjustment. 

TLRA is built on three architectural corrections. First, to avoid the "local evidence illusion" where final-layer LLM visual tokens are irreversibly contaminated by text-attention, TLRA extracts visual representations from intermediate LLM layers dynamically profiled across architectures. Second, to bridge the manifold mismatch between static mid-layer states and the dynamic pre-LM-head lexical space, we train a lightweight MLP projector. Crucially, we enforce discrete logit-space routing rather than continuous hidden-state addition, isolating the routing gain against a rigorous, zero-initialized gated continuous-fusion baseline to prevent manifold collapse. Third, to protect grammatical structure from this inherently context-free "Bag-of-Words" visual boosting, we bound the intervention window via a Top-$M$ hardware kernel and apply Vocabulary-Anchored Semantic Masking (VASM).

To enforce an unforgiving evaluation contract, we separate our hybrid architectural mechanism from the training objective. Any claim about TLRA's routing superiority must survive a mandatory parameter-and-data-matched parity against an objective-matched `Base + LoRA` model, as well as zero-shot visual CFG. Furthermore, we explicitly audit the method's context-blindness by testing not just on object-existence benchmarks (POPE, CHAIR), but on relational/compositional tasks (MMHal) where static noun-boosting is theoretically predicted to fail. We expose the true wall-clock latency of our Top-K routing kernel, quantify the mathematical ceiling of bounded logit interventions (the "Hijacking Problem"), and mandate a `TLRA w/o VASM` ablation to expose the catastrophic grammar collapse underlying continuous manifold bleeding. The resulting paper establishes a positive, falsifiable protocol for bounded, context-agnostic visual control in MLLMs.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable capabilities, yet their autoregressive decoding remains vulnerable to hallucination. Current inference-time mitigation strategies represent major advances. Contrastive decoding methods (`VCD`), Classifier-Free Guidance (`CFG`), and layer-contrast methods (`DoLa`) are training-free, plug-and-play techniques relying on zero-shot distribution shifts. They leave a complementary architectural question unanswered: how can we dynamically route token-local visual evidence directly into logits adjustment using a trained auxiliary head, without conflating the routing gains with training-objective superiority?

This paper introduces Token-Local Resonance Anchoring (TLRA), advancing a narrow, falsifiable hypothesis: a hybrid framework scoring candidate tokens using spatially localized visual support can reduce object hallucination more faithfully than zero-shot global visual perturbations or trivial continuous fusion, provided it extracts truly local states and explicitly bounds its own grammatical interference.

However, we explicitly define and bound the identity of TLRA: it is fundamentally a **context-agnostic visual noun-booster**. Because it maps static prefill visual states directly to static LM-head vocabulary embeddings, it measures the static affinity between a visual patch and a word, blind to the dynamic grammatical prefix. Executing this intervention honestly requires overcoming four specific architectural barriers and evaluating against the resulting blind spots:

1. **The Local Evidence Illusion:** Final-layer visual tokens are heavily contextualized, text-contaminated global mixtures. TLRA extracts visual states from intermediate layers, systematically profiling the optimal layer depth across backbones (e.g., LLaMA-2 vs. LLaMA-3).
2. **Manifold Mismatch & The Continuous Fusion Strawman:** Static mid-layer visual states and dynamic pre-LM-head lexical states do not natively share an embedding space. We train a lightweight projector ($\Phi_{calib}$) to bridge this. To prove discrete logit routing is fundamentally superior to continuous manifold addition, we defeat a rigorous, zero-initialized gated continuous fusion baseline, measuring strict Textual Perplexity (PPL) preservation.
3. **Hardware Latency and The Top-M Ceiling:** Retrieving local support during generation requires $O(M \cdot N_v)$ dot products and a dynamic Top-$K$ sort at every step. We bound this with a Top-$M$ candidate window ($M=50$). This introduces a hard mathematical ceiling (the "Hijacking Problem"): if the base LLM hallucination expels the ground truth token outside the Top-50, TLRA is structurally blind to it. 
4. **Context-Blindness and The VASM Bottleneck:** Because the $\Phi_{calib}$ projection is context-agnostic, it will indiscriminately boost matching vocabulary tokens regardless of relational constraints, and risks spiking on abstract grammatical syntax. To prevent catastrophic grammar collapse, we introduce Vocabulary-Anchored Semantic Masking (VASM)—a static, English-derived statistical dictionary. We treat VASM not merely as an engineering crutch, but as a severe structural bottleneck, explicitly ablating `TLRA w/o VASM` to expose the raw manifold bleeding, and evaluating on relational tasks (MMHal) where context-blind noun boosting inherently fails.

TLRA is a *trained hybrid method*. We therefore enforce a mandatory parity baseline: a LoRA model trained on the exact same dense region-caption data using the exact same alignment objective. If TLRA fails to outperform this objective-matched control, its routing claim is falsified.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Fairness Boundary: Isolating the Objective in Hybrid Routing

TLRA utilizes a lightweight MLP projector $\Phi_{calib}$ to map static, mid-layer visual states into the dynamic pre-LM-head lexical space. During training, the base LLM and Vision Encoder are **strictly frozen**; only $\Phi_{calib}$ is updated using a localized Next-Token Prediction objective over a dense region-caption subset.

**Objective Matching for Fair Evaluation:**
Confounding the routing mechanism with the training objective is a fatal flaw. Any claim about decode-time local routing must survive a mandatory parity ablation against an objective-matched `Base + LoRA` model fine-tuned on the exact same calibration subset using identical Cross-Entropy. 

**Defeating the Continuous Fusion Strawman:**
To justify discrete logit-space routing, we enforce parity against a robust `ContinuousAdd_Gated` baseline. This baseline projects $\Phi_{calib}(h_{prefill}^{(v_j)})$ and continuously adds it to the final hidden state via a learned gating scalar $\gamma$: $h_{t, final} + \gamma \sum \alpha_j \Phi_{calib}(v_j)$. We initialize $\gamma = 0$ to ensure pre-training preservation, but explicitly verify $\nabla \gamma \neq 0$ during training to prove the pathway is active. If TLRA outperforms this continuous baseline on pure Textual Perplexity (WikiText PPL), we prove that bypassing the pre-trained LM-head manifold shift is an architectural necessity.

### 3.2 Bridging the Mismatch: Static Prefill and Context-Agnostic Scoring

At decoding step $t$, TLRA restricts intervention to a bounded candidate set $\mathcal{C}_t = \text{Top-}M(L_{orig})$, locked at $M=50$ to optimize CUDA thread synchronization.

**The Context-Agnostic Score Function:**
$\Phi_{calib}(h_{prefill}^{(v_j)})$ is executed **exactly once** during prefill, caching an $N_v \times D$ projected tensor. During autoregression, raw local support for a candidate token $c \in \mathcal{C}_t$ is computed via:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{\Phi_{calib}(h_{prefill}^{(v_j)}) \cdot W_{head}[c]}{\tau_{sim}}
$$
Crucially, $W_{head}[c]$ represents the **static, final LM unembedding weight** for token $c$. Because this dot product relies entirely on static prefill visual states and static vocabulary embeddings, it is fundamentally **context-agnostic**. It measures the isolated affinity between a visual patch and a standalone word, completely blind to the dynamic grammatical prefix generated at step $t$.

### 3.3 Temperature-Stabilized Relative Resonance Penalty

We convert raw support into a relative evidence score: $\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)$. To prevent the visual head from overriding confident structural tokens, we anchor the penalty to the local logit spread $\Delta_L = \max(L_{orig}(\mathcal{C}_t)) - \min(L_{orig}(\mathcal{C}_t))$. 

Because logit spread is highly sensitive to the generation temperature $T$, standard bounding ($\Delta_L / T$) explodes during greedy decoding ($T \to 0$). To stabilize this, we clamp the temperature divisor and bound the overall penalty by hyperparameter $\beta$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot M_{VASM}(c) \cdot \min\left(\frac{\Delta_L}{\max(T, 1.0)}, \beta\right) \cdot (1 - \hat S(c))
$$

### 3.4 The VASM Bottleneck

Because $\Phi_{calib}$ maps to static embeddings without contextual awareness, continuous manifold bleeding inevitably causes it to spike on functional subwords (e.g., boosting `[car]` in `[car, dinal]`). 

To protect syntax, TLRA relies on Vocabulary-Anchored Semantic Masking (VASM), a static $M_{VASM} \in \{0, 1\}^V$ mask derived from frequency-weighted BPE resolution in an English C4 corpus. We explicitly define VASM not merely as an engineering crutch, but as a **fatal structural bottleneck**. A true multi-modal LLM cannot rely on an English BPE lookup table. We formally audit this bottleneck by mandating a `TLRA w/o VASM` ablation, exposing the catastrophic grammar collapse that occurs when this hardcoded dictionary is removed.

## 4. Evaluation Protocol and Experimental Design

The paper is organized around three evidence chains and strict boundary audits. This section represents the executable contract of the paper, deliberately designed to expose both the strengths and the structural blindness of the method.

### 4.1 Stage 0: The Gated Continuous Fusion Validation & VASM Ablation

**Goal:** Mathematically verify that discrete logit-space routing is structurally superior to continuous hidden-state addition, while exposing the raw necessity of the VASM bottleneck.
- **Protocol:** Compare TLRA against `TLRA_ContinuousAdd_Gated` (where $\gamma$ is initialized to 0 but verified to train) and `TLRA w/o VASM`. Evaluate on standard Textual Perplexity (WikiText PPL).
- **Hypothesis:** `ContinuousAdd_Gated` will degrade language modeling compared to discrete routing. However, `TLRA w/o VASM` will exhibit catastrophic PPL explosion, proving that the continuous mapping objective fundamentally fails to separate visual nouns from grammar natively.

### 4.2 Evidence Chain A: Object Grounding vs. Relational Blindspots

**Goal:** Test hallucination reduction while explicitly testing the method's context-agnostic "Bag-of-Words" limitation.
- **Benchmarks:** 
  - *Object Existence:* `POPE`, `CHAIR`. (Hypothesis: TLRA excels here by indiscriminately boosting visible nouns).
  - *Relational/Compositional:* `MMHal` or the spatial subset of `GQA`. (Hypothesis: TLRA will show degraded or marginal performance because static noun-boosting cannot resolve spatial constraints like "the apple *left* of the cup").
- **Partitioned Baselines:** 
  - *Training-Free:* Base, `Visual CFG`, `VCD`, `MM-DoLa`.
  - *Matched Objective:* `Base + LoRA` (trained on identical data/cross-entropy).
- **Hard Rule:** TLRA must significantly outperform objective-matched baselines on object existence, and we will honestly report the performance delta on relational tasks as the method's hard boundary.

### 4.3 Evidence Chain B: Hyperparameter Sensitivity and Limits

**Goal:** Transparently document the mathematical ceilings and stability of the routing mechanism.
- **The Temperature Sweep:** Plot CHAIR-s vs. Generation Temperature ($T \in [0.1, 1.0]$) for Base vs. TLRA to prove the $\max(T, 1.0)$ clamp effectively stabilizes the routing head under greedy decoding.

## 5. Discussion and Limitations

The strongest version of TLRA explicitly bounds its claims. By defining itself as a *hybrid trained routing head*, it isolates its routing gains against an unforgiving objective-matched `Base + LoRA` baseline. However, its architectural design reveals profound structural limits for multi-modal grounding.

The primary unresolved risks and limitations are:
1. **The Context-Agnostic "Bag-of-Words" Limitation:** Because TLRA computes dot products between static prefill visual states and static LM-head vocabulary embeddings, it is fundamentally context-blind. It operates as an indiscriminate visual noun-booster. While highly effective for object-existence hallucination mitigation (POPE, CHAIR), it structurally fails at relational and spatial composition (e.g., MMHal) where the dynamic text prefix must dictate visual selection.
2. **The VASM Bottleneck:** A perfectly calibrated objective should teach $\Phi_{calib}$ to ignore syntax natively. In reality, continuous manifold bleeding necessitates VASM. As proven by our `TLRA w/o VASM` ablation, without this hardcoded English C4-derived dictionary, grammar collapses. This statistical crutch fails ungracefully in multilingual contexts.
3. **The Top-M Ceiling (The Hijacking Problem):** Bounded routing has a hard mathematical ceiling. If the base LLM's language prior completely expels the correct visual entity from the candidate window (Rank $> 50$), TLRA cannot recover it. 
4. **GPU Kernel Latency:** Sorting over $N_v \approx 4000$ elements for $M=50$ candidates at every decoding step creates a CUDA thread synchronization bottleneck, introducing a non-trivial wall-clock latency hit per token, which worsens exponentially if $M$ is expanded to capture deeper hallucinations.

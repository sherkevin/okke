# Thresholded Spatial Logits Intervention: Token-Local Visual Evidence for Structure-Preserving MLLM Decoding

## Abstract

Multimodal Large Language Models (MLLMs) remain vulnerable to hallucination when strong language priors overshadow the localized visual evidence required for dense grounding. While existing inference-time mitigation strategies (e.g., contrastive decoding, attention pattern penalties) serve as highly competitive, orthogonal regularizers for language or attention generation dynamics, they lack a mechanism to inject strictly **token-local visual evidence** directly into the terminal logits space while preserving complex language structure. We present **Thresholded Spatial Logits Intervention (TSLI)**, a decode-time method that reweights candidate tokens using localized visual support extracted from contextualized final-layer hidden states. 

TSLI operates on three core mechanisms. First, **token-local logits intervention** maps final-layer visual states into the LLM's latent dimension to evaluate candidate-wise spatial support. Second, **Threshold-Gated Adaptive Top-$k$ Pooling** extracts stable local evidence while mathematically preventing background patches from casting forced votes. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via a WordNet-lemmatized dictionary and dynamic Byte-Pair Encoding (BPE) continuation inheritance. To rigorously handle input-output embedding asymmetry and architectural spatial washout (e.g., 1D sequence vs. 2D-RoPE differences), we establish a strict evaluation boundary separating a zero-shot probe (`TSLI_zero`) from a lightweight, strictly linear calibrated bound (`TSLI_calib`). Furthermore, to prevent catastrophic computational overheads, we introduce entropy-gated triggers and candidate-window bounds.

To validate TSLI, we propose a pre-registered, hypothesis-driven evaluation protocol restricted to 2D spatial grids. Evaluated across three definitive evidence chains—(A) Hallucination Reduction against parameter-matched LoRA baselines, including tracking high-confidence hallucination bypasses, (B) Structure Preservation with exact Out-Of-Vocabulary (OOV) and BPE Collision tracking, and (C) Local Evidence Value via geometric attention mapping—alongside rigorous transparent profiling of latency and peak VRAM, this framework establishes a scientifically sound, structurally protected proposition for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable zero-shot capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to clarify that these methods serve as state-of-the-art orthogonal regularizers that effectively discipline the language-side or attention-side generation heuristics. However, because they operate on overarching generation dynamics rather than explicit spatial grounding, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, defensible hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination more faithfully than pure language/attention heuristics, provided it incorporates explicit structure-preserving safeguards. We conceptualize this intervention as **Thresholded Spatial Logits Intervention (TSLI)**. In this work, we explicitly bound our scope to Image-LLMs and 2D spatial reasoning, deliberately avoiding unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims.

Executing TSLI requires acknowledging and overcoming three structural barriers. First, **input-output embedding asymmetry**: raw visual tokens reside in the input space, rendering direct dot-product similarity with the terminal output vocabulary matrix invalid. Second, the **post-hoc entanglement gamble**: leveraging the final-layer hidden states of visual patches assumes localized support survives deep attention mixing. This risks severe spatial washout, particularly varying between 1D-sequence architectures (e.g., LLaVA) and 2D-RoPE architectures (e.g., Qwen-VL). Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, raw byte fallbacks, functional syntax, or multi-token entities.

TSLI resolves these barriers via three core mechanisms, supported by critical latency and memory optimizations:
1. **Token-local logits intervention**, mapping final-layer visual states to the LLM's pre-vocabulary dimension (`TSLI_zero` vs. `TSLI_calib`), trained under strict semantic disjoint constraints to prevent contrastive poisoning.
2. **Threshold-Gated Adaptive Top-$k$ pooling**, extracting localized spatial support via abstention-aware patch activations.
3. **VASM (Vocabulary-Anchored Semantic Masking)**, utilizing dynamic BPE continuation inheritance to shield structural language, explicitly confining intervention to visual nouns.
4. **Entropy-Gated Execution**, restricting the $O(L \times |V| \times N_v)$ latency and VRAM trap by confining intervention strictly to high-uncertainty decoding steps and bounded candidate windows.

To ensure rigorous scientific boundaries, we deploy our experimental design under a stringent pre-registered protocol encompassing three primary evidence chains. Crucially, we proactively structure our hypotheses to track known decode-time vulnerabilities, including the "arrogance of language priors" (highly confident hallucinations) and tokenizer fragmentation constraints. 

- **Chain A (Hallucination Reduction):** Validating mitigation on POPE/CHAIR against a parameter-matched `Base + 5k LoRA` baseline to isolate the true advantage of the decode-time mechanism over pure data exposure.
- **Chain B (Structure and Reasoning Preservation):** Proving the necessity of VASM and BPE Inheritance on MMMU(Hard) while transparently reporting benchmark Out-Of-Vocabulary (OOV) rates and BPE Collision Rates.
- **Chain C (Local Evidence Value):** Demonstrating the geometric superiority of localized evidence via 4-way side-by-side temporal and attention mapping against natively diffuse baselines.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Overcoming Post-Hoc Entanglement: `TSLI_zero` vs `TSLI_calib`

A fundamental vulnerability in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, TSLI evaluates the *contextualized* final-layer transformer hidden states of the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$. 

We explicitly acknowledge the gamble of **post-hoc entanglement**: deep self-attention implicitly mixes visual states with prefix context. Furthermore, we hypothesize this washout is heavily architecturally dependent (e.g., 1D MLP projectors vs. 2D-RoPE embeddings). To handle vocabulary projection cleanly without falsely claiming zero-shot alignment if an architecture inherently destroys spatial fidelity, we define two distinct tracks:

1. **`TSLI_zero`**: A strict zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$.
2. **`TSLI_calib`**: If `TSLI_zero` fails due to spatial washout, we pivot to a lightweight calibrated decoding method. $\Phi_{calib}: \mathbb{R}^{D_{vision}} \rightarrow \mathbb{R}^{D_{llm}}$ is strictly defined as a **single linear layer (zero hidden layers)** projecting the visual state to the LLM's final hidden dimension *prior* to $W_{vocab}$. It is trained on a frozen set of 5,000 COCO image-patch to bounding-box label pairs. The base LLM remains entirely frozen.

**Strict Semantic Negative Sampling:** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. The positive target $W_{vocab}[c^+]$ is the contextualized hidden state of the final subword of the ground-truth label. To handle overlapping objects in dense images, we assign intersecting patches strictly to the smallest bounding box (foreground priority). Negative patches ($W_{vocab}[c^-]$) are cross-referenced with panoptic/instance annotations to ensure they definitively belong to a different semantic class than the target, ensuring a pure gradient signal for spatial localization.

### 3.2 Threshold-Gated Adaptive Top-$k$ Pooling and Latency Optimization

At decoding step $t$, TSLI evaluates spatial support. A naive Softmax over all visual patches would force background patches to sum to 1.0, artificially inflating noise. We apply a threshold-gated Sigmoid activation, allowing irrelevant patches to mathematically abstain:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{noise}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{noise}\Big)
$$
To prevent $\theta_{noise}$ from becoming a brittle hyperparameter, it is dynamically defined per-step as the moving median of the activation distribution strictly across the $N_v$ spatial patches at current step $t$. We then apply an Adaptive Top-$k$ rule: $k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)$, aggregating only the top-$k$ most supportive activated patches for candidate $c$.

**Evading the Latency and VRAM Trap:** Computing similarities across $V$ tokens and $N_v$ patches at every single decoding step inherently introduces an $O(L \times |V| \times N_v)$ latency and memory bandwidth trap. To salvage real-time viability, we introduce two optimizations:
1. **Candidate Restriction:** We bound intervention strictly to $\mathcal{C}_t = \text{Top-}M(L_{orig})$ where $M \ll |V|$ (e.g., $M=100$), reducing the computation surface.
2. **Entropy-Gated Execution:** TSLI is bypassed if the base LLM's native confidence (probability of the Top-1 token) exceeds $0.90$. We explicitly acknowledge a critical vulnerability here: the *arrogance of language priors*, where MLLMs confidently hallucinate ($p > 0.90$). Our experimental protocol strictly audits the false-negative rate of this entropy gate to transparently report unmitigated highly confident hallucinations.

### 3.3 VASM (Vocabulary-Anchored Semantic Masking)

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. This penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$

VASM prevents structural collapse via a dual-tier architecture:
1. **Root Token Expectation (WordNet-Augmented Visual Nouns):** $\gamma=1$ *only* for an unambiguous dictionary of visual object nouns. Out-of-dictionary terms default to $\gamma=0$, guaranteeing structural syntax safety. 
2. **BPE Continuation Inheritance (Dynamic):** Any subword token marked as a continuation fragment automatically inherits the exact $\gamma$ weight of its root token. Crucially, if the tokenizer outputs raw byte fallbacks (e.g., `<0xE2>`), the sequence inherits the $\gamma$ of the immediately preceding structural subword. This mathematically synchronizes logit adjustments across fragmented entities, preventing multi-token tearing.

## 4. Evaluation Protocol and Experimental Design

Because TSLI introduces a specific structural hypothesis regarding decode-time local evidence, spatial washout, and latency optimization, we organize the evaluation plan into predefined falsifiable evidence chains. The protocol utilizes **LLaVA-1.5** (MLP projector, 1D sequence) and **Qwen-VL-Chat** (2D-RoPE) to directly test architectural variance. All experiments outline our planned, pre-registered execution.

### 4.1 Chain A: Hallucination Reduction vs. Confident Priors

**Hypothesis:** TSLI mitigates object hallucination through localized evidence reweighting. These gains derive from the decode-time intervention mechanism, not mere parameter exposure.
- **Zero Data Leakage:** The 5,000 COCO images used to train $\Phi_{calib}$ are strictly verified to have zero intersection with evaluation sets.
- **Planned Execution (Table 1):**
  - **Metrics:** POPE (Acc/F1) | CHAIR (Object vs Attribute breakdown) | **High-Confidence Hallucination Rate (False Negatives of Entropy Gate)**
  - **Zero-Shot Baselines:** Base, VCD, OPERA, DoLa, `TSLI_zero`.
  - **Calibrated Baselines:** `Base + 5k LoRA` (fine-tuned on the exact same 5k pairs, with LoRA rank $r$ carefully selected so the parameter footprint exactly matches the ~16M parameters of $\Phi_{calib}$), `TSLI_calib`.
- **Claim Retreat Contingency:** If `TSLI_zero` fails severely on LLaVA-1.5 but succeeds on Qwen-VL, we will explicitly isolate 1D spatial washout as the cause. If both fail, we formally downgrade the claim from "zero-shot intervention" to "calibrated post-hoc alignment" via `TSLI_calib`.
- **Success Criteria:** `TSLI_calib` must outperform the zero-shot heuristics and the parameter-matched `Base + 5k LoRA` on Object-CHAIR and POPE. We will transparently report how many hallucinations bypassed the $p>0.90$ entropy gate due to strong language priors.

## 5. Discussion and Limitations

The strength of TSLI lies in its falsifiable, structurally bounded framework: intervening at decode-time using strictly token-local visual evidence, and mathematically protecting language integrity via VASM. However, its execution carries severe theoretical and operational constraints:

1. **The Post-Hoc Entanglement Gamble:** The core assumption that localized visual support survives deep self-attention in final layers is precarious. If `TSLI_zero` completely fails across architectures, the premise of extracting zero-shot local states must be downgraded, relying entirely on the $\Phi_{calib}$ alignment track.
2. **The "Arrogance of Language Priors" (Entropy Gate Blindspot):** Relying on top-1 confidence to trigger intervention assumes the LLM is unconfident when hallucinating. Highly confident hallucinations ($p > 0.90$) driven by biased language priors will completely bypass TSLI.
3. **VASM’s Noun-Centric Blindspot & Dictionary Exhaustion:** By restricting $\gamma=1$ exclusively to nouns, VASM is entirely blind to visual verbs and attributes. Furthermore, if a valid visual noun is out-of-vocabulary (OOV) or bizarrely tokenized, it receives $\gamma=0$, rendering TSLI completely inactive for that entity.
4. **Latency and Memory Bandwidth Trap:** Despite entropy-gating and bounding the candidate window to $M$, dynamic BPE mask lookups and Top-$k$ spatial pooling inherently impact VRAM and throughput. TSLI may ultimately serve strictly as a high-precision, offline analytical decoding tool rather than a real-time production API.
5. **Out-of-Candidate Unrecoverability:** TSLI mathematically reweights the LLM's initial candidate distribution. It cannot synthesize a visually grounded token if the base model completely suppresses it from the initial candidate window (e.g., rank $>M$).

## 6. Conclusion and Future Execution

This paper outlines Thresholded Spatial Logits Intervention (TSLI), a rigorously bounded framework for mitigating MLLM hallucination through structure-preserving, token-local logits intervention. By mapping embedding asymmetry risks, utilizing threshold-gated Top-$k$ pooling to avoid background dilution, engineering robust language protection via VASM BPE inheritance, and tracking severe computational bottlenecks via entropy-gating, we establish a technically precise foundation for decode-time visual control. The claims herein are strictly governed by our pre-registered evaluation protocol spanning Hallucination, Structure, and Local Evidence chains, awaiting full empirical execution.

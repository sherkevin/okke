# Bounded Region Alignment: Token-Local Visual Evidence for Structure-Preserving MLLM Decoding

## Abstract

Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when strong language priors overshadow the localized visual evidence required for dense spatial grounding. While existing inference-time mitigation strategies (e.g., contrastive decoding, attention pattern penalties) serve as highly competitive, orthogonal regularizers for overarching language or attention dynamics, they lack a mechanism to inject strictly **token-local visual evidence** directly into the terminal logits space while preserving complex language structure. We present **Bounded Region Alignment (BRA)**, a decode-time intervention that reweights candidate tokens using localized visual support extracted from contextualized final-layer hidden states. 

Strictly scoped to 2D Image-LLMs to secure defensible claims regarding spatial locality, BRA operates on three core mechanisms designed to balance intervention with computational honesty. First, to handle input-output embedding asymmetry and architectural spatial washout, we establish a strict evaluation boundary separating a zero-shot projection (`BRA_zero`) from a lightweight, dynamically calibrated bound (`BRA_calib`). This boundary is governed by an offline, mathematically formalized feature-level spatial retrieval threshold. Second, **Threshold-Gated Adaptive Top-$k$ Pooling** extracts stable local evidence using a moving median—executed strictly via GPU-native tensor quantiles—bounded by a once-per-prompt prefix limit to prevent signal collapse in dense images. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax via dynamic Byte-Pair Encoding (BPE) continuation inheritance tied to an explicitly defined WordNet spatial dictionary, operating alongside an **Arrogance-Triggered Entropy Gate** that forces visual scrutiny precisely when the LLM hallucinates with high confidence.

To validate BRA and strictly prevent heuristic pipeline bloat, we propose a pre-registered, hypothesis-driven evaluation protocol. Evaluated across three definitive evidence chains—(A) Hallucination Reduction isolating mechanisms via step-by-step ablation (tracking Tokens/Sec throughput) against a parameter-matched `Base + 5k LoRA`, (B) Structure Preservation tracking exact Out-Of-Vocabulary (OOV) intervention bypass and polysemy false positive rates, and (C) Local Evidence Value comparing `BRA_zero` against global pooling baselines (`BRA_MeanPool`). Alongside transparent defense lines profiling separated prefill and decode batched latency—including explicit logging of Out-of-Memory (OOM) failures at large batch sizes—this framework establishes a scientifically rigorous, structurally bounded proposition for decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable zero-shot capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to clarify that these methods serve as state-of-the-art orthogonal regularizers that effectively discipline the overarching language-side or attention-side generation heuristics. However, because they operate on broader generation dynamics rather than explicit spatial grounding, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, defensible hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination more faithfully than pure language/attention heuristics, provided it incorporates explicit structure-preserving safeguards. We conceptualize this intervention as **Bounded Region Alignment (BRA)**. In this work, we strictly bound our scope to Image-LLMs and 2D spatial reasoning, deliberately downgrading unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims surrounding spatial locality.

Executing BRA requires acknowledging and mathematically overcoming three structural barriers. First, **input-output embedding asymmetry**: raw visual tokens reside in the input space, rendering direct dot-product similarity with the terminal output vocabulary matrix invalid. Second, the **post-hoc entanglement gamble**: leveraging the final-layer hidden states of visual patches assumes localized support survives deep attention mixing. This risks severe spatial washout, varying drastically across architectures. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments, functional syntax, polysemous words, or multi-token entities.

BRA resolves these barriers via a structured, ruthlessly ablated framework:
1. **Bi-Level Logits Intervention (`BRA_zero` / `BRA_calib`)**: A strict evaluation boundary testing whether final-layer visual states natively align with the pre-vocabulary dimension (`BRA_zero`). We establish a mathematically formalized, offline feature-level spatial washout threshold. If native alignment fails, we pivot to a lightweight calibrated projection (`BRA_calib`) trained under stringent SAM-assisted semantic negative sampling.
2. **Threshold-Gated Adaptive Top-$k$ Pooling**, extracting localized spatial support via abstention-aware patch activations grounded by a GPU-native moving median and bounded by a once-per-prompt prefix-derived limit.
3. **VASM (Vocabulary-Anchored Semantic Masking)**, utilizing dynamic BPE continuation inheritance and an explicit ~85k WordNet physical entity dictionary to shield structural language, transparently tracking OOV bypass rates.
4. **Arrogance-Triggered Entropy Gate**: Resolving the paradox of intervening on all confidence bands, our logic actively prunes "uncertainty" heuristics to exclusively force visual verification when language priors are most arrogant (high confidence on visual nouns).

To ensure rigorous scientific boundaries and prevent pipeline bloat, our experimental design is organized into an execution protocol strictly defining three primary evidence chains and transparent defense lines. 
- **Chain A (Hallucination Reduction):** Validating true mechanism value on POPE, CHAIR, and AGL against a parameter-matched `Base + 5k LoRA` baseline. This chain mandates a step-by-step architectural ablation tracking exact **Tokens/Sec** throughput; any heuristic (e.g., the Entropy Gate) failing to yield $\ge 2\%$ absolute gain will be aggressively pruned.
- **Chain B (Structure and Reasoning Preservation):** Proving VASM dynamics via MMBench, MME, and MMMU(Hard), transparently tracking exact Out-Of-Vocabulary (OOV) bypass rates and polysemy false positives.
- **Chain C (Local Evidence Value & Domain Shift):** Demonstrating the definitive superiority of localized evidence via FREAK and DocVQA, strictly isolating geometric localization by comparing `BRA_zero` and `BRA_calib` against a `BRA_MeanPool` (global average) baseline.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Overcoming Post-Hoc Entanglement: The Washout Threshold

A fundamental vulnerability in decode-time visual intervention is embedding space asymmetry. The LLM's output vocabulary matrix ($W_{vocab}$) resides in the terminal logit space, while visual tokens enter at the input layer. To construct a valid local support metric, BRA evaluates the *contextualized* final-layer transformer hidden states. Specifically, we extract the LLM's final hidden states corresponding to the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{vision}}$, immediately prior to the terminal $W_{vocab}$ projection.

We explicitly acknowledge the gamble of **post-hoc entanglement**: deep self-attention implicitly mixes visual states with prefix context. To handle vocabulary projection cleanly without falsely claiming zero-shot alignment if an architecture inherently destroys spatial fidelity, we define a strict operational boundary governed by a direct, feature-level **Washout Threshold**, evaluated **offline, once per architecture**:

1. **`BRA_zero`**: A zero-shot variant where the projection $\Phi_{zero}$ is the model's native `lm_head`. We compute local support via $logit^{(v_j)} = W_{vocab} \cdot \text{LayerNorm}(h_L^{(v_j)})$. 
2. **The Washout Threshold (Offline Verification)**: We test zero-shot bounding box retrieval on a held-out 500-sample spatial validation set. For a target class token $c_{target}$, we extract the sequence of patch logits $logit^{(v_j)}[c_{target}]$, reshape them into a 2D spatial grid, apply Gaussian smoothing to prevent localized fragmentation, and apply Min-Max normalization to yield a heatmap $\tilde{\mathcal{M}} \in [0,1]$. A predicted bounding box is extracted by binarizing $\tilde{\mathcal{M}}$ using Otsu's thresholding algorithm. If the native projection fails to achieve an average bounding box $\text{IoU} > 0.15$ against ground truth, native spatial fidelity is mathematically declared lost for that architecture.
3. **`BRA_calib`**: Upon triggering the washout threshold, we pivot to a lightweight calibrated alignment. $\Phi_{calib}: \mathbb{R}^{D_{vision}} \rightarrow \mathbb{R}^{D_{llm}}$ is defined as a 2-layer MLP projection ($\approx 4.2$M parameters, appending $<0.01$ TFLOPs per step) trained on 5,000 COCO image-patch to bounding-box label pairs, with the base LLM entirely frozen.

**Strict Semantic Negative Sampling:** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. Negative patches ($W_{vocab}[c^-]$) are cross-referenced to ensure an $\text{IoU} < 0.1$ with the target *and* membership in a differing COCO semantic super-category. Crucially, the negative candidate set $\mathcal{C}^-$ explicitly filters out any BPE subwords or continuations that share a root string with the target visual noun.

### 3.2 Threshold-Gated Adaptive Top-$k$ Pooling

At decoding step $t$, a naive Softmax over all visual patches would force background patches to sum to 1.0, artificially inflating noise. We apply a threshold-gated Sigmoid activation:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{noise}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{noise}\Big)
$$
To prevent $\theta_{noise}$ from becoming a brittle static hyperparameter, it is dynamically defined per-step as the moving **median** of the activation distribution across the $N_v$ patches. This is executed strictly via GPU-native tensor approximations (`torch.kthvalue`) bypassing CPU synchronization barriers, bounded by a $\mathcal{O}(N_v)$ operational complexity per candidate token. 

To address dense images where the median naturally spikes, wiping out valid evidence, we apply a historical upper-bound clamp $\theta_{max}$. This clamp is derived **strictly once per prompt**, calculating the 90th percentile of visual activations from the text prefix. We then apply an Adaptive Top-$k$ rule: $k = \max\Big(k_{min}, \lceil \rho \cdot N_v \rceil\Big)$, aggregating only the top-$k$ most supportive activated patches. The sorting operation adds a $\mathcal{O}(N_v \log k)$ complexity overhead per step.

### 3.3 VASM and the Arrogance-Triggered Entropy Gate

The final candidate logits are adjusted by penalizing tokens lacking local visual backing. This penalty is gated by a deterministic semantic mask $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$

VASM prevents structural collapse via a dual-tier protection architecture:
1. **Explicit Root Dictionary Expectation:** We define the visual noun dictionary offline using the WordNet *physical entity* subtree (approx. 85,000 synsets). $\gamma=1$ *only* for tokens matching this dictionary. Out-of-dictionary terms (including specialized domain nouns) default to $\gamma=0$.
2. **BPE Continuation Inheritance:** Any subword token marked as a continuation fragment inherits the exact $\gamma$ weight of its root token. To preserve $\mathcal{O}(|\mathcal{C}_t|)$ complexity per decoding step, VASM is implemented as a pre-compiled GPU-side bitmask mapping against the model's vocabulary.

**The Arrogance-Triggered Entropy Gate:** 
Intervening across all confidence bands introduces mathematical paradoxes and pipeline bloat. We actively prune intervention logic to target a single, critical failure mode: MLLMs hallucinating objects with extreme, arrogant confidence. 
Intervention is **forced** ONLY IF the model exhibits extreme confidence ($Entropy < H_{low}$) AND the candidate token is a visual noun ($\gamma=1$). High confidence on a visual noun *invites* visual scrutiny. $H_{low}$ is calibrated once as the 10th percentile of the base model's confidence distribution over a validation set. Candidate sets are strictly bounded to $\mathcal{C}_t = \text{Top-}M(L_{orig})$ where $M \ll |V|$. If empirical ablations demonstrate this gate yields $< 2\%$ absolute gain, it will be discarded entirely to prioritize throughput.

## 4. Evaluation Protocol and Experimental Design

Because BRA introduces specific structural hypotheses—and risks computational bloat if unverified—we organize the evaluation plan into predefined falsifiable evidence chains and strict defense lines. The protocol utilizes **LLaVA-1.5** (MLP projector, 1D sequence) and **Qwen-VL-Chat** (2D-RoPE) to test architectural variance. All experiments outline our planned, hypothesis-driven execution. 

### 4.1 Chain A: Hallucination Reduction via Step-by-Step Attribution

**Hypothesis:** BRA mitigates object hallucination through localized evidence reweighting. These gains derive from the decode-time mechanism, not parameter exposure, and each pipeline component contributes uniquely enough to justify its latency cost.
- **Evaluation Protocol:**
  - **Metrics:** POPE (Acc/F1), CHAIR (reporting Object vs. Attribute separately), and AGL (Attention-Grounded Localization, IoU). 
  - **Planned Output (Table 1):** Rows evaluated: Base, VCD, OPERA, DoLa, and the critical control `Base + 5k LoRA` (fine-tuned on the identical 5k SAM-filtered pairs to isolate representation learning from inference-time adjustment), alongside `BRA_MeanPool`, `BRA_zero`, and `BRA_calib`.
  - **Mandatory Step-by-Step Ablation (with Latency Tracking):** To prove pipeline necessity, we mandate a sequential build-up: `Base` $\rightarrow$ `+ MeanPool` $\rightarrow$ `+ Adaptive Top-k` $\rightarrow$ `+ VASM` $\rightarrow$ `+ Entropy Gate (BRA_Full)`. A strict **Tokens/Sec** column will accompany this table. If a module (e.g., Entropy Gate) fails to provide $\ge 2\%$ absolute gain, it will be explicitly pruned from the recommended methodology.

### 4.2 Chain B: Structure Preservation and VASM Limitations

**Hypothesis:** Hallucination mitigation via visual suppression does not degrade complex reasoning if VASM protects structural tokens. However, reliance on an explicit ~85k WordNet dictionary introduces a quantifiable domain limitation where OOV visual nouns safely bypass intervention, but polysemous words risk false suppression.
- **Evaluation Protocol:**
  - **Benchmarks:** MMBench V1.1, MME, MMMU (Hard Subset).
  - **OOV Exhaustion Tracking (Figure 1):** A scatterplot correlating performance with dictionary limitations. **X-axis = % of ground truth visual nouns missing from VASM (OOV rate)**, and **Y-axis = Accuracy drop relative to Base**. 
  - **Polysemy False Positive Rate:** We will randomly sample 500 generation steps across MMBench where VASM triggered ($\gamma=1$) and manually evaluate the False Positive Rate of suppressing non-visual word usages (e.g., suppressing "bank" in a financial context).

### 4.3 Chain C: Local Evidence Value & Domain Shift Crucible

**Hypothesis:** On dense spatial tasks, token-local visual reweighting geometrically isolates evidence that diffuse pooling natively washes out.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (OCR/Position) and DocVQA (ANLS).
  - **Planned Output (Table 2):** Strictly comparing `Base`, `Base + 5k LoRA`, `BRA_MeanPool` (naive global averaging), and `BRA_calib`. This is the ultimate crucible: if `BRA_MeanPool` outperforms the Adaptive Top-$k$ variant here, the core hypothesis of localized evidence is falsified.

### 4.4 Defense Lines: Explicit Complexity and Separated Latency Profiling

**Hypothesis:** The latency penalty can be managed via pre-compiled GPU bitmasks, though batched scaling remains a severe constraint for moving-median and Top-$k$ sorts.
- **Evaluation Protocol (Figure 3):** Executed strictly on a standardized single A100 (80GB) GPU. We plot **Tokens/Sec** (left y-axis) and **Peak VRAM Usage** (right y-axis) against Candidate Window Size $M \in \{10, 50, 100\}$, across **Batch Sizes = 1, 4, and 8**.
- **Crucial Separation and OOM Transparency:** We strictly separate **Prefill Latency** from **Decode Latency**. Because BRA intervenes entirely during decode, blending the two artificially masks the true $\mathcal{O}(N_v \log k)$ computational cost. If BRA triggers Batched Out-of-Memory (OOM) failures at BS=8 due to tensor expansions, we will explicitly plot this as a red 'X' on the graph, acknowledging the method as a high-precision offline analytical tool rather than a high-throughput real-time API.

## 5. Discussion and Limitations

The strength of Bounded Region Alignment (BRA) lies in its structurally bounded proposition: intervening at decode-time using token-local visual evidence while protecting language integrity via VASM. However, its execution is governed by severe theoretical and operational constraints:

1. **The Post-Hoc Entanglement Gamble:** The assumption that localized visual support survives deep self-attention in final layers is precarious. If the defined offline Washout Threshold is crossed (IoU < 0.15), the premise of zero-shot local states relies entirely on the $\Phi_{calib}$ track, shifting the method from pure inference to hybrid alignment.
2. **VASM Brittleness and Intervention Bypass:** An explicitly defined WordNet superset is brittle for domain shifts. In specialized tasks like DocVQA or MMMU, if a critical visual noun is OOV, VASM defaults to $\gamma=0$, rendering BRA completely inactive. Our "structure preservation" in niche domains inherently conflates with "intervention bypass." Furthermore, polysemy false positives remain an unresolvable vulnerability of any dictionary-based mask.
3. **Pipeline Bloat vs. Attribution:** Deploying an MLP calibrator, moving-median Top-$k$ pooling, VASM, and an Entropy Gate introduces significant complexity. We explicitly commit to our strict step-by-step ablation protocol: if any component (notably the Entropy Gate) fails to demonstrate $\ge 2\%$ absolute, independent gains, it will be actively pruned from the final proposed methodology.
4. **Decode-Time Latency Traps:** Even with GPU-native tensor quantiles, step-wise spatial pooling and Top-$k$ sorting inherently bottleneck generation throughput. Isolating decode-time latency scaling at larger batch sizes (e.g., BS=8) severely risks VRAM OOM, explicitly confining BRA to high-precision offline applications rather than real-time serving.

## 6. Conclusion and Future Execution

This paper outlines Bounded Region Alignment (BRA), a rigorously defined framework for mitigating MLLM hallucination through structure-preserving, token-local logits intervention. By formalizing embedding asymmetry (`BRA_zero` vs `BRA_calib`) via offline mathematical bounding box retrieval thresholds, utilizing Threshold-Gated Adaptive Top-$k$ pooling bounded by once-per-prompt prefix limits, engineering robust language protection via an ~85k VASM dictionary, and enforcing visual scrutiny via an Arrogance-Triggered Entropy Gate, we establish a technically precise foundation for decode-time visual control. The claims herein are strictly governed by our hypothesis-driven evaluation protocol—mandating step-by-step ablations and aggressive latency transparency—establishing a verifiable roadmap for empirical execution.

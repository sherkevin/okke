Multimodal large language models (MLLMs) remain vulnerable to hallucination: fluent but weakly grounded text, invented objects, and language-prior-driven errors. We argue that strict ``hot-swappable'' learned control is impossible when a module consumes host-native hidden states or writes directly into host lexical geometry. We formalize this \emph{universality boundary} and propose \textbf{UniGround}, a training-free-at-attachment decode-time sidecar whose sole learned core $\Psi_{\mathrm{univ}}$ operates in a shared external semantic space built from frozen public vision and text encoders, decoded candidate spans, and deterministic attachment logic. UniGround performs \emph{candidate-driven visual resonance}: each candidate span awakens locally relevant region evidence, and a safety-gated, conservative logit redistribution nudges only the host's active Top-$M$ frontier. We specify an evaluation protocol separating module-level validation, cross-model hot-swap validity, global-prior baselines, structure preservation, and explicit failure boundaries.

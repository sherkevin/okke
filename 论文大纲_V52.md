# Dictionary-Gated Token-Local Visual Intervention: A Hybrid Decode-Time Framework for Hallucination Reduction

## Abstract
Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when overarching language priors overshadow the localized visual evidence required for dense spatial grounding. Existing inference-time mitigation strategies (e.g., DoLa, VCD, OPERA) successfully regularize global token probabilities or attention distributions. However, as purely zero-shot, training-free methods, they operate orthogonally to the explicit routing of spatial evidence. This paper investigates how to inject **token-local visual evidence** directly into the terminal logits space at decode-time without damaging functional language structure. We present **Token-Local Visual Intervention**—a dictionary-gated logits adjustment framework utilizing localized visual support extracted from contextualized final-layer hidden states. 

Strictly scoped to 2D Image-LLMs, this framework addresses three structural barriers. First, acknowledging the near-universal spatial washout inherent to deep self-attention, we utilize a zero-shot diagnostic (`BRA_zero`). Upon expected failure, the framework shifts to a lightweight, dynamically calibrated bound (`BRA_calib`). This makes our method a *hybrid test-time adaptation* rather than a purely zero-shot regularizer. The calibration is optimized via spatially-aware hard negative sampling, mathematically fortified by a dual-veto Compositional Coherence Margin to avoid destroying overlapping objects. Second, to combat autoregressive threshold drift, we extract localized support via an Entropy-Scaled Adaptive Top-$k$ mechanism governed by a **Rolling Decay** activation threshold, carefully bounded by window size sensitivity. Third, to protect functional syntax from subword tokenization splits, we implement a **Visual-Aware Semantic Masking (VASM)** module—a dual-tier shield utilizing an ~85k WordNet dictionary via a greedy prefix-trie bitmask, and deterministic regex triggers.

To rigorously assess algorithmic overhead and genuine spatial utility, we propose a pre-registered execution protocol across three definitive evidence chains: (A) Hallucination Reduction (POPE/CHAIR + AGL), (B) Structure and Reasoning Preservation (`BRA_no_VASM` ablation on MMMU/MMBench), and (C) Local Evidence Value (`BRA_zero` vs `BRA_MeanPool`). Backed by five explicit defense lines tracking everything from Inter-Token Latency (ITL) to BPE bypass trajectories, this framework establishes a verifiable proposition for spatially-aware, hybrid decode-time intervention.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable visual-linguistic capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to establish that these methods serve as highly successful, state-of-the-art regularizers that effectively discipline overarching language-side or attention-side generation heuristics. As purely zero-shot, training-free algorithms, they operate on global generation dynamics. Consequently, they are not natively designed to address the following orthogonal structural question:

**How can we explicitly inject strictly token-local visual evidence into decode-time logits adjustment without destroying structural language, multi-token entities, or downstream reasoning?**

This paper advances a direct, functional hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination faithfully, provided it incorporates explicit structure-preserving safeguards. We functionalize this as **Token-Local Visual Intervention**. In this work, we strictly bound our scope to 2D Image-LLMs, deliberately discarding unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims regarding spatial locality.

Executing this intervention requires mathematically overcoming three structural barriers. First, **spatial washout** in embedding spaces. Deep self-attention almost universally diffuses localized patch coordinates by the final layer. We formalize this via a zero-shot Bounding Box Retrieval Assessment (`BRA_zero`). Acknowledging this limitation, our framework operates as a *hybrid test-time adaptation*. Upon washout detection, it shifts to a lightweight test-time calibration (`BRA_calib`) optimized via a Compositional-Aware InfoNCE loss, ensuring naturally co-occurring objects are not repelled during contrastive sampling. Second, the computational complexity and drift of **dynamic spatial pooling**. We replace static thresholds with an entropy-scaled dynamic activation governed by a continuous Rolling Decay mechanism, eliminating brittle step-function fallbacks during long-context generation. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments or OCR-critical entities. We resolve this via a Visual-Aware Semantic Masking (VASM) module, engineered with a greedy prefix-trie to handle subword mapping openly tracking when the mask actively preserves structure versus when it defaults to inaction.

To ensure rigorous scientific boundaries, our experimental design is organized into a pre-registered execution protocol defining three primary evidence chains and five explicit defense lines:
- **Chain A (Hallucination Reduction):** Validating mechanism value on POPE, CHAIR, and AGL against a strictly parameter-matched `Base + 5k LoRA` baseline, isolating routing utility from parametric injection.
- **Chain B (Structure and Reasoning Preservation):** Proving VASM dynamics via MMBench, MME, and MMMU(Hard), strictly ablating `BRA_no_VASM` to prevent conflating "preservation" with "inaction."
- **Chain C (Local Evidence Value):** Demonstrating the definitive superiority of localized evidence on FREAK and DocVQA by contrasting `BRA_zero` against `BRA_MeanPool`, backed by direct main-text visual proofs of spatial routing.
Through these bounded chains and corresponding defense lines quantifying Inter-Token Latency (ITL) and BPE exact-match trajectories, this framework establishes a verifiable proposition for spatially-aware, hybrid visual intervention.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Embedding Asymmetry, Spatial Washout, and Compositional InfoNCE

To construct a valid local support metric, we evaluate the *contextualized* final-layer LLM hidden states corresponding to the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{model}}$. However, deep self-attention demonstrably diffuses spatial coordinates by the final layer. We formalize this via an operational boundary:

1. **`BRA_zero` (Zero-Shot Diagnostic)**: We evaluate zero-shot bounding box retrieval using the model's native `lm_head` as the projection $\Phi_{zero}$. If this fails to achieve an average bounding box $\text{IoU} > 0.15$ on a 500-sample spatial validation set, spatial fidelity is mathematically declared washed out.
2. **`BRA_calib` (Hybrid Test-Time Adaptation)**: Upon triggering the washout threshold, the framework shifts to $\Phi_{calib}$, a 2-layer MLP mapping $\mathbb{R}^{D_{model}} \rightarrow \mathbb{R}^{D_{model}}$. This explicitly transitions our method from zero-shot to a lightweight hybrid intervention. It is optimized on 5,000 size-stratified Visual Genome (VG) pairs disjoint from all test sets.

**Compositional-Aware Hard-Negative Sampling:** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. A naive spatial exclusion zone ($\text{IoU} < 0.1$) fails to disentangle overlapping objects. Therefore, valid negative patches include *Hard Negatives* ($\text{IoU} > 0.5$, heavily overlapping with the anchor but differing in semantic category). To prevent **Compositional Destruction** (e.g., repelling a "rider" from a "horse"), we enforce a dual-layered *Compositional Coherence Margin*. 

Let $z_a = \Phi_{calib}(h_L^{(v_{anchor})})$ and $z_c$ be a candidate patch embedding. The optimized loss is formulated as:
$$
\mathcal{L}_{InfoNCE} = - \log \frac{\exp(sim(z_a, z^+)/\tau)}{\exp(sim(z_a, z^+)/\tau) + \sum_{k=1}^{K} \mathbb{I}_{comp}^{(k)} \exp(sim(z_a, z_k^-)/\tau)}
$$
The compositional validity indicator $\mathbb{I}_{comp}^{(k)} \in \{0,1\}$ determines if a hard negative $z_k^-$ is mathematically permissible. It is defined via two vetos:
1. *Graph Veto ($\mathbb{I}_{graph}$):* $1$ if the anchor and negative share a parent-child relationship in the VG scene graph, $0$ otherwise.
2. *Semantic Veto ($\mathbb{I}_{sem}$):* Using frozen CLIP text embeddings, if $\cos(c_{anchor}, c_{candidate}) > \tau_{comp}$, $\mathbb{I}_{sem} = 1$, safely preserving unannotated co-occurring objects.
Thus, $\mathbb{I}_{comp}^{(k)} = (1 - \mathbb{I}_{graph}) \cdot (1 - \mathbb{I}_{sem})$. A negative is only contrasted if both vetos are clear.

### 3.2 Entropy-Scaled Top-$k$ Pooling & Rolling Decay Activation

At decoding step $t$, we apply a threshold-gated Sigmoid activation:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{active}^{(t)}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{active}^{(t)}\Big)
$$
We derive an **entropy-scaled dynamic threshold base** during the prefill stage: $\theta_{base} = \mu_A + \lambda\left(\frac{\mathcal{H}_{vis}}{\log(N_v)}\right) \cdot \sigma_A$.

**The Rolling Decay Fallback Mechanism:** Deriving $\theta_{base}$ strictly at prefill risks threshold obsolescence during long context generation. Replacing a brittle step-function fallback, we implement a **Sliding Window Exponential Moving Average (EMA)**. We monitor the *Valid Activation Rate* $R_t$. The active threshold dynamically decays:
$$
\theta_{active}^{(t)} = \theta_{base} \cdot \min\left(1.0, \frac{\text{EMA}(R_{t-W:t}) + \epsilon}{\tau_{target}}\right)
$$
**Parameter Sensitivity:** The window size $W$ and target rate $\tau_{target}$ dictate the decay severity. If $W$ is too small (e.g., $<5$), the threshold plummets instantaneously, mimicking the step-function we aim to avoid. If $W$ is too large, the intervention forces hallucinated visual grounding on purely linguistic deductive steps. In practice, $W$ must be scaled proportionally to the average explanation length of the target task.

### 3.3 Visual-Aware Semantic Masking (VASM)

The final candidate logits are adjusted by penalizing tokens lacking local visual backing, gated by a deterministic VASM module $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$
We utilize a **Dual-Tier Protection Architecture**:
1. **Tier 1 (Root Dictionary):** Offline WordNet *physical entity* subtree ($\approx 85,000$ synsets).
2. **Tier 2 (Deterministic Format Triggers):** For DocVQA and dense reasoning, numerical strings and capitalized entities are assigned $\gamma=1$ via deterministic regex logic.

**Formalizing the BPE-to-WordNet Mapping:** Subword tokenization (e.g., `[" base", "ball"]`) creates structural brittleness against dictionary matching. We resolve this via a **Greedy Prefix-Trie Bitmask**. During generation step $t$, if the generated subword `c` forms a valid prefix path within the WordNet trie, it inherits $\gamma=1$ conditionally. If the subsequent step $t+1$ breaks the prefix path (e.g., completing a non-entity word), the token is retrospectively unmasked in the contextual cache. 

**The Broken OCR Concession:** We explicitly acknowledge a severe boundary: broken OCR strings containing erroneous characters (e.g., "199$") will fail the Tier 2 regex and break the Prefix-Trie, defaulting to $\gamma=0$. Consequently, the intervention is inherently bypassed precisely when the OCR engine struggles the most. Rather than artificially forcing a match, our evaluation protocol maps these exact logit trajectories to quantify the practical ceiling of dictionary-gated intervention.

## 4. Evaluation Protocol and Experimental Design

Because Token-Local Visual Intervention operates as a hybrid test-time adaptation with specific structural hypotheses, we organize the evaluation plan into three pre-registered falsifiable evidence chains and five distinct defense lines. The planned protocol utilizes **LLaVA-1.5** (MLP projector) and **Qwen-VL-Chat**. All subsequent sections outline our hypothesis-driven execution blueprint.

### 4.1 Chain A: Hallucination Reduction

**Hypothesis:** The intervention mitigates object hallucination specifically through localized decode-time routing, not merely through the parametric knowledge exposed during the 5k-sample test-time calibration.
- **Evaluation Protocol:**
  - **Metrics:** POPE (reported across Random, Popular, and Adversarial splits), CHAIR (Object), and AGL (Acc).
  - **Planned Output (Table 1):** 
    - *Rows:* Base | `Base + 5k LoRA` | DoLa* | VCD* | OPERA* | `BRA_calib` (Full VASM)
  - **Defense Line 1 (The LoRA Objective Boundary - Crucial Execution):** To definitively isolate "routing" from "parametric injection," the `Base + 5k LoRA` control is explicitly trained via standard autoregressive next-token prediction on the exact 5,000 VG samples used for $\Phi_{calib}$. If this LoRA control alone closes $>95\%$ of the performance gap compared to full `BRA_calib`, the decode-time routing hypothesis is mathematically redundant and falsified.

### 4.2 Chain B: Structure and Reasoning Preservation

**Hypothesis:** The VASM module protects structural language, but reliance on explicit dictionaries introduces a vulnerability where Out-Of-Vocabulary (OOV) and specialized terms bypass intervention entirely, conflating "preservation" with mere "inaction."
- **Evaluation Protocol:**
  - **Metrics:** MMBench, MME, MMMU(Hard).
  - **Planned Output (Table 2 & Figure 1):** 
    - *MMMU Stratification:* We will report MMMU(Hard) performance strictly stratified by discipline (e.g., Math vs. Biology) to document variance (e.g., regex excelling on Math numbers, WordNet failing on specialized Biology terms).
    - *Ablation Scatterplot:* We contrast `BRA_calib` (Full VASM) against a strict `BRA_no_VASM` ablation, mapping an Intervention Trigger Rate Profiling scatterplot: **OOV Rate (%)** (X-axis) vs. **Accuracy Drop relative to Base** (Y-axis).
  - **Defense Line 2 (The Inaction Boundary):** We establish a hard visual boundary at a 10% trigger rate. If the VASM intervention rate drops below 10% on highly specialized MMMU subsets, we explicitly conclude that the mask preserves reasoning primarily by disabling the intervention.

### 4.3 Chain C: Local Evidence Value

**Hypothesis:** On dense spatial tasks, token-local visual reweighting isolates specific evidence that diffuse global pooling natively washes out (`BRA_zero` failure), mapping exact localized patch coordinates to specific token logits.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (Accuracy/ANLS) and DocVQA (ANLS).
  - **Visual Proof of Locality (Mandatory Main-Text Figure):** We will provide a side-by-side qualitative heatmap overlay on the source image. The figure will contrast the diffuse, non-specific activation of `BRA_MeanPool` across the entire image against the strictly localized bounding box activation achieved by `BRA_calib` for a specific generated token.
  - **Defense Line 3 (Compositional Coherence Ablation):** Evaluated strictly on CHAIR for highly overlapping objects (e.g., "person" and "clothing"), we ablate $\Phi_{calib}$ trained *with* vs *without* the dual-layered Compositional Coherence Margin (Graph + Semantic Sim) to prove the necessity of the complex InfoNCE math.

### 4.4 Defense Lines 4 & 5: Algorithmic Bottlenecks and OCR Bypasses

We monitor execution feasibility through the following empirical defense lines:
- **Defense Line 4 (Inter-Token Latency & Rolling Decay):** Logits manipulation across the full vocabulary using spatial routing introduces CUDA bottlenecks. We will plot Inter-Token Latency (ms/token) against generation step $t$. Additionally, we will plot the Sliding Window EMA ($\theta_{active}^{(t)}$) to prove that the rolling decay smoothly mitigates threshold drift without the jarring generative shifts of a step-function fallback.
- **Defense Line 5 (BPE/OCR Bypass Logit Trajectory):** To transparently document BPE/Regex brittleness (Section 3.3), we will map out the precise logit trajectory (top-5 logits across step $t$) of a broken OCR token. This graph will explicitly show where the correct localized token was suppressed because $\gamma=0$ bypassed the visual intervention, demonstrating the exact magnitude of the missed analytical opportunity.

## 5. Discussion and Limitations

The strength of Token-Local Visual Intervention lies in its structurally bounded proposition: injecting token-local visual evidence at decode-time while protecting language integrity via Dictionary-Gated VASM. However, its execution is governed by severe theoretical and operational constraints:

1. **The Falsifiability of Hybrid Calibration:** Deep self-attention demonstrably washes out localized patch fidelity. The required shift to the hybrid `BRA_calib` pipeline differentiates this from purely zero-shot, training-free regularizers (DoLa, VCD). Its scientific validity rests entirely on surpassing the standard autoregressive `Base + 5k LoRA` boundary (Defense Line 1).
2. **Rolling Decay Window Sensitivity:** While the Sliding Window EMA eliminates the semantic breaks of a hard fallback, its efficacy is highly sensitive to the window size $W$. Continuous threshold softening inherently means the model relies progressively less on strict visual evidence as sequences lengthen.
3. **BPE/OCR Brittleness:** Deterministic regex gating (Tier 2) and Prefix-Trie matching are brittle to broken OCR strings; fragmented tokens that fail the tree path default to $\gamma=0$, bypassing visual intervention exactly when the vision encoder struggles most. 
4. **Latency Bottlenecks:** Executing spatial pooling and dictionary bitmasking incurs non-trivial CUDA kernel overheads, directly impacting ms/token latency, positioning this framework strictly as a high-precision analytical tool rather than a high-throughput serving API.

## 6. Conclusion and Future Execution

This paper outlines a functionally precise framework for mitigating MLLM hallucination through VASM-gated, token-local logits intervention. By formalizing spatial washout (`BRA_zero` vs `BRA_calib`), integrating mathematically rigorous Compositional InfoNCE margins, engineering a rolling-decay prefill threshold, and establishing greedy prefix-trie semantic masking, we construct a technical foundation for decode-time visual routing. To prevent conflating inaction with preservation, the claims herein are governed by a pre-registered evaluation protocol spanning three evidence chains and five distinct defense lines, establishing a verifiable roadmap for empirical execution.

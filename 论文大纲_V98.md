# Token-Local Resonance Anchoring: Decode-Time Visual Logit Bias for Structure-Preserving MLLM Grounding

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. Existing inference-time mitigation strategies, including contrastive decoding, attention control, and layer-contrast methods, are highly competitive, but they do not directly provide a framework for injecting **token-local visual evidence** into decode-time logits adjustment while preserving language structure. We present **Token-Local Resonance Anchoring (TLRA)**, a decode-time intervention framework that reweights candidate tokens using localized support extracted from cached visual states.

TLRA operates on three principles. First, **token-local evidence reweighting** replaces coarse global visual contexts with candidate-wise support computed from localized visual states, mapping spatial tokens to lexical logit biases. Second, **Adaptive Top-k resonance** stabilizes local evidence extraction under a bounded candidate window with dynamic logit clipping. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via an offline precomputed vocabulary lookup and deterministic BPE continuation inheritance. We explicitly separate a strict zero-shot probe (`TLRA_zero`) from a lightweight calibrated variant (`TLRA_calib`). To ensure rigorous evaluation, `TLRA_calib` is tested against a matched-budget `Base + LoRA` baseline trained on identical calibration data. 

The evaluation protocol explicitly stress-tests structural vulnerabilities and computational boundaries. It measures hallucination reduction against Average Generation Length (AGL) collapse, audits VASM's polysemy failure rate via an online Oracle-VASM ablation, and isolates the specific value of token-local routing against a Global Visual State baseline. By tracking both task accuracy and exact per-step latency overhead ($O(M \times N_v)$), this work establishes a falsifiable, boundary-aware framework for decode-time visual control in MLLMs.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable capabilities, yet their autoregressive decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible but weakly supported by the visual input. This failure is particularly damaging in **fine-grained physical grounding** scenarios such as small-object recognition, spatial positioning, and counting, where the requisite evidence is localized rather than global.

Current inference-time mitigation strategies represent major advances. Contrastive decoding methods (e.g., `VCD`) compare distributions under perturbed and clean inputs. Attention-based control methods (e.g., `OPERA`) penalize pathological autoregressive behavior. Layer-contrast methods (e.g., `DoLa`) exploit discrepancies between shallow and deep representations. While these approaches successfully modulate internal model confidence, they leave a specific, complementary question unanswered: **How can we inject token-local visual evidence directly into decode-time logits adjustment without damaging syntax, multi-token entities, or downstream reasoning?**

Executing localized intervention at decode time is nontrivial due to semantic fragility and computational walls. If an intervention suppresses visually unsupported tokens without understanding language structure, it risks destroying functional syntax or fragmenting named entities. Conversely, computing cross-modal similarity between $M$ candidate tokens and $N_v$ spatial visual patches at every single autoregressive step introduces a severe latency bottleneck. Furthermore, if localized decode-time interventions fail to outperform simple prefill-time adaptations (e.g., LoRA) or global visual pooling, their architectural complexity is unjustified.

We introduce **Token-Local Resonance Anchoring (TLRA)**, a decode-time framework designed to score candidate tokens using localized visual support. TLRA avoids the per-step matrix multiplication bottleneck by conceptualizing the intervention as a **static late-fusion visual logit bias**: visual-to-lexical projections are computed once post-prefill, reducing the per-step decode overhead to an $O(M \times N_v)$ scalar lookup and top-k routing operation over a bounded candidate window. To prevent structural collapse, TLRA introduces Vocabulary-Anchored Semantic Masking (VASM), an offline BPE-level dictionary lookup that explicitly shields non-visual functional tokens from intervention. 

To ensure methodological rigor, we enforce strict architectural and experimental boundaries. TLRA assumes a spatial-preserving visual architecture (e.g., ViT-to-LLM MLP projectors) and does not natively support dense bottleneck architectures (e.g., Q-Formers) where spatial patch provenance is destroyed. Experimentally, the paper enforces a strict split between a training-free probe (`TLRA_zero`) and a lightweight calibrated variant (`TLRA_calib`). `TLRA_calib` is evaluated against internal matched-calibrator controls (e.g., `Global_Visual_State`) and a matched-budget `Base + LoRA` baseline to isolate the precise value of decode-time locality. Finally, we explicitly audit the semantic fragility of VASM by comparing our static heuristic against a computationally heavy Oracle VASM (online POS tagger), quantifying the exact penalty of polysemy and context-blindness in our design.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Architectural Scope and Fairness Boundaries

TLRA relies on the extraction of token-local support from visual states. Consequently, it requires MLLM architectures that preserve spatial provenance between the vision encoder and the LLM (e.g., standard ViT + MLP projector setups like LLaVA). Architectures that heavily compress or entangle spatial dimensions (e.g., Q-Formers or Perceiver resamplers) fall outside the supported scope, as spatial tokens no longer map consistently to localized image patches.

We explicitly separate TLRA into two evaluation tracks:
1. **`TLRA_zero`**: a strict decode-time probe in which the model-native readout is applied directly to cached visual states.
2. **`TLRA_calib`**: a calibrated variant utilizing a lightweight linear projection `Phi_calib` to map cached visual states into the lexical logit space while the base MLLM remains frozen.

Because `Phi_calib` introduces external supervision (trained on a bounded conceptual-caption subset), any claim about the efficacy of decode-time local routing must survive two fairness checks. First, internal parity: localized routing must outperform both a `TLRA_MeanPool` and a `Global_Visual_State` baseline sharing the exact same `Phi_calib`. Second, matched-budget fairness: `TLRA_calib` must be benchmarked against a `Base + LoRA` model adapted on the identical calibration subset, proving that dynamic decode-time visual grounding achieves something static prefill-time adaptation mathematically cannot.

### 3.2 Static Late-Fusion and $O(M \times N_v)$ Complexity

To avoid prohibitive latency penalties, TLRA shifts the heavy visual-to-lexical projection out of the autoregressive loop. After the prefill phase, `Phi_calib` maps the $N_v$ spatial visual states to the full vocabulary dimension $V$, yielding a static visual logit bias matrix $B \in \mathbb{R}^{N_v \times V}$. This operation is executed exactly once per image.

At each decoding step $t$, TLRA restricts intervention to a bounded candidate set:
$$
\mathcal{C}_t = \text{Top-}M(L_{orig}),
$$
where $M$ is a globally locked candidate window (e.g., $M=50$). The routing overhead per step is thus reduced to extracting the specific $M$ columns from $B$, resulting in an $O(M \times N_v)$ scalar lookup and routing operation, rather than a dense neural network forward pass.

For each candidate $c \in \mathcal{C}_t$, TLRA computes local support by aggregating the top spatial patches:
$$
k = \max(k_{min}, \lceil \rho \cdot N_v \rceil)
$$
The raw local support is:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{B_{j, c}}{\tau_{sim}}
$$

### 3.3 Stabilized Resonance Penalty with Logit Clipping

We convert raw local support into a relative evidence score within the candidate window:
$$
\hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)
$$

Intervening directly based on the candidate logit spread $\Delta_L = \max(L_{orig}) - \min(L_{orig})$ can be destructive if the base model's distribution is highly spiked (anomalously large $\Delta_L$). To prevent distribution destruction, TLRA applies a clipped, scaled penalty:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \min(\Delta_L, \delta_{max}) \cdot (1 - \hat S(c))
$$
where $\delta_{max}$ mathematically bounds the maximum logit penalization, and $\gamma(c)$ is the binary structural protection mask.

### 3.4 Vocabulary-Anchored Semantic Masking (VASM)

To preserve syntax and multi-token entities, TLRA utilizes VASM, an offline precomputed mask:
- $\gamma(c) = 1$ for visually groundable physical-entity root tokens.
- $\gamma(c) = 0$ for structural, functional, and ungroundable tokens.

Because running a heavy online POS tagger inside the autoregressive loop violates latency constraints, VASM relies on static dictionary lookups combined with deterministic BPE continuation inheritance (subwords inherit the mask state of their active root token). 

We acknowledge the inherent semantic fragility of this approach: static lookups cannot resolve contextual polysemy (e.g., distinguishing "Apple" the fruit from "Apple" the company). Consequently, arbitrary text-in-image reasoning (OCR) is treated as a strict negative control. To quantify the exact performance penalty of static masking, the evaluation protocol mandates a comparison against an `Oracle_VASM` ablation, which utilizes a computationally unrestricted online POS tagger to establish the theoretical upper bound of the framework.

## 5. Discussion and Limitations

The viability of Token-Local Resonance Anchoring depends entirely on surviving strict computational and structural boundaries. The primary limitations identified within this framework include:

1. **The Semantic Fragility of Static VASM:** Offline dictionary lookups inherently fail on context-dependent polysemy. While VASM protects basic syntax, words that transition between physical entities and abstract concepts (e.g., "Apple") trigger false positives. The performance gap between static VASM and Oracle VASM defines the cost of bypassing an online POS tagger.
2. **The Pareto Wall of $O(M \times N_v)$ Routing:** Even with late-fusion static logits precomputation, extracting and routing candidate tokens against $N_v$ spatial states at every autoregressive step introduces non-trivial kernel launch overhead. If large-scale models expand visual token volume significantly (e.g., $N_v > 4000$), the per-step scalar lookup becomes a severe bottleneck.
3. **The `Global_Visual_State` Threshold:** If the empirical gap between `TLRA_AdaptiveTopK` and a uniformly pooled `Global_Visual_State` baseline is negligible, it indicates that spatial locality at decode time is overridden by the LLM's internal representation, falsifying the core need for fine-grained patch-level routing. 
4. **Architectural Restraints:** TLRA mandates spatial patch preservation. MLLMs utilizing aggressive visual compression, such as Q-Formers or Perceiver modules, destroy the direct patch-to-token provenance required for adaptive resonance.
5. **Top-M Unrecoverability:** The intervention is mathematically bounded by the base model's candidate window. If the correctly grounded token has been marginalized below the Top-$M$ cutoff by strong language priors prior to intervention, TLRA cannot rescue it.

Future execution of this protocol will definitively confirm whether local decode-time visual intervention can exist as a practical, context-aware mechanism, or whether its computational and semantic trade-offs restrict it to a theoretical probe.

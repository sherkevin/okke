# Token-Local Resonance Anchoring: A Bounded Late-Fusion Visual Logit Bias for Entity-Centric MLLM Grounding

## Abstract

Multimodal Large Language Models (MLLMs) frequently hallucinate when strong language priors overpower the localized visual evidence required for grounding physical entities. While existing inference-time mitigation strategies modulate internal model confidence, they do not provide a framework for injecting **token-local visual evidence** directly into decode-time logits adjustment without damaging syntax or overwhelming hardware bandwidth. We present **Token-Local Resonance Anchoring (TLRA)**, a late-fusion decode-time intervention framework that acts as a linear visual classifier bias, reweighting candidate tokens using localized support extracted from cached visual states.

TLRA operates on three strict architectural principles. First, **late-fusion visual-to-lexical projection** maps spatial patches to BPE-level logit biases, executed dynamically or statically to navigate High Bandwidth Memory (HBM) uncoalesced read bottlenecks. Second, **Adaptive Top-k / Top-p resonance** stabilizes local evidence extraction under a bounded candidate window with dynamic logit clipping. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax via an offline BPE-level dictionary strictly bound to physical entity synsets, fortified against tokenization leakiness (e.g., prefix spaces, plurals). We rigorously delineate a geometric isotropy probe (`TLRA_zero`)—hypothesized to fail as a negative control verifying input-space anisotropy—from a lightweight calibrated variant (`TLRA_calib`). To ensure evaluation fairness, `TLRA_calib` is tested against a matched-parameter `Global_Visual_State` baseline and a `Base + LoRA` control trained on identical data.

The pre-registered evaluation protocol enforces a falsifiable audit of structural vulnerabilities and computational boundaries. It measures hallucination reduction against Average Generation Length (AGL) collapse, audits VASM's noun-specific bias by testing against action/verb hallucinations, and bounds the unrecoverable error via a Top-M "Hijacking CDF." By explicitly tracking HBM bandwidth utilization across High-Definition (HD) MLLM resolutions ($N_v \ge 3000$) and evaluating dynamic sliced-matmul versus static precomputation, this work establishes a strict, boundary-aware framework for decode-time visual routing.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable generative capabilities, yet their autoregressive decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible but weakly supported by the visual input. This failure is particularly damaging in **fine-grained physical grounding** scenarios—such as small-object recognition, counting, and spatial positioning—where the requisite evidence is highly localized rather than global.

Current inference-time mitigation strategies represent major advances. Contrastive decoding methods compare distributions under perturbed and clean inputs, while attention-based and layer-contrast methods penalize pathological autoregressive decay or exploit discrepancies between shallow and deep representations. While these approaches successfully modulate general model confidence, they leave a specific, complementary question unanswered: **How can we inject fine-grained, token-local visual evidence directly into decode-time logits adjustment without destroying syntax or hitting severe hardware memory walls?**

Executing localized intervention at decode time is nontrivial due to semantic fragility, geometric misalignment, and hardware physics. If an intervention suppresses visually unsupported tokens without understanding language structure, it risks destroying functional syntax. Furthermore, computing cross-modal similarity between $M$ candidate tokens and $N_v$ spatial visual patches at every single autoregressive step introduces a severe latency bottleneck. In modern HD MLLMs processing thousands of patches and massive vocabularies, naive intervention devolves into uncoalesced reads from High Bandwidth Memory (HBM), catastrophically degrading empirical throughput. Finally, if localized decode-time interventions fail to outperform simple prefill-time adaptations (e.g., LoRA) or global visual pooling, their architectural complexity is mathematically unjustified.

We introduce **Token-Local Resonance Anchoring (TLRA)**, a strictly bounded late-fusion framework functioning as a token-local visual classifier ensemble. TLRA extracts local spatial support to bias decode-time logits, strictly limited to **entity-centric grounding**. To prevent structural collapse, TLRA introduces Vocabulary-Anchored Semantic Masking (VASM), an offline BPE-level dictionary explicitly mapping physical entity synsets to binary subword masks (accounting for tokenizer quirks like prefix spaces and capitalization). VASM restricts the intervention to groundable nouns while shielding functional syntax, verbs, and prepositions.

To ensure methodological rigor, we enforce strict architectural and experimental boundaries across three evidence chains and five defenses. First, we formally interrogate the geometry of the joint multimodal input space via `TLRA_zero`, a zero-shot probe serving as a negative control to verify the anisotropy of MLP-projected features. The primary method, `TLRA_calib`, is evaluated against an internal matched-calibrator control (`Global_Visual_State`) and a matched-budget `Base + LoRA` baseline to isolate the precise value of decode-time spatial locality. Second, we explicitly audit the semantic fragility of VASM by comparing it against an `Oracle_VASM` (online POS tagger) and bounding its failure modes via an Action/Verb Hallucination stress test. Finally, we subject the system to a strict hardware scaling audit to identify the exact HBM uncoalesced read crossover point between static precomputation and dynamic sliced-matmul on high-resolution ($N_v > 3000$) inputs.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits under different visual conditions. Attention-based methods such as `OPERA` detect abnormal generation patterns during decoding. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and carry different robustness and latency trade-offs.

### 2.2 Local Evidence vs. Coarse Internal Control

High-resolution MLLMs operate over large spatial token fields rather than a single compact visual descriptor. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should explicitly test that locality.

This is not a sweeping claim about all external baselines. It is a narrower methodological claim: when a task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support should be tested against coarser internal alternatives under matched conditions.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Function words, discourse markers, and continuation fragments play a structural role distinct from visually grounded entity tokens. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this via a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Architectural Scope and the Geometric Isotropy Probe

TLRA targets standard ViT + MLP projector architectures (e.g., LLaVA) where spatial provenance is preserved across visual tokens $X_v$. To rigorously isolate the source of visual grounding improvements, we separate TLRA into two variants, establishing a strict theoretical baseline before introducing supervision:

1. **`TLRA_zero` (Geometric Isotropy Probe / Negative Control):** `TLRA_zero` computes cosine similarity directly between post-projector visual features ($X_v$) and the discrete text embeddings ($E_{in}$) within the LLM's input space. We formally hypothesize that this zero-shot retrieval will fail. The LLM's MLP projector maps visual tokens into the *input* activation space of the first transformer layer—a space optimized for forward-pass transformation, heavily anisotropic, and geometrically incoherent for symmetric dot-product retrieval against a static vocabulary matrix. `TLRA_zero` serves strictly as a pre-registered negative control to empirically prove this representation anisotropy to the community.
2. **`TLRA_calib` (Hybrid Calibrated Variant):** Stripped of complex routing jargon, `TLRA_calib` is a late-fusion visual classifier ensemble. A lightweight linear projection $W_{calib} \in \mathbb{R}^{D \times V}$ maps cached spatial visual states directly to the lexical logit space $V$, trained on a bounded conceptual-caption subset while the base MLLM remains frozen. 

Because `TLRA_calib` introduces external supervision, any claim about dynamic decode-time visual grounding must survive strict fairness checks against prefill-time adaptation.

### 3.2 The HBM Bandwidth Wall: Static vs. Dynamic Late-Fusion

At each decoding step $t$, TLRA restricts intervention to a bounded candidate set $\mathcal{C}_t = \text{Top-}M(L_{orig})$ (with dynamic $\text{Top-}p$ evaluated as an extension). To extract local support, we must map spatial features $X_v$ to lexical biases for these $M$ candidates. We evaluate two distinct architectural implementations to navigate the GPU hardware physics:

**A. Static Precomputation (The HBM Wall):** After the prefill phase, cached visual states are mapped to the full vocabulary dimension, yielding a static bias matrix $B = X_v \cdot W_{calib}$, where $B \in \mathbb{R}^{N_v \times V}$. While FLOP-efficient inside the autoregressive loop, gathering arbitrary non-contiguous token columns $c \in \mathcal{C}_t$ from $B$ exposes a severe hardware boundary. For HD MLLMs ($N_v \ge 3000$, $V \ge 150k$), $B$ balloons to $>1.2$GB. Because this far exceeds the L2 cache size of an H100 ($\sim 50$MB), gathering these columns at every step demands massive uncoalesced reads directly from High Bandwidth Memory (HBM), bottlenecking empirical `tokens/s`.

**B. Dynamic Sliced-Matmul:** Instead of precomputing $B$, we dynamically compute the projection *only* for the Top-$M$ tokens at each step: $B_{step} = X_v \cdot W_{calib}[:, \mathcal{C}_t]$. While $W_{calib}$ is fixed size, reading the sliced weights still incurs HBM read costs, trading FLOPs for reduced matrix footprint. 

For each candidate $c \in \mathcal{C}_t$, TLRA aggregates local support from the top spatial patches, converting raw local support into a relative evidence score $\hat S(c)$:
$$
S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{B_{j, c}}{\tau_{sim}} \quad \Rightarrow \quad \hat S(c) = \text{Softmax}\big(S_{raw}(c) / \tau_{evidence} \mid c \in \mathcal{C}_t\big)
$$

### 3.3 Stabilized Resonance Penalty with Logit Clipping

Directly intervening based on candidate logit spread $\Delta_L = \max(L_{orig}) - \min(L_{orig})$ can destroy spiked, high-confidence distributions. TLRA applies a clipped penalty to boundedly suppress visually unsupported tokens:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot \min(\Delta_L, \delta_{max}) \cdot (1 - \hat S(c))
$$
where $\delta_{max}$ bounds the maximum penalization, $\alpha$ scales intensity, and $\gamma(c) \in \{0, 1\}$ is the structural protection mask.

### 3.4 VASM: Entity-Centric Masking and BPE Leakiness

To preserve syntax without triggering massive latency penalties via online POS tagging, TLRA utilizes VASM, an offline precomputed deterministic binary mask ($\gamma \in \{0, 1\}^V$). We extract physical entity synsets (nouns mapping to physical objects/animals/structures) from English WordNet. 

**The BPE Tokenization Mismatch:** WordNet maps to *words*, but LLMs output *BPE subwords*. A naive exact-match dictionary will catastrophically drop valid subwords due to prefix spaces (e.g., " table" vs "table"), capitalization, and pluralization. VASM explicitly expands the WordNet string dictionary across tokenizer permutations (adding standard prefixes like `" "` and `\n`) before applying the BPE encoding to capture the full surface area of physical entities. We quantify the success of this mapping via an explicit **BPE Collision Audit** metric in the evaluation.

**Noun Bias and Semantic Fragility:** Because VASM explicitly forces $\gamma(c) = 1$ for physical entities and $0$ for everything else, it systematically ignores actions (verbs) and spatial relations (prepositions). TLRA is strictly an **Entity-Centric** grounding mechanism. We quantify the exact cost of this semantic fragility by testing against a Verb Hallucination stress test and comparing the offline dictionary against an `Oracle_VASM` (online POS tagger).

## 5. Discussion and Limitations

The viability of Token-Local Resonance Anchoring depends entirely on surviving strict computational, structural, and hardware boundaries. By explicitly reframing TLRA as a late-fusion visual classifier ensemble tailored exclusively for entity-centric grounding, we identify the following primary limitations:

1. **The Noun-Bias and Semantic Fragility of VASM:** Offline dictionary lookups fail on context-dependent polysemy (e.g., "crane") and BPE tokenization anomalies. By forcing $\gamma(c) = 0$ outside physical entities, TLRA is fundamentally blind to action hallucinations (verbs) and relation hallucinations (prepositions). The performance gap on the Verb Stress-Test and against Oracle VASM formally defines the strict semantic boundaries of this method.
2. **The Geometric Anisotropy of `TLRA_zero`:** Assuming zero-shot dot-product alignment between post-projector visual features ($X_v$) and input token embeddings ($E_{in}$) is geometrically flawed. The expected failure of the `TLRA_zero` negative control empirically proves that MLP-projected visual tokens require subsequent transformer layers to contextualize and cannot be symmetrically retrieved without external supervision.
3. **The HBM Uncoalesced Read Bottleneck:** Gathering arbitrary token columns from a precomputed 1.2+ GB visual logit matrix demands massive uncoalesced reads from High Bandwidth Memory. As modern models scale visual patches ($N_v \ge 3000$), the method is strictly bottlenecked by memory bandwidth physics, requiring dynamic sliced-matmul optimizations or custom Triton kernels to remain viable.
4. **The `Global_Visual_State` Parity:** If the empirical gap between `TLRA_calib` and uniformly pooled `Global_Visual_State` is negligible under identical parameter and calibration budgets, it falsifies the core need for fine-grained spatial routing at decode time.
5. **Top-M Unrecoverability (The Hijacking CDF):** The intervention is mathematically bounded by the base model's candidate window. If strong language priors marginalize the ground-truth visual token below the Top-$M$ cutoff prior to intervention, no late-fusion logit penalty can rescue it.

Future execution of this protocol will definitively confirm whether local decode-time visual intervention can exist as a practical, context-aware mechanism, or if semantic fragility and HBM bandwidth walls restrict it to a theoretical probe.

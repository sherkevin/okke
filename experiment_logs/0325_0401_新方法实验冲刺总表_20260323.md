# 新方法实验冲刺总表（3.25 摘要 + 4.1 初稿）

日期：`2026-03-23`

本文档定位：

- 这是 **新方法正式实验清单**，不是消融备忘录。
- 优先级顺序必须是：`主对比实验 > 跨模型主表 > 泛化/成本表 > 模块验证 > 消融`
- 所有数值先写 `TBD`，后续结果出来后直接回填。

当前正式比较边界：

- 宿主模型：`Qwen3VL`、`LLaVA1.5`、`InstructBLIP`
- 对比方法：`beam_search`、`DoLa`、`OPERA`
- 我们的方法：`UniGround-new`
- 主数据集：`POPE`
- 泛化数据集：`CHAIR`

填写规则：

- 所有主表必须可回溯到具体结果文件
- 所有时间字段必须注明 `ms` 或 `s`
- 所有显存字段必须注明 `GB`
- 如果某行尚未跑通，不填猜测值，只写 `TBD`

## 0. 统一命名与正式口径

### 0.1 模型与方法命名

- `Qwen3VL`：当前主宿主，优先作为摘要与初稿第一宿主
- `LLaVA1.5`：第二宿主，对应跨家族验证
- `InstructBLIP`：第三宿主，对应更老式 VLM 架构验证

- `beam_search`：宿主默认生成基线
- `DoLa`：decode-time 对比基线
- `OPERA`：attention / retrospection 路线强基线
- `UniGround-new`：我们当前新方法主线

### 0.2 当前主方法定义

`UniGround-new` 统一指：

- `object-query verification`
- `object-specific detector retrieval`
- frozen external encoder space
- `Psi_univ` sidecar scoring
- decode-time logits control

### 0.3 当前主结果默认配置

- 主运行卡：`GPU1`
- 主表默认配置：`full-gpu + eval_batch_size=1`
- `eval_batch_size > 1` 只作为吞吐/效率实验，不自动进入主结果表，除非后续明确证明与 `batch=1` 完全等价

### 0.4 每行结果最小字段

- `result_json`
- `host_model`
- `dataset`
- `split`
- `method`
- `sample_count`
- `accuracy`
- `precision`
- `recall`
- `f1` 或 `chair_i / chair_s`
- `peak_vram_gb`
- `avg_precompute_ms`
- `candidate_construction_ms`
- `sidecar_scoring_ms`
- `bridge_redistribution_ms`
- `total_step_ms`

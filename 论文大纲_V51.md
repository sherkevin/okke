# Dictionary-Gated Token-Local Visual Intervention for Hallucination Reduction in Image-LLMs

## Abstract
Multimodal Large Language Models (MLLMs) remain vulnerable to object hallucination when overarching language priors overshadow the localized visual evidence required for dense spatial grounding. Existing inference-time mitigation strategies (e.g., DoLa, VCD, OPERA) successfully regularize global token probabilities or attention distributions; however, they are orthogonal to the explicit routing of spatial evidence. This paper investigates how to inject **token-local visual evidence** directly into the terminal logits space at decode time without damaging functional language structure. We present a functional **Token-Local Visual Intervention** framework—a dictionary-gated logits adjustment utilizing localized visual support extracted from contextualized final-layer hidden states. 

Strictly scoped to 2D Image-LLMs, this framework addresses three structural barriers. First, acknowledging the near-universal spatial washout inherent to deep self-attention, we utilize a zero-shot diagnostic (`BRA_zero`) that operationally necessitates a shift to a lightweight, dynamically calibrated bound (`BRA_calib`). This hybrid test-time adaptation uses an isolated 5k-sample prior optimized via spatially-aware hard negative sampling, fortified by both graph-based and semantic-similarity compositional margins to avoid destroying overlapping objects. Second, to combat autoregressive threshold drift without introducing brittle step-functions, we extract localized support via an Entropy-Scaled Adaptive Top-$k$ mechanism governed by a **Rolling Decay** activation threshold. Third, to protect functional syntax, we implement a **Visual-Aware Semantic Masking (VASM)** module—a dual-tier shield utilizing an ~85k WordNet dictionary and deterministic regex triggers.

To rigorously assess algorithmic overhead and genuine spatial utility, we propose a pre-registered evaluation protocol across three definitive evidence chains: (A) Hallucination Reduction (POPE/CHAIR + AGL), (B) Structure and Reasoning Preservation (`BRA_no_VASM` ablation on MMMU/MMBench), and (C) Local Evidence Value (`BRA_zero` vs `BRA_MeanPool` on FREAK/DocVQA). Backed by five explicit defense lines tracking everything from Inter-Token Latency (ITL) to BPE bypass rates, this framework establishes a verifiable proposition for spatially-aware, decode-time intervention.

## 1. Introduction

Multimodal Large Language Models (MLLMs) exhibit remarkable visual-linguistic capabilities, yet their decoding behavior remains fundamentally susceptible to object hallucination. The model frequently generates text that is fluent and syntactically plausible but weakly supported by the actual image input. This failure mode is exceptionally penalizing in dense spatial grounding scenarios—such as OCR-heavy documents, small-object recognition, and complex spatial reasoning—where the relevant visual evidence is inherently token-local rather than diffuse.

Current inference-time mitigation strategies typically intervene via noise-contrastive decoding (e.g., VCD), layer-contrast decoding (e.g., DoLa), or attention-based pattern penalties (e.g., OPERA). It is critical to establish that these methods serve as highly successful, state-of-the-art regularizers that effectively discipline overarching language-side or attention-side generation heuristics. We tackle an orthogonal problem: spatial evidence routing. Because existing methods operate on global generation dynamics, they are not natively designed to address the following structural question:

**How can we inject strictly token-local visual evidence into decode-time logits adjustment without damaging language structure, multi-token entities, or downstream reasoning ability?**

This paper advances a direct, functional hypothesis: a framework that scores candidate tokens using strictly localized visual support can reduce object hallucination faithfully, provided it incorporates explicit structure-preserving safeguards. We functionalize this intervention as **Token-Local Visual Intervention**. In this work, we strictly bound our scope to 2D Image-LLMs, deliberately discarding unsupported generalizations to spatiotemporal video domains to ensure tight, defensible claims regarding spatial locality.

Executing this intervention requires mathematically overcoming three structural barriers. First, **spatial washout** in embedding spaces. Deep self-attention almost universally diffuses localized patch coordinates by the final layer. We formalize this via a Bounding Box Retrieval Assessment (`BRA_zero`), which, upon expected failure, triggers a shift to a lightweight, context-preserving test-time calibration (`BRA_calib`). Crucially, this calibration requires spatially-aware contrastive sampling that does not inadvertently destroy naturally co-occurring compositional objects. Second, the computational complexity and drift of **dynamic spatial pooling**. We replace static thresholds with an entropy-scaled dynamic activation governed by a continuous rolling decay mechanism, eliminating brittle step-function fallbacks during long-context generation. Third, **structural collapse**: hallucination suppression inherently risks penalizing Byte-Pair Encoding (BPE) fragments or OCR-critical entities. We resolve this via a Visual-Aware Semantic Masking (VASM) module, openly tracking when the mask actively preserves structure versus when it merely defaults to inaction.

To ensure rigorous scientific boundaries, our experimental design is organized into a pre-registered execution protocol defining three primary evidence chains and five explicit defense lines:
- **Chain A (Hallucination Reduction):** Validating mechanism value on POPE, CHAIR, and AGL against a strictly parameter-matched `Base + 5k LoRA` baseline.
- **Chain B (Structure and Reasoning Preservation):** Proving VASM dynamics via MMBench, MME, and MMMU(Hard), strictly ablating `BRA_no_VASM` to prevent conflating "preservation" with "inaction."
- **Chain C (Local Evidence Value):** Demonstrating the definitive superiority of localized evidence on FREAK and DocVQA by contrasting `BRA_zero` against `BRA_MeanPool`.
Through these bounded chains and corresponding defense lines tracking Inter-Token Latency (ITL) and BPE trace trajectories, this framework establishes a verifiable proposition for spatially-aware, decode-time intervention.

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as VCD compare logits produced under different visual conditions. Attention-based methods such as OPERA detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as DoLa compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they focus on different intervention signals and induce different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large token fields that encode spatial or spatio-temporal layout. For dense grounding tasks, the useful evidence for a word is often localized to a small region or a small frame interval. This makes token-local evidence particularly attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. BRA is designed around the claim that decode-time hallucination suppression should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser alternatives. That claim is experimentally verifiable through internal ablations such as `BRA_zero` vs `BRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded nouns or verbs. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. BRA addresses this through a probabilistic structural mask rather than through entropy-only heuristics.

## 3. Methodology

### 3.1 Embedding Asymmetry, Spatial Washout, and Compositional InfoNCE

To construct a valid local support metric, we evaluate the *contextualized* final-layer LLM hidden states corresponding to the visual sequence, $h_L^{(v_j)} \in \mathbb{R}^{D_{model}}$. However, deep self-attention demonstrably diffuses spatial coordinates by the final layer. We formalize this via an operational boundary:

1. **`BRA_zero` (Bounding Box Retrieval Assessment - Zero-Shot)**: We evaluate zero-shot bounding box retrieval using the model's native `lm_head` as the projection $\Phi_{zero}$. If this fails to achieve an average bounding box $\text{IoU} > 0.15$ on a 500-sample spatial validation set, spatial fidelity is mathematically declared washed out.
2. **`BRA_calib` (Hybrid Test-Time Adaptation)**: Upon triggering the washout threshold, the framework shifts to $\Phi_{calib}$, a 2-layer MLP projection mapping $\mathbb{R}^{D_{model}} \rightarrow \mathbb{R}^{D_{model}}$. This exactly matches the pre-trained `lm_head` dimension, adding negligible FLOPs strictly during candidate evaluation. It is optimized on 5,000 size-stratified Visual Genome (VG) pairs strictly disjoint from test sets.

**Compositional-Aware Hard-Negative Sampling:** We optimize $\Phi_{calib}$ via a contrastive InfoNCE loss. A naive spatial exclusion zone ($\text{IoU} < 0.1$) fails to disentangle overlapping objects. Therefore, valid negative patches ($W_{vocab}[c^-]$) include *Hard Negatives* ($\text{IoU} > 0.5$, heavily overlapping with the anchor, differing semantic category). However, to prevent **Compositional Destruction** (e.g., repelling a "rider" from a "horse"), we enforce a dual-layered *Compositional Coherence Margin*: 
1. *Graph Veto:* Hard negatives are vetoed if the anchor and candidate share a parent-child relationship in the VG scene graph.
2. *Semantic Veto:* Acknowledging VG annotation gaps, we implement a secondary semantic similarity threshold (via frozen CLIP text embeddings); if $\cos(c_{anchor}, c_{candidate}) > \tau_{comp}$, the negative pair is vetoed, safely preserving unannotated co-occurring objects.

### 3.2 Entropy-Scaled Top-$k$ Pooling & Rolling Decay Activation

At decoding step $t$, we apply a threshold-gated Sigmoid activation:
$$
\tilde{P}^{(v_j)}(c) = \text{Sigmoid}\left(\frac{logit^{(v_j)}[c] - \theta_{active}^{(t)}}{\tau_{sim}}\right) \cdot \mathbb{I}\Big(logit^{(v_j)}[c] > \theta_{active}^{(t)}\Big)
$$
We derive an **entropy-scaled dynamic threshold base** during the prefill stage. We calculate the visual hidden-state entropy $\mathcal{H}_{vis}$ over the spatial grid, normalizing it by $f(N_v) = \log(N_v)$. The base threshold is: $\theta_{base} = \mu_A + \lambda\left(\frac{\mathcal{H}_{vis}}{f(N_v)}\right) \cdot \sigma_A$.

**The Rolling Decay Fallback Mechanism:** Deriving $\theta_{base}$ strictly at prefill risks threshold obsolescence during long autoregressive generation ($t > 100$). Replacing a brittle step-function fallback, we implement a **Sliding Window Exponential Moving Average (EMA)**. We monitor the *Valid Activation Rate* $R_t$ ($\%$ of targeted terms surpassing the threshold). The active threshold dynamically decays:
$$
\theta_{active}^{(t)} = \theta_{base} \cdot \min\left(1.0, \frac{\text{EMA}(R_{t-W:t}) + \epsilon}{\tau_{target}}\right)
$$
This ensures a smooth, continuous softening of the intervention requirement as the generated context diverges from the initial visual anchor, eliminating sudden semantic shifts mid-sentence.

### 3.3 Visual-Aware Semantic Masking (VASM)

The final candidate logits are adjusted by penalizing tokens lacking local visual backing, gated by a deterministic Visual-Aware Semantic Masking (VASM) module $\gamma(c) \in \{0,1\}$:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c))
$$
We utilize a **Dual-Tier Protection Architecture**:
1. **Tier 1 (Root Dictionary):** Offline WordNet *physical entity* subtree ($\approx 85,000$ synsets).
2. **Tier 2 (Deterministic Format Triggers):** For DocVQA and dense reasoning, numerical strings and capitalized entities are assigned $\gamma=1$ via deterministic regex logic, forcing localized evidence checking on dates and numbers.

**BPE Fragmentation & The Broken OCR Concession:** Subword fragments inherit the $\gamma$ weight of their root via a greedy string matching protocol and a GPU-side bitmask. However, we explicitly acknowledge a severe boundary: broken OCR strings containing erroneous characters (e.g., "199$") will fail the Tier 2 regex, defaulting to $\gamma=0$. Consequently, the intervention is inherently bypassed precisely when the OCR engine struggles the most. Rather than artificially forcing a threshold, we commit to mapping these exact logit trajectories in our evaluation protocol to quantify the practical ceiling of OCR intervention.

## 4. Evaluation Protocol and Experimental Design

Because Token-Local Visual Intervention introduces specific structural hypotheses, we organize the evaluation plan into three pre-registered falsifiable evidence chains and five distinct defense lines. The protocol utilizes **LLaVA-1.5** (MLP projector) and **Qwen-VL-Chat** (2D-RoPE). All subsequent sections outline our hypothesis-driven execution plan.

### 4.1 Chain A: Hallucination Reduction

**Hypothesis:** The intervention mitigates object hallucination specifically through localized decode-time routing, not merely through the parametric knowledge exposed during test-time calibration.
- **Evaluation Protocol:**
  - **Metrics:** POPE (F1, FPR), CHAIR (Object), and AGL (Acc).
  - **Planned Output (Table 1):** 
    - *Rows:* Base | `Base + 5k LoRA` | DoLa* | VCD* | OPERA* | `BRA_calib` (Full VASM)
  - **Defense Line 1 (The LoRA Objective Boundary):** To definitively isolate "routing" from "parametric injection," the `Base + 5k LoRA` control is explicitly trained via standard autoregressive next-token prediction on the 5,000 VG samples. If this LoRA control alone closes $>90\%$ of the performance gap compared to full `BRA_calib`, the decode-time routing hypothesis is falsified.

### 4.2 Chain B: Structure and Reasoning Preservation

**Hypothesis:** The VASM module protects structural tokens, but reliance on explicit dictionaries introduces a vulnerability where Out-Of-Vocabulary (OOV) terms bypass intervention entirely, conflating "preservation" with mere "inaction."
- **Evaluation Protocol:**
  - **Metrics:** MMBench, MME, MMMU(Hard).
  - **Planned Output (Figure 1 & Ablation):** We contrast `BRA_calib` (Full VASM) against a strict `BRA_no_VASM` ablation. We will map an Intervention Trigger Rate Profiling scatterplot: **OOV Rate (%)** (X-axis) vs. **Accuracy Drop relative to Base** (Y-axis).
  - **Defense Line 2 (The Inaction Boundary):** We establish a hard visual boundary at a 10% trigger rate. If the VASM intervention rate on MMMU(Hard) drops below 10%, we explicitly conclude that the mask preserves reasoning primarily by disabling the intervention, acting as a safeguard rather than an active routing agent.

### 4.3 Chain C: Local Evidence Value

**Hypothesis:** On dense spatial tasks, token-local visual reweighting isolates specific evidence that diffuse global pooling natively washes out (`BRA_zero` failure), provided Tier 2 regex triggers catch OCR targets.
- **Evaluation Protocol:**
  - **Benchmarks:** FREAK (Accuracy/ANLS) and DocVQA (ANLS).
  - **Planned Output (Table 2):** Isolating `Base`, `BRA_MeanPool`, `BRA_calib` (Tier 1 Only), and `BRA_calib` (Tier 1+2).
  - **Defense Line 3 (Compositional Coherence Ablation):** Evaluated strictly on CHAIR for highly overlapping objects (e.g., "person" and "clothing"), we ablate $\Phi_{calib}$ trained *with* vs *without* the dual-layered Compositional Coherence Margin (Graph + Semantic Sim) to prove the necessity of the complex InfoNCE sampling.

### 4.4 Defense Lines 4 & 5: Algorithmic Bottlenecks and OCR Bypasses

We monitor execution feasibility through the following empirical defense lines:
- **Defense Line 4 (Threshold Attrition & Rolling Decay):** Plotting Sequence Length ($t=1 \dots 500$) against the Valid Activation Rate. This graph will track the sliding window EMA ($\theta_{active}^{(t)}$), proving that the rolling decay smoothly mitigates threshold drift without the jarring generative shifts of a step-function fallback. Furthermore, we plot Inter-Token Latency (ITL) against generation step $t$ across expanding Candidate Window Sizes $M$ to prove the $M \times N_v$ sorting operation does not geometrically explode.
- **Defense Line 5 (BPE/OCR Bypass Logit Trajectory):** To transparently document Regex brittleness on subwords (Section 3.3), we will map out the precise logit trajectory of broken OCR tokens across the vocabulary space when $\gamma=0$ is forced, demonstrating the exact magnitude of the missed intervention opportunity.

## 5. Discussion and Limitations

The strength of Token-Local Visual Intervention lies in its structurally bounded proposition: injecting token-local visual evidence at decode-time while protecting language integrity via Dictionary-Gated VASM. However, its execution is governed by severe theoretical and operational constraints:

1. **The Falsifiability of Calibration:** Deep self-attention demonstrably washes out localized patch fidelity. The shift to the hybrid `BRA_calib` pipeline differentiates this from purely zero-shot regularizers (DoLa, VCD). Its validity rests entirely on surpassing the standard autoregressive `Base + 5k LoRA` boundary (Defense Line 1).
2. **Rolling Decay vs. Exactness:** While the Sliding Window EMA eliminates the semantic breaks of a hard fallback, continuous threshold softening inherently means the model relies progressively less on strict visual evidence as sequences lengthen.
3. **BPE/OCR Brittleness:** Deterministic regex gating (Tier 2) is brittle to broken OCR strings; fragmented BPE tokens default to $\gamma=0$, bypassing visual intervention exactly when the vision encoder struggles most. 
4. **Latency Bottlenecks:** Executing spatial pooling and $M \times N_v$ Top-$k$ sorting incurs CUDA kernel overheads, positioning this framework strictly as a high-precision analytical tool rather than a high-throughput serving API.

## 6. Conclusion and Future Execution

This paper outlines a functionally precise framework for mitigating MLLM hallucination through VASM-gated, token-local logits intervention. By formalizing spatial washout (`BRA_zero` vs `BRA_calib`), integrating semantic-similarity-backed compositional margins, engineering a rolling-decay prefill threshold, and establishing deterministic semantic masking, we construct a technically rigorous foundation for decode-time visual routing. To prevent conflating inaction with preservation, the claims herein are governed by a pre-registered evaluation protocol spanning three evidence chains and five distinct defense lines, establishing a verifiable roadmap for empirical execution.

# Token-Local Resonance Anchoring: Structure-Preserving Decode-Time Intervention for Multimodal Hallucinations

## Abstract

Multimodal Large Language Models (MLLMs) often hallucinate when strong language priors overpower the localized visual evidence required for the next decoding step. Existing inference-time mitigation strategies, such as contrastive decoding (VCD) and layer-contrast methods (DoLa), are highly competitive baselines, but they do not provide a direct framework for injecting **token-local visual evidence** into decode-time logits adjustment while preserving language structure. We present **Token-Local Resonance Anchoring (TLRA)**, a decode-time intervention framework that reweights candidate tokens using localized support extracted from contextualized visual states.

TLRA is built around three mathematical commitments. First, **token-local evidence reweighting** replaces coarse correction signals with candidate-wise support computed from localized visual patches. Second, **Adaptive Top-$k$ resonance with Prompt-Conditioned Pruning** bounds the decode-time overhead. To mitigate the architectural blindspot of prefill-only attention, pruning incorporates an entropy-driven fallback mechanism that dynamically defaults to dense visual evaluation during long-horizon, open-ended generation where prefill attention is highly dispersed. Third, **Vocabulary-Anchored Semantic Masking (VASM)** protects functional syntax and multi-token entities via a deterministic, WordNet-backed whitelist. To address modern BPE fragmentation without sacrificing common short words (e.g., "cat"), VASM employs a tokenizer-aware subword boundary check, intervening on valid physical entity prefixes while relying on the base LLM's autoregressive momentum to safely complete subword suffixes.

Because strict zero-shot intervention often collapses due to visual-lexical embedding asymmetry, TLRA utilizes a lightweight, single-layer calibrator (`TLRA_calib`) trained solely on generic image-text pairs. We explicitly acknowledge this cost: unlike purely training-free methods, TLRA introduces generic external data, but it operates as a strictly **base-model-frozen plug-in**. We organize the paper around a strict experimental contract: three evidence chains covering hallucination reduction (controlling for length collapse via AGL variance), structure preservation (validating VASM via perplexity and utilizing document-understanding as an exact negative control), and isolating the distinct value of local adaptive selection against global pooling (`TLRA_MeanPool`). The resulting evaluation protocol offers a positive, falsifiable, and rigorously bounded proposal for structure-preserving decode-time visual control.

## 1. Introduction

Multimodal Large Language Models (MLLMs) are highly capable, but their decoding behavior remains vulnerable to hallucination: the model may generate text that is fluent and plausible yet weakly supported by the actual visual input. This failure is especially damaging in fine-grained visual grounding scenarios such as small-object recognition, spatial positioning, and counting, where the required evidence is distinctly local.

Current inference-time mitigation strategies typically intervene in one of three ways. Contrastive decoding methods such as `VCD` compare different input conditions to reshape the next-token distribution. Attention-based control methods such as `OPERA` monitor abnormal generation patterns and penalize them during decoding. Layer-contrast methods such as `DoLa` exploit differences between shallow and deep model distributions. These are all legitimate, highly competitive baselines, but none of them directly answer the following positive hypothesis:

**How can we inject token-local visual evidence into decode-time logits adjustment without damaging language structure, subword morphology, or downstream reasoning ability?**

Executing this idea is nontrivial due to four foundational challenges. First, **embedding asymmetry**: visual states and output logits do not naturally live in the same space, making strict zero-shot lookup noisy. Second, **candidate limitation**: post-hoc logits intervention can only reweight candidates that the base model already keeps alive. Third, **structural collapse risk**: naive suppression of visually unsupported tokens easily damages function words and BPE-fragmented subwords. Fourth, **system cost at high resolutions**: computing patch-level support at every generation step scales poorly when modern vision encoders use dynamic patching ($N_v > 4096$).

TLRA centers on three design commitments to address these challenges:
1. **Decode-time token-local logits intervention via a Base-Model-Frozen Plug-in.** We utilize a lightweight, single-linear-layer calibrator to bridge embedding asymmetry. We explicitly clarify our taxonomy upfront: unlike purely training-free inference methods, TLRA introduces generic external distribution data. However, it leaves the base MLLM 100% frozen. Through strict parity ablations (e.g., controlling against global pooling and random patch selection using the exact same calibrator), we mathematically isolate the value of token-local decoding from this external prior.
2. **Adaptive Top-$k$ resonance with Entropy-Driven Fallback** as the local evidence extractor. By filtering patches based on prefill-stage prompt attention and evaluating only the Top-$M$ logits, TLRA bounds decode-time overhead. We explicitly acknowledge the "prefill pruning blindspot"—where visually grounded objects absent from the prompt emerge late in open-ended generation. Consequently, we incorporate a dynamic fallback mechanism based on attention entropy that defaults to full dense visual context preservation when generation horizons expand.
3. **VASM (Vocabulary-Anchored Semantic Masking)** as a deterministic structure-preserving layer. Built on a strict WordNet lexical whitelist, VASM defaults to bypassing intervention for structural and abstract tokens. We explicitly formalize two operational concessions: the *OCR paradox* (VASM natively ignores arbitrary OCR strings, effectively serving as a negative control on dense-text reading tasks) and *Tokenizer-Aware BPE Momentum* (VASM triggers on valid subwords while utilizing BPE boundary flags to prevent false-prefix collisions on abstract words, without blindly filtering out valid short words like "cup").

To maintain scientific rigor, this paper avoids exploratory "fishing expeditions" and commits to a definitive evaluation protocol. We establish three distinct chains of evidence: hallucination reduction, structural preservation (proving VASM's necessity via perplexity and a strict ~0% trigger rate on OCR tasks), and local-evidence validation (isolating the value of Top-$k$ anchoring against global pooling specifically for spatial relations).

## 2. Related Work

### 2.1 Decode-Time Hallucination Mitigation

Inference-time mitigation for MLLM hallucination has developed along several lines. Contrastive methods such as `VCD` compare logits produced under different visual conditions. Attention-based methods such as `OPERA` detect and penalize pathological generation behavior during the autoregressive process. Layer-contrast methods such as `DoLa` compare distributions from different network depths. These methods demonstrate that hallucination can be mitigated at inference time, but they rely on different intervention signals and involve different latency, robustness, and implementation trade-offs.

### 2.2 Local Evidence vs. Coarse Control Signals

High-resolution MLLMs no longer operate over a single compact visual descriptor. They operate over large spatial token fields, and in video settings they further inherit spatio-temporal structure. For dense grounding tasks, the useful evidence for a word is often concentrated in a small region of the image or a short temporal window. This makes token-local evidence attractive: if the next token is visually supported, the support is often sparse, local, and candidate-dependent. TLRA is built around the claim that decode-time hallucination mitigation should take that locality seriously.

This is not a sweeping claim about all existing baselines. It is a narrower methodological claim: when the task depends on local visual evidence, a framework that explicitly reweights candidate logits using localized support is worth testing against coarser internal alternatives. That claim is experimentally verifiable through internal ablations such as `TLRA_zero` vs. `TLRA_MeanPool`.

### 2.3 Structure Preservation in Decoding

A recurring difficulty in token-level intervention is that not all tokens should be treated equally. Functional syntax, discourse markers, and continuation fragments play a structural role that differs from semantically grounded object words or OCR-bearing entities. A practical decode-time method must therefore distinguish hallucination suppression from indiscriminate token suppression. TLRA addresses this through a deterministic structural mask rather than entropy-only heuristics.

## 3. Methodology

### 3.1 Resolving Embedding Asymmetry: The Calibration Protocol

The first principle of TLRA is fairness transparency. In our preliminary viability probes, a strict zero-shot readout (`TLRA_zero`) mapping contextualized visual states directly to lexical logits yielded near-random overlap. The embedding asymmetry is too large. 

Therefore, the definitive intervention relies on **`TLRA_calib`**. To maintain a fair comparison against inference baselines, the calibration protocol is strictly constrained:
1. **Plug-in Location & Low Cost:** We train a single linear projection matrix, $\Phi_{calib} \in \mathbb{R}^{D_v \times |V|}$, mapping the MLLM's visual tokens—*after* they have passed the native vision-language connector—directly into the LM head's pre-softmax vocabulary space. The parameter overhead is negligible.
2. **Generic Data via Dense InfoNCE:** $\Phi_{calib}$ is trained solely on a strictly limited generic dataset (e.g., a 100K subset of CC3M). We extract spaCy noun chunks, align them to pseudo-bounding boxes via a frozen GroundingDINO matcher, and train $\Phi_{calib}$ using dense token-to-patch InfoNCE. By using noun chunks, we capture explicit attribute-noun combinations (e.g., "red car", "two cats"), though we acknowledge an inherent generic bias against complex spatial prepositions—a limitation we evaluate explicitly in Section 4. Let $\mathcal{P}$ denote the set of aligned token-patch pairs $(t_i, v_i)$ and $\mathcal{N}$ denote in-batch negative textual tokens. The calibrator is optimized via:
   $$ \mathcal{L}_{calib} = - \sum_{i \in \mathcal{P}} \log \frac{\exp(sim(\Phi_{calib}(v_i), t_i) / \tau_{nce})}{\sum_{j \in \mathcal{N}} \exp(sim(\Phi_{calib}(v_i), t_j) / \tau_{nce})} $$
   Importantly, $\Phi_{calib}$ is *never* trained on downstream task instruction-tuning data.
3. **Frozen Base Model:** The vision encoder, vision-language connector, and the LLM remain 100% frozen. 

### 3.2 High-Resolution Efficiency: Prompt-Conditioned Pruning and Fallback

At decoding step $t$, TLRA must avoid evaluating the entire vocabulary against all visual patches, especially when visual tokens $N_v$ exceed 4096. 

**1. Candidate Bounding:** We restrict intervention strictly to $\mathcal{C}_t = \text{Top-}M(L_{orig})$, where $M \ll |V|$ (e.g., $M=10$).

**2. Prompt-Conditioned Pruning with Dynamic Fallback:** Unconditional ViT attention is noisy. We compute the aggregated cross-attention from the *text prompt tokens* to the visual sequence during the prefill stage, pruning visual patches that receive negligible attention to retain an active subset $N_{active}$. 

Crucially, we explicitly acknowledge a **Prefill Pruning Blindspot**: an autoregressive model executing long-horizon generation may require visual evidence for objects never mentioned in the static prompt. Furthermore, attention often collapses into architectural "attention sinks". We explicitly mask out known sinks before computing the prefill attention distribution. To prevent the catastrophic deletion of late-stage latent evidence, TLRA enforces an **Entropy-Driven Dynamic Fallback**: we measure the normalized Shannon entropy $H_{attn}$ of the sink-excluded attention weights. If $H_{attn} > \theta_{fallback}$, pruning is bypassed entirely ($N_{active} = N_v$), preserving all patches for decoding. Thus, for long-form reasoning, TLRA gracefully defaults to dense visual evaluation, relying solely on Candidate Bounding ($M$) for efficiency.

For each candidate $c \in \mathcal{C}_t$, TLRA computes local support using an adaptive Top-$k$ rule over the active patches:
$$
k = \max\Big(k_{min}, \lceil \rho \cdot N_{active} \rceil\Big), \quad S_{raw}(c) = \frac{1}{k}\sum_{j \in \text{Top-}k}\frac{sim(\Phi_{calib}(v_j), c)}{\tau_{sim}}.
$$

### 3.3 Relative Resonance Penalty

To convert raw local support into a stable logits intervention, TLRA computes a relative evidence score:
$$
\hat S(c) = \text{Softmax}\big(S_{raw}(c) \mid c \in \mathcal{C}_t\big).
$$
The final candidate logits are adjusted as:
$$
L_{final}(c) = L_{orig}(c) - \alpha \cdot \gamma(c) \cdot (1 - \hat S(c)).
$$

### 3.4 Structure Preservation via Deterministic VASM

A naive token-level intervention inevitably suppresses functional syntax. TLRA introduces **Vocabulary-Anchored Semantic Masking (VASM)** to deterministically protect these tokens via a binary mask $\gamma(c) \in \{0, 1\}$.

VASM is algorithmically defined using a strict WordNet fail-safe:
1. **Deterministic Whitelist Generation:** A token receives $\gamma=1$ *if and only if* its string representation maps to a lemma under structural synsets like `physical_entity.n.01` or `color.n.01`.
2. **Polysemy Delegation:** If a token matches *any* physical sense, it receives $\gamma=1$. We rely on the base LLM's dominant original logits ($L_{orig}$) to naturally suppress grammatically incorrect verb usages (e.g., "table the discussion").
3. **Tokenizer-Aware BPE Momentum & Collision Avoidance:** Modern tokenizers fragment words. VASM triggers $\gamma=1$ only on the valid initial prefix. Once the initial subword is generated, VASM relies on the base LLM's unmodified autoregressive momentum to complete the suffix tokens ($\gamma=0$). To prevent **false-prefix collisions** (e.g., the abstract word "bulletin" splitting into the physical noun `bullet` and suffix `in`), we utilize a **Tokenizer-Aware Subword Boundary Check**. A length filter ($L_{char} \ge 4$) is *only* applied if the token acts as a subword prefix (e.g., lacking a start-of-word metacharacter like ` ` in SentencePiece, or matching known BPE fragment markers). Standalone whole words, regardless of length (e.g., "cat", "cup"), bypass the length filter and are evaluated normally.
4. **The OCR Concession:** Because VASM explicitly defaults to $\gamma=0$ for out-of-vocabulary text to guarantee grammatical safety, it inherently bypasses arbitrary OCR strings. Preserving LLM fluency takes absolute precedence over hallucination intervention on out-of-dictionary text.

## 4. Evaluation Protocol and Experimental Design

Rather than presenting cherry-picked subsets, our experimental validation is governed by a definitive execution blueprint. The evaluation is structured into three interlocking evidence chains, followed by efficiency and sensitivity audits. All planned results represent the execution of this pre-registered protocol.

### 4.1 Evidence Chain A: Hallucination Reduction (The Length-Collapse Check)

**Goal:** Prove that TLRA reduces object hallucination, that the improvement is not a deceptive artifact of the model generating shorter responses, and that the performance gain stems explicitly from *local extraction* rather than the mere presence of CC3M training data.
- **Table 1 Target:** `POPE` (Accuracy / F1) and `CHAIR` ($CHAIR_s$, $CHAIR_i$) evaluated across two decoding temperatures ($T=0.2$ and $T=0.7$) to test stability.
- **Baselines:** Base MLLM, `DoLa`, `VCD`, `OPERA`, **`TLRA_MeanPool`**, and **`TLRA_RandomK`**.
- **Mandatory Control Metrics:** Average Generated Length (`AGL`), **AGL Standard Deviation**, and **VASM Trigger Rate (%)**.
- **Hypothesis:** `TLRA_AdaptiveTopK` will yield lower `CHAIR` scores and higher `POPE` F1 than all baselines. The `VASM Trigger Rate (%)` will mathematically prove that the intervention is actively firing during these object-heavy tasks. Crucially, it must significantly outperform both `TLRA_MeanPool` (global aggregation) and `TLRA_RandomK` (uniform random patch selection using the exact same $\Phi_{calib}$). 
- **The Asterisk Rule:** Any configuration (baseline or TLRA) achieving SOTA CHAIR scores by collapsing AGL variance by $> 5\%$ compared to the base model will be flagged (*) as an invalid structural collapse artifact.

### 4.2 Evidence Chain B: Structure and Reasoning Preservation (The Negative Control)

**Goal:** Verify that local-evidence intervention does not irreparably degrade general multimodal cognition, prove that VASM effectively protects subword grammar, and validate the "OCR Concession" via an explicit negative control.
- **Table 2 Target:** `MMBench V1.1` (Overall), `MMMU` (Hard - Broken down by discipline), **Language Perplexity (PPL)** measured on WikiText, and `DocVQA`.
- **Ablation Setup:** Base model vs. `TLRA_no_VASM` vs. `TLRA_full`. Mandatory reporting of `VASM Trigger Rate (%)`.
- **Hypothesis:** `TLRA_no_VASM` will exhibit severe perplexity spikes and crash on complex reasoning tasks, particularly destroying mathematical or abstract symbolic disciplines in `MMMU`. `TLRA_full` must recover these scores to baseline levels, ensuring PPL remains strictly $\le +5\%$ of the Base model. 
- **The Strict Negative Control:** Because VASM masks out OCR tokens to protect grammar, `TLRA_full` must demonstrate a `VASM Trigger Rate` statistically indistinguishable from $0\%$ on `DocVQA`. This will result in exactly zero hallucination improvement, but successfully guarantee zero degradation of the Base model's native reading capabilities.

### 4.3 Evidence Chain C: The Value of Local Evidence

**Goal:** Validate the core thesis that *token-local adaptive selection* is superior to coarse visual aggregation, specifically for spatial grounding where global mean-pooling traditionally fails.
- **Table 3 Target:** `FREAK` (explicitly broken down into **Spatial Relations** vs. **Object Existence**) and `Object HalBench`. 
- **Ablation Setup:** Base vs. `TLRA_zero` vs. `TLRA_MeanPool` ($k = N_{active}$) vs. `TLRA_AdaptiveTopK`. 
- **Critical Parity Constraint:** `TLRA_MeanPool` utilizes the identical $\Phi_{calib}$ weights as `TLRA_AdaptiveTopK`.
- **Hypothesis:** `TLRA_zero` will perform poorly, justifying the necessity of $\Phi_{calib}$. For fine-grained **Spatial Relations**, `TLRA_AdaptiveTopK` will demonstrate a statistically significant margin over global aggregation (`TLRA_MeanPool`), whereas the gap on simple *Object Existence* may be narrower. This will isolate the "spatial blurring" failures of global pooling that are corrected by the sharp spatial anchoring of Top-$k$ selection.

### 4.4 Efficiency Audits, Sensitivity, and Failure Analysis

To map the operational boundaries of TLRA, the execution plan mandates the following diagnostic visualizations:

- **Figure 1 (Efficiency & Fallback Audit):** A dual-axis plot showing `Tokens/Second` against Visual Token Volume ($N_v$). Crucially, this must include a scatter plot of the **Fallback Activation Rate** across datasets (e.g., POPE vs. MMMU). If the fallback triggers near $95\%$ on complex reasoning (MMMU), we will report it transparently to confirm that the fallback correctly bypasses the prefill pruning blindspot during long-horizon generation.
- **Figure 2 (Hyperparameter Sensitivity):** A grid showing `POPE F1` and `Perplexity` across varying $M$ (candidate window), $\alpha$ (intervention strength), and the Entropy Threshold ($\theta_{fallback}$).
- **Figure 3 (Qualitative Failure Analysis):** We mandate specific heatmaps showcasing intervention limits:
  1. **BPE Stuttering Heatmap:** Visualizing base logits vs. intervened logits on a multi-token fragmented word where VASM correctly triggered on the prefix but the autoregressive momentum failed to carry the suffix (e.g., "straw-straw-berry").
  2. **Out-of-Candidate Unrecoverability:** An instance where the correct physical token was ranked $> M$ by the base model, rendering it invisible to TLRA.

## 5. Discussion and Limitations

The strongest version of TLRA is achieved by tightly bounding its claims to its empirical execution. The core scientific contribution is a positive, decode-time framework that leverages token-local visual evidence while deploying VASM to prevent structural collapse. However, several hard limitations define the boundaries of this method:

1. **The "Training-Free" Distinction and Calibrator Asymmetry:** While TLRA operates as a base-model-frozen plug-in during inference, the $\Phi_{calib}$ projector is trained on generic external data. This creates a fundamental information asymmetry compared to purely unsupervised methods like DoLa or VCD. The scientific validity of TLRA relies entirely on the `TLRA_MeanPool` and `TLRA_RandomK` ablations to strictly isolate the decode-time local extraction mechanism from the dataset contribution.
2. **Prefill Pruning Blindspot and Entropy Fallback Hypersensitivity:** Relying on prefill cross-attention for visual pruning is architecturally perilous. In open-ended generations, the model often requires visual evidence for objects not queried in the static prompt. Consequently, our entropy-driven fallback does heavy lifting. For long-horizon multimodal reasoning, the fallback will frequently trigger, entirely bypassing pruning and defaulting to dense evaluation. While computationally bounded by candidate selection ($M$), the "pruning efficiency" claim is therefore heavily restricted to short-horizon targeted QA.
3. **InfoNCE Calibrator Bias:** Because $\Phi_{calib}$ is trained on spaCy noun chunks aligned to bounding boxes, it is naturally biased toward physical object nouns and attribute-noun pairs (e.g., "red car"). It is inherently weak at scoring localized spatial prepositions or abstract temporal attributes.
4. **English-Centric Brittleness:** VASM's deterministic protection mechanism is strictly bound to the English WordNet database. TLRA inherently lacks the ability to scale to multilingual reasoning without parallel, manually curated lexical databases for target languages.
5. **The False-Prefix BPE Collision and Suffix Momentum:** Even with Tokenizer-Aware Subword Boundary Checks, pathological tokenization can trigger unwarranted visual intervention on abstract words sharing prefixes with physical entities. TLRA fundamentally relies on the base LLM's unmodified autoregressive momentum to complete BPE fragments, which can occasionally result in observable BPE stuttering if the base logits are weak.
6. **The OCR / OOV Paradox:** To guarantee the safety of functional syntax, VASM explicitly sacrifices intervention on out-of-vocabulary strings. TLRA is inherently ineffective for arbitrary OCR transcription, prioritizing absolute grammatical safety over in-the-wild text intervention.
